# 创建你自己的列表

- 阅读 [Awesome 宣言](awesome_zh.md) 和 [列表指南](pull_request_template_zh.md)，确保你的列表符合要求。
- 在创建新列表之前搜索本列表，你的列表可能是重复的。如果是这样，尝试为最好的那个做贡献，而不是创建自己的。
- 你可能会发现[这个 Yeoman 生成器](https://github.com/dar5hak/generator-awesome-list)很有用。
- **在创建列表后至少等待 30 天再提交，让它有时间成熟。**
- **在提交 pull request 将你的列表添加到这里之前，请确保重新阅读[列表指南](pull_request_template_zh.md)。**

感谢你的 awesome！😎
