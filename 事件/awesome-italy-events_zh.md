## 一月
- 27-31 - [Global Game Jam](https://globalgamejam.org/) - 多个地点 - 在一个周末聚在一起制作游戏。

## 二月
- 11-12 - [航运4.0](https://shipping-40.sharevent.it/) - 在线 - 航运业的技术改进。
- 21 - [ServerlessDays 2021](https://rome.serverlessdays.io/) - 罗马 - 关于无服务器技术的面向开发人员的会议。
- 24 - [AWS Innovate ML 和 AI](https://aws.amazon.com/events/aws-innovate/machine-learning/) - 在线 - 使用 AWS 进行 AI 和机器学习。

## 三月
- 2-4 - [Microsoft Ignite](https://myignite.microsoft.com/home) - 在线 - 微软年会。
- 11 - [CSSDay 2021](https://2021.cssday.it/) - 在线 - CSS 专题会议。
- 23-25 - [2021 年 Codemotion 在线技术会议](https://events.codemotion.com/conferences/online/2021/online-tech-conference-italian-edition-spring/) - 在线 - 意大利最大的技术会议。

- 15 - [RubyDay 2021](https://2021.rubyday.it/) - 在线 - Ruby 全国会议。
- 28-30 - [Secsolutionforum](https://www.secsolutionforum.it/) - 在线 - 安全行业的数字盛会。
- 29 - [VueDay 2021](https://2021.vueday.it/) - 在线 - Vue 的全国会议。
