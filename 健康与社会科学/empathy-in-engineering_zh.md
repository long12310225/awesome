关于如何在工作中建立更具同理心的文化及其好处的文章 (📰)、书籍 (📚) 和视频 (📹)。


### 对你的用户有同理心

- [太敏感：关于用户体验中的微侵犯](https://medium.com/@sara_ann_marie/too-sensitive-9752a86a8382) 📰 - Medium 上的 Sara Ann Marie 讨论了用户体验中的小问题如何疏远用户。
- [可操纵用户的幻想和滥用](https://modelviewculture.com/pieces/the-fantasy-and-abuse-of-the-manipulable-user) 📰 - Betsy Haibel 在模型视图文化中讨论了用户体验“灰色模式”和“黑暗模式”如何在网络中流行。
- [同理心：软件项目成功的关键](https://www.oreilly.com/ideas/empathy-the-key-to-a-successful-software-project) 📰 - O'Reilly 的 Gregory Brown 讨论了如何识别最重要的利益相关者并提出问题，以便为他们构建最好的产品。
- [关于 Facebook 的“同情团队”](http://www.nytimes.com/2016/03/13/fashion/facebook-breakup-compassion-team.html?_r=0) 📰 - 《纽约时报》关注 Facebook 的“同情团队”，其任务是让 Facebook 更加人性化。
- [为灾难而设计](https://vimeo.com/148927676) 📹 - An Event Apart 的 Eric Meyer 告诉我们他一生中最困难的一天，以及为那些也可能经历一生中最糟糕日子的人们进行设计的重要性。
- [现实生活设计](https://abookapart.com/products/design-for-real-life) 📚 - Eric Meyer 和 Sara Wachter-Boettcher 所著的这本书重点介绍了识别压力情况和富有同情心的设计技巧。


### 对同龄人有同理心

- [我们邀请 Etsy 的每个人进行工程轮换：原因如下](https://codeascraft.com/2014/12/22/engineering-rotation/) 📰 - 在 Etsy 的 Code as Craft 博客中，他们讨论了为什么让设计师和产品经理进行工程轮换。
- [Slack 的同理心和拉取请求](https://slack.engineering/on-empathy-pull-requests-979e4257d158) 📰 | [后续](https://slack.engineering/how-about-code-reviews-2695fb10d034) 📰 - Slack Engineering 博客探讨了拉取请求是我们团队成员同理心的一种表现。
- [无责备事后剖析和公正文化](https://codeascraft.com/2012/05/22/blameless-postmortems/) 📰 - Etsy 的 Code as Craft 博客讨论了他们如何从无责备事后剖析中学习的角度看待错误。
- [Google 从构建完美团队的探索中学到了什么](http://www.nytimes.com/2016/02/28/magazine/what-google-learned-from-its-quest-to-build-the-perfect-team.html) 📰 - 《纽约时报》综合了 Google 的数据驱动文化如何发现以沟通和同理心为中心的规范可以打造更好的团队。
- [困难的对话：如何讨论最重要的事情](http://smile.amazon.com/gp/product/B004CR6ALA/) 📚 - 哈佛谈判项目的 Douglas Stone、Bruce Patton 和 Sheila Heen 撰写了如何通过一系列考虑“第三个故事”的技巧来处理困难的对话。
- [工程同理心：在我们的代码学校培养包容性文化](http://www.alterconf.com/talks/engineering-empathy-fostering-inclusive-culture-our-code-school) 📹 - Lateesha Thomas 讨论了为什么同理心是 Alterconf SF 2016 的 DevBootcamp 课程的一部分。
- [同理心解释器](https://www.youtube.com/watch?v=1Evwgu369Jw) 📹 - Brené Brown 的快速动画讲述什么是同理心以及它与同情有何不同。

### 招聘中的同理心

- [同理心招聘流程指南](https://medium.com/@fox/a-guide-to-empathetic-hiring-processes-c11c7ce0cd49) 📰 - Medium 上的 @Fox 讨论了雇主可以做的一些简单的事情，以善待求职者。


## JavaScript

有时您需要代码示例来实施上述策略。

### 模拟
- [JS Simulation of Dyslexia](http://geon.github.io/programming/2016/03/03/dsxyliea) - 这是对患有阅读障碍的人阅读一段文本的模拟。

### 小书签
- [JS Bookmarklet Simulating Colorblindness](https://github.com/Altreus/colourblind) - 这是一个 JS 书签，可以获取任何页面并模拟色盲导航的情况。
- [NoCoffee: JS Bookmarklet Simulating Low Vision Scenarios](https://accessgarage.wordpress.com/2013/02/09/458/) - 这是一个JS书签，可以抓取任何页面并模拟低视力场景。
