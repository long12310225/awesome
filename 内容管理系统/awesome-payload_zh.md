## 插件

有趣的社区插件列表。

- [r1tsuu / payload-enchants](https://github.com/r1tsuu/payload-enchants) - Payload 增强包的集合。
- [authsmith / payload-auth-plugin](https://github.com/authsmith/payload-auth-plugin) - 该插件扩展了 Payload 身份验证过程并实现了基于 OAuth 和 Passkey 的身份验证功能。
- [payload-auth / payload-auth](https://github.com/payload-auth/payload-auth) - 用于将 Payload 与各种身份验证提供程序集成的 monorepo。
- [CrawlerCode / payload-authjs](https://github.com/CrawlerCode/payload-authjs) - Auth.js 的有效负载插件。
- [ashbuilds / payload-ai](https://github.com/ashbuilds/payload-ai) - Payload AI 插件是一个强大的扩展，集成了先进的 AI 功能来增强内容创建和管理。
- [elliott-w / imageAspectRatios.ts](https://gist.github.com/elliott-w/19676d9373d79ee20b9195601dc45808) - 有效负载图像长宽比插件。
- [AdrianMaj / payload-lexical-typography](https://github.com/AdrianMaj/payload-lexical-typography) - 用于扩展词汇排版选项的有效负载插件（文本颜色、字体大小、字母间距、行高、字体系列等）
- [rilrom / payload-bites](https://github.com/rilrom/payload-bites) - 各种小型 Payload v3 插件的集合，包括图像搜索、全屏编辑器、审核字段、软删除和活动日志。
- [shefing / payload-tools](https://github.com/shefing/payload-tools) - 包含一组强大的插件，旨在增强您的有效负载项目（授权、作者信息、富文本注释、右面板、自定义版本视图、跨集合配置、字段类型组件覆盖、颜色选择器字段、图标选择、快速过滤器、重置列表视图等）。
- [Crayonan / payload-plugin-ai-localization](https://github.com/Crayonan/payload-plugin-ai-localization) - 向有效负载中的非主要本地化字段添加人工智能驱动的“翻译”按钮。使用 OpenAI 或其他模型自动翻译主要语言的内容。
