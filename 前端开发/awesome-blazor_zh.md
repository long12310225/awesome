## 简介

### 什么是 Blazor？

Blazor 是一个 .NET Web 框架，用于使用 C# 构建客户端 Web 应用程序。

Blazor 允许您使用 C# 而不是 JavaScript 构建交互式 Web UI。 Blazor 应用程序由使用 C#、HTML 和 CSS 实现的可重用 Web UI 组件组成。客户端和服务器代码都是用 C# 编写的，允许您共享代码和库。
有关 [Blazor 官方网站](https://blazor.net) 的更多信息。

### 开始使用

要开始使用 Blazor，请按照 [Blazor 入门](https://docs.microsoft.com/aspnet/core/blazor/get-started) 文档中的说明进行操作。

在 Microsoft Learn 上进行[使用 Blazor 构建 Web 应用](https://docs.microsoft.com/en-us/learn/modules/build-blazor-web assembly-visual-studio-code/) 学习课程也是一个好主意。另一个很好的入门资源是 Jeff Fritz 在 [Channel9](https://channel9.msdn.com/Series/Beginners-Series-to-Blazor) 或 [YouTube](https://www.youtube.com/playlist?list=PLdo4fOcmZ0oUJCA3DCzKT79Oe3kdKEceX) 上的初学者系列。
