## 帆布

### 定义

“在 HTML5 中添加的 HTML <canvas> 元素可用于通过 JavaScript 中的脚本绘制图形。例如，它可用于绘制图形、制作照片合成、创建动画，甚至进行实时视频处理或渲染。”由 [Mozilla 开发者网络](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)

## 图书馆

### 使用画布进行绘制

- [Akihabara](http://www.kesiev.com/akihabara/) 是 HTML5 游戏库，用于使用 Javascript 和 canvas 标签制作基于像素的游戏。
- [CanvasKit](https://skia.org/docs/user/modules/canvaskit/) 是 Skia 的 Canvas API 编译为 WebAssembly，为 Web 应用程序提供高性能 2D 图形渲染。
- [Chart.js](https://github.com/chartjs/Chart.js) 是一个轻量级 JavaScript 库，用于使用 HTML5 Canvas 元素创建动态且具有视觉吸引力的图表。
- [ChemDoodle](http://web.chemdoodle.com/) 是一个开源化学和化学信息学工具包，其中画布用于解决常见的化学相关任务，以各种不同的方式显示分子。
- [d3](https://github.com/d3/d3)（或 D3.js）是一个 JavaScript 库，用于使用 Web 标准可视化数据。 D3 可帮助您使用 SVG、Canvas 和 HTML 将数据变为现实。
- [EaselJS](https://createjs.com/easeljs) 是一个 JavaScript 库，可以轻松使用 HTML5 Canvas 元素。对于创建游戏、生成艺术和其他高度图形化的体验很有用。 EaselJS 是 CreateJS 的一部分，CreateJS 是一个模块化库和工具，可以协同或独立工作，通过 HTML5 在开放 Web 技术上实现丰富的交互式内容。
- [fabric.js](http://fabricjs.com) 在画布元素之上提供交互式对象模型
并且还有 SVG 到画布（和画布到 SVG）解析器
- [iio.js](https://github.com/iioinc/iio.js) - 一个 JavaScript 库，可加速 HTML5 Canvas 应用程序的创建和部署
- [isomerjs](http://jdan.github.io/isomer/) - HTML5 画布的等距图形库
- [Javascript-Voronoi](https://github.com/gorhill/Javascript-Voronoi) - 计算 Voronoi 单元的 Fortune 算法的 Javascript 实现
- [Konva](https://github.com/konvajs/konva) - Konva.js 是一个 HTML5 Canvas JavaScript 框架，它通过为桌面和移动应用程序启用画布交互性来扩展 2d 上下文。
- [Origami.js](https://github.com/raphamorim/origami.js) - JS Lib 重新设计 canvas API 接口
- [p5.js](https://p5js.org) - p5.js 是一个 JS 客户端库，用于创建图形和交互体验
- [Paper.js](https://github.com/paperjs/paper.js) - Scriptographer 使用 HTML5 Canvas 移植到 JavaScript 和浏览器。
- [Pencil.js](https://pencil.js.org/) - 漂亮的模块化 Javascript 库，具有清晰的 OOP 语法和大量功能。
- [Pixi.js](http://www.pixijs.com) - 超快速 HTML 5 2D 渲染引擎，使用带有画布后备的 webGL
- [Processingjs](http://ejohn.org/blog/processingjs/) 是一种数据可视化编程语言。
    - [Overview](http://ejohn.org/blog/overview-of-processing/)
    - [91 个基本演示](http://ejohn.org/apps/processing.js/examples/basic/)
- [Proton](https://github.com/a-jie/Proton) 是一个轻量级且功能强大的 javascript 粒子引擎。有了它你可以轻松创造无数炫酷效果
- [Pts.js](https://ptsjs.org) - Pts 是一个用于可视化和创意编码的 JavaScript 库。
- [Rough.js](https://roughjs.com/) - Rough.js 是一个图形库，可让您以类似手绘的粗略风格进行绘制
- [Scrawl-canvas](https://scrawl-v8.rikweb.org.uk/) - 轻松向网页添加多个响应式、可访问且交互式的 <canvas> 元素
- [Sketch](https://github.com/soulwire/sketch.js) - 跨平台 JavaScript 创意编码框架
- [Three.js](http://thirdjs.org/) 是一个 javascript 库，可在浏览器中制作 WebGL - 3D，但是您可以[使用画布而不是 WebGL 进行渲染](http:// Threejs.org/docs/#Reference/Renderers/CanvasRenderer)
    - [Introduction](http://threejs.org/docs/#Manual/Introduction/Creating_a_scene)
    - [一些令人惊奇的例子](http://threejs.org/examples/)
- [textmode.js](https://github.com/ humanbydefinition/textmode.js) 是一个轻量级、与框架无关的创意编码库，用于浏览器中的实时 ASCII 和文本模式图形。
- [tsParticles](https://articles.matteobruni.it/) 是一个轻量级库，用于轻松创建粒子动画。它包括最常用的 Javascript 框架（ReactJS、VueJS、Angular 等）的现成组件
- [Visualize](https://github.com/filamentgroup/jQuery-Visualize) 是一个 JQuery 插件，它使用 HTML canvas 元素从表格数据创建图表和图形。
- [zDog](https://github.com/metafizzy/zdog) - 适用于画布和 SVG 的扁平、圆形、设计师友好的伪 3D 引擎
- [ZIM](https://zimjs.com/) - ZIM 是一个通用的 Canvas 框架，具有简单、强大的 JavaScript，让每个人（从初学者到专业人士）都可以编写创意代码。
- [zrender](https://github.com/ecomfe/zrender) - 一个轻量级画布库，为 Apache ECharts 提供 2d 绘图（孵化中）

### 用于其他目的，但仍使用画布
- [React Canvas](https://github.com/Flipboard/react-canvas) - React 组件的高性能 <canvas> 渲染。
