* [grid-styled](https://github.com/jxnblk/grid-styled) - 基于 Flexbox 的响应式 ReactJS 网格系统，使用样式组件构建
* [iron-flex-layout](https://www.webcomponents.org/element/PolymerElements/iron-flex-layout) - 跨平台弹性盒布局的样式混合
* [后启示录 Flexbox](https://github.com/afonsopacifer/post-apocalypse-flexbox)
* [React-flexbox](https://github.com/tcoopman/react-flexbox) - CSS Flexbox 的实现与内联样式发生反应。
* [Reflexbox](https://github.com/jxnblk/reflexbox) - 响应式 React Flexbox 网格系统高阶组件。
* [Solved by flexbox](https://github.com/philipwalton/solved-by-flexbox) - 展示曾经仅用 CSS 很难或不可能解决的问题，现在使用 Flexbox.:metal 变得非常简单：
* [Visualizing Flexbox](http://codepen.io/paultrone/pen/xwxNmQ?utm_campaign=CSS%2BLayout%2BNews&utm_medium=email&utm_source=CSS_Layout_News_6) - 一个简洁的小 CodePen 演示，用于使用 Flex 属性。
