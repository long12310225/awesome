> 根据类名自动生成相应的 CSS 规则，通过增强的 CSS 语法，您可以用更少的代码构建 UI，并在生产中直接使用高性能 JIT。

[Master CSS](https://css.master.co) 是一种具有增强语法的虚拟 CSS 语言，仅使用 HTML 即可高效构建您的 UI 和设计系统。

## 工具
- [Master Styles CSS Converter](https://github.com/serkodev/master-styles-css-converter) - 在线将 CSS 转换为主样式。

## 插件
- [@master/normal.css](https://github.com/master-co/css/tree/beta/packages/normal.css) - 规范化浏览器的样式。
- [@master/keyframes.css](https://github.com/master-co/css/tree/beta/packages/keyframes.css) - 简单实用的 CSS 关键帧。
- [@master/style-element.react](https://github.com/master-co/style-element.react) - 使用条件类名快速创建样式化的 React 元素。
- [@master/literal](https://github.com/master-co/literal) - 使用模板文字有条件地构造类名和字符串。
