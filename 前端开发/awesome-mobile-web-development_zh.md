## W3C 增强移动 Web 体验的规范

- [Battery API](https://www.w3.org/TR/battery-status/) - 关于获取有关电池的信息。
- [Connection API](http://wicg.github.io/netinfo/) - 关于获取有关连接的信息。
- [Generic Sensor API](https://www.w3.org/TR/generic-sensor/) - 公开传感器数据的规范。
- [Geolocation API](https://www.w3.org/TR/geolocation-API/) - 关于访问地理位置。
- [HTML Media Capture](https://www.w3.org/TR/html-media-capture/) - 用户访问摄像头或麦克风时。
- [Payment Request API](https://www.w3.org/TR/payment-request/) - 具有最小集成度的支付方式。
- [Presentation API](https://www.w3.org/TR/presentation-api/) - 关于访问演示显示。
- [Remote Playback API](https://www.w3.org/TR/remote-playback/) - 从网页控制媒体的远程播放。
- [Vibration API](https://www.w3.org/TR/vibration/) - 关于访问托管设备的振动机制。
- [Visual Viewport API](https://wicg.github.io/visual-viewport/) - 访问屏幕的视觉部分（不包括屏幕键盘、捏合缩放区域之外的区域等）。
- [Wake Lock Api](https://www.w3.org/TR/wake-lock/) - 访问唤醒锁定请求（防止设备的某些方面进入省电状态）。
- [Web Bluetooth](https://webbluetoothcg.github.io/web-bluetooth/) - 通过 API 发现设备并通过蓝牙 4 与设备通信。
- [Web NFC API](https://w3c.github.io/web-nfc/) - 关于访问设备的近场通信功能。
- [Web Notifications](https://www.w3.org/TR/notifications/) - 用于最终用户通知的 API。

## 审计工具

- [Lighthouse](https://github.com/GoogleChrome/lighthouse) - Web 应用程序和网页的分析工具。
- [Mobile-Friendly Test](https://search.google.com/test/mobile-friendly) - 用于测试您的网站是否适合移动设备的工具。
