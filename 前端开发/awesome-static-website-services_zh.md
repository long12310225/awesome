## 音频

- [SoundCloud](https://soundcloud.com/) - 使用嵌入式播放器进行音频托管。最多 3 小时的内容免费。
- [Mixcloud](https://www.mixcloud.com/) - 具有无限上传和嵌入式播放器的音频托管。
- [Spotify](https://www.spotify.com/) - 您可以使用 [Spotify 播放按钮](https://developer.spotify.com/documentation/widgets/guides/adding-a-spotify-embed/) 嵌入任何歌曲、专辑或播放列表。

## 书籍

- [Goodreads API and widgets](https://www.goodreads.com/api) - 允许您访问任何 Goodreads 数据。小部件可以在[设置页面](https://www.goodreads.com/user/edit) 的小部件选项卡上找到。
- [Open Library Open API](https://openlibrary.org/developers/api) - 允许从地球上最大的图书开放数据集中检索图书数据，无需任何跟踪。

## 日历和日程安排

- [Google Calendar](http://calendar.google.com/) - 您可以与其他人协作的嵌入式日历。
- [Booking.js](http://booking.timekit.io/) - 漂亮的嵌入式预订小部件。
- [zenplanner](http://zenplanner.com/) - 付费 - 在线安排健身。

## 图片

- [Flickr](https://www.flickr.com/) - 雅虎在线照片托管。
- [Cloudinary](https://cloudinary.com/) - 图像托管、操作和交付。

## 地图

- [Google Maps](http://maps.google.com/) - 谷歌地图很容易嵌入。
- [Mapbox](https://www.mapbox.com/) - 非常漂亮的嵌入式地图。
- [uMap](https://umap.openstreetmap.fr/en/) - 基于出色的 [OpenStreetMap](https://openstreetmap.org) 数据集创建并嵌入个性化地图。

## 演讲

- [Prezi](https://prezi.com/) - 具有可缩放和旋转的真正过渡的在线演示。
- [Reveal.js](http://lab.hakim.se/reveal-js/) - HTML 表示框架。
- [Slides.com](http://slides.com/) - 用于创建、演示和共享幻灯片的地方。
- [SpeakerDeck](https://speakerdeck.com/) - 以 PDF 格式上传您的幻灯片，并获得可共享的在线演示文稿。

## 功能即服务

- [1Backend](https://github.com/1backend/1backend) - 在几秒钟内部署您的后端。包括免费套餐。开源。
- [AWS Lambda](https://aws.amazon.com/lambda/) - AWS Lambda 让您无需预置或管理服务器即可运行代码。您只需为消耗的计算时间付费
- [Google Cloud Functions](https://cloud.google.com/functions/) - 创建单一用途的独立函数来响应云事件，而无需管理服务器或运行时环境
- [Webtask by Auth0](https://webtask.io/) - 使用简单的 HTTP 调用服务器上的代码，比 Lambda 或 Google 更容易设置
- [Azure Functions](https://azure.microsoft.com/en-us/services/functions/) - 由 Microsoft 提供 - 与 Azure 云上的 Lambda 相同的前提
- [IronWorkers](https://www.iron.io/platform/ironworker/) - 由 Iron.io - 在多语言容器化环境中运行代码，具有无限的规模和简单的定价
- [IronFunctions](http://open.iron.io/) - Iron.io - IronFunctions 是一个开源无服务器计算平台，适用于任何云（私有云、公共云或混合云）。
- [OpenWhisk by IBM](https://console.ng.bluemix.net/openwhisk/) - 他们的 BlueMix 托管平台的一部分，并且是开源的，与他们的 Watson AI 生态系统很好地联系在一起
- [StackPath EdgeEngine](https://www.stackpath.com/products/edgeengine/) - 用您选择的语言将函数编写为服务，并将其部署到全球数据中心网络。所有网络（包括智能路由和负载平衡）均由 StackPath 通过专用主干网进行管理。
- [Cloud 66](https://www.cloud66.com/) - 构建静态网站并将其部署到您自己的云帐户。
- [Vercel](https://vercel.com/home#features) - Vercel 允许人们用自己选择的语言将函数编写为服务，并作为单一存储库的一部分进行部署。
- [Azure Static Web Apps](https://azure.microsoft.com/en-us/services/app-service/static/#features) - 全栈静态应用程序托管，包括无服务器功能、身份验证、CDN 等
- [Qoddi](https://qoddi.com) - 包含数据存储的全栈应用程序托管，类似于具有以开发人员为中心的方法的 Heroku。最多可免费使用 3 个应用程序。
- [Autocode](https://autocode.com) - 自动编码可立即将功能转变为可扩展的 Web 服务，无需配置；无服务器托管可通过浏览器内 IDE、易于使用的 API 库以及无缝管理的身份验证来实现。

## GraphQL

- [FaunaDB](https://fauna.com) - 无服务器 GraphQL 数据库。免费套餐，无时间限制。轻松包含在 Netlify 应用程序中。

## 社区

### 评论

- [Staticman](https://github.com/eduardoboucas/staticman) - Staticman 是一个 Node.js 应用程序，它接收用户生成的内容并将其作为数据文件上传到 GitHub 存储库。实际上，这允许您将动态内容（例如博客文章评论）作为完全静态网站的一部分，只要您的网站在每次推送到 GitHub 时自动部署，如 GitHub Pages、Netlify 等上所示。
- [Disqus](https://disqus.com/) - 通过嵌套回复、多种登录方法和电子邮件通知轻松嵌入评论。

- [IntenseDebate Comments](http://www.intensedebate.com/) - 带有嵌套回复、多种登录方法和电子邮件通知的嵌入评论。

- [utterances](https://github.com/utterance) - 一个基于 GitHub 问题构建的轻量级评论小部件。

## 搜索

自托管：

- [lunr.js](http://lunrjs.com/) - 在浏览器中进行简单的全文搜索。
- [itemsjs](https://github.com/itemsapi/itemsjs) - JavaScript 中的全文、分面、几乎无依赖的搜索引擎
- [minisearch](https://github.com/lucaong/minisearch) - 适用于浏览器和 Node 的小型且强大的 JavaScript 全文搜索引擎
- [flexsearch](https://github.com/nextapps-de/flexsearch) - 适用于 Browser 和 Node.js 的下一代全文搜索库
- [fuse.js](https://fusejs.io/) - 功能强大、轻量级的模糊搜索库，零依赖性
- [static-search](https://github.com/dchest/static-search) - 用于生成 HTML 文件的 JSON 索引的 Go 程序，以及带有可选 UI 的 JavaScript 组件来搜索该索引
- [elasticlunr](http://elasticlunr.com/docs/index.html) - 用JavaScript开发的轻量级全文搜索引擎，用于浏览器搜索和基于Lunr.js的离线搜索
- [tinysearch](https://github.com/tinysearch/tinysearch) - 用于使用 Rust 和 Wasm 构建的静态网站的小型全文搜索引擎
- [js-search](https://www.npmjs.com/package/js-search) - 客户端搜索 JavaScript 和 JSON 对象，兼容 ES5，不需要 jQuery 或任何其他第三方库
- [search-index](https://github.com/fergiemcdowall/search-index) - 适用于浏览器和 Node.js 的持久性、网络弹性、全文搜索库
- [fuzzysearch](https://github.com/bevacqua/fuzzysearch) - JavaScript 中的微小且快速的模糊搜索
- [fuzzy](https://github.com/mattyork/fuzzy) - 浏览器和节点的模糊搜索/过滤
- [fullproof](https://reyesr.github.io/fullproof/) - Javascript 库，在浏览器中提供高质量的全文搜索
- [Jets.js](https://jets.js.org/) - 原生 CSS 搜索引擎
- [Hulipaa](https://github.com/sambuccid/hulipaa) - 静态全文搜索引擎能够处理数千个页面，并使用简约的搜索栏 UI 添加到您的静态网站。

第三方集成：

- [Google Custom Search Engine](https://cse.google.com/cse/) - 使用自定义 Google 搜索搜索您的网站。
- [Algolia](https://www.algolia.com/) - 托管搜索 API，第一次击键即可提供即时且相关的结果。
- [CloudSh](https://cloudsh.com/) - 只需几行 JavaScript 即可对您的网站进行强大的搜索。
