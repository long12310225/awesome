- 🌍🔧 [Base-2 Rounding](https://objectivelyround.dev) - Tailwind CSS v4 的无括号任意值的间距和大小值选择器。
- 🔼🌍 [前缀](https://github.vue.tailwind-prefix.cbass.dev) - Tailwind 类的前缀工具。
- 🔼 [RustyWind](https://github.com/avencera/rustywind) - 用于对 Tailwind CSS 类进行排序的 CLI 工具。
- 🔼 [Tailwind 到内联样式转换器](https://github.com/vardan-arm/tailwind-to-inline) - 将 Tailwind CSS 类转换为电子邮件模板中的内联样式。
- 🚀 [Maizzle](https://maizzle.com/) - 使用 Tailwind CSS 进行快速电子邮件原型设计的框架。
- 💼 [`@nuxtjs/tailwindcss`](https://github.com/nuxt-community/tailwindcss-module) - Nuxt.js 的 Tailwind CSS 模块，具有 PurgeCSS 和现代 CSS（预设环境 1）。
- 💼 [tailwindcss-rails](https://github.com/rails/tailwindcss-rails) - 将 Tailwind CSS 与 Rails 资产管道结合使用的 Gem。
- 💼 [配置查看器](https://github.com/rogden/tailwind-config-viewer) - 用于可视化 Tailwind CSS 配置文件的本地 UI 工具。
- 💼 [Raycast 扩展](https://www.raycast.com/vimtor/tailwindcss) - 在 Raycast Launcher 中搜索类、文档和颜色。
- 💼 [NativeWind](https://www.nativewind.dev) - 使用 Tailwind CSS 作为脚本语言为 React Native 创建通用样式系统。
- 🌐 [Gimli Tailwind](https://chromewebstore.google.com/detail/gimli-tailwind/fojckembkmaoehhmkiomebhkcengcljl) - Tailwind CSS 的智能工具作为浏览器扩展。
- 🌐 [CSS 变量编辑器](https://www.cssvariables.com) - AI 支持的 Chrome 扩展，用于管理 daisyUI 和 shadcn/ui 中的颜色。
- 🌐 [DivMagic](https://divmagic.com) - 将任何 Web 元素和样式复制为 Tailwind CSS 组件。

## 插件

**图例**： 💙 官方插件 · 🎨 主题 · 💼 实用程序 · 🧩 组件 · 🛑 已弃用

- 💙🧩 [Typography](https://github.com/tailwindlabs/tailwindcss-typography) - 添加一个 `prose` 类以实现漂亮的默认排版。
- 💙 [表单](https://github.com/tailwindlabs/tailwindcss-forms) - 为表单元素添加更好的默认样式。
- 🎨 [Themer](https://github.com/RyanClementsHax/tailwindcss-themer) - 使用 CSS 变量和变体添加对 Tailwind CSS 的主题支持。
- 💼 [Bootstrap grid](https://github.com/karolis-sh/tailwind-bootstrap-grid) - 生成 Bootstrap 风格的 Flexbox 网格系统。
- 💼 [点和网格背景](https://github.com/TheNaubit/tailwind-dot-grid-backgrounds) - 添加 `bg-grid` 和 `bg-dot` 类，仅使用 CSS 添加易于自定义的网格和点图案背景。
- 💼 [Leading Trim](https://github.com/stormwarning/tailwindcss-capsize) - 添加实用程序以使用 [Capsize](https://github.com/seek-oss/capsize) 修剪文本空白。
- 💼 [滚动条隐藏](https://github.com/reslear/tailwind-scrollbar-hide) - 添加 `scrollbar-hide` 类以可视化隐藏滚动条。
- 💼 [px 到视口](https://github.com/the-lemonboy/tailwindcss-px-to-viewport) - 添加实用程序以自动将 px 转换为 vw / vh。
- 💼 [tailwind-hitslop](https://github.com/Jubstaaa/tailwind-hitslop) - 添加 `hit-slop` 实用程序以无形地扩展触摸目标，如 React Native 的 `hitSlop`。
- 💼🧩 [Fluid](https://github.com/barvian/fluid-tailwind) - 为每个内置实用程序添加流体 `clamp()` 版本。
- 🧩 [调试屏幕](https://github.com/jorenvanhee/tailwindcss-debug-screens) - 添加一个显示当前活动屏幕的组件（响应断点）。
