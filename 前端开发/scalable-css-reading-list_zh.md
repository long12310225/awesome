# 可扩展的 CSS 阅读列表 [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

解决这两个问题的值得阅读或观看的内容列表：**什么是可扩展 CSS？** 和 **我们如何创建可扩展 CSS？**

通过*可扩展 CSS*，我的意思是 CSS
- 随着项目的不断发展，能够保持有效、连贯、可扩展和可维护
- 能够被许多不同的人以一致、系统的方式理解和工作

其中包括阐明关键原则和实践的资源。该列表仅限于我认为最“重要”阅读的文章，即对解决问题的不同方法的最佳解释（我所知道的）。

**如果您知道应该添加的资源，请分享！提出问题或拉取请求。**

（还有很多与 CSS 相关的其他问题：了解它的工作原理、使用特定属性、完成特定样式、实现响应式设计、提高性能等。此列表严格关注创建可扩展 CSS 的问题。）

*所有部分均不提供任何类型的“排名”或“建议阅读顺序”。只需阅读全部内容即可。*
