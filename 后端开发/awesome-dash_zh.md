> [Dash](https://plot.ly/products/dash/) 是一个用于构建 Web 应用程序的高效 Python 框架。
> Dash 是在 Flask、Plotly.js 和 React.js 之上编写的，非常适合用纯 Python 构建具有高度自定义用户界面的数据可视化应用程序。它特别适合任何使用 Python 处理数据的人。

## 组件库

- [Dash PDF](https://github.com/ploomber/dash-pdf) - 显示内联 PDF
- [Dash MUI](https://github.com/ploomber/dash-mui) - 材质 UI 组件
- [Dash React Simple Maps](https://github.com/ploomber/dash-react-simple-maps) - 创建交互式地图
- [Dash Mosaic](https://github.com/ploomber/mosaic-python#dash-mosaic) - 显示[马赛克](https://github.com/uwdata/mosaic)图
- [Dash Tabler Icons](https://github.com/ploomber/dash-tabler-icons) - 为您的 Dash 应用程序提供漂亮的图标
- [Dash React Syntax Highlighter](https://github.com/ploomber/dash-react-syntax-highlighter) - 使用复制按钮显示代码片段
- [Dash Canvas Components](https://dash.plotly.com/canvas) - 用于图像注释和图像处理的模块
冲刺。
- [Awesome React Components](https://github.com/brillout/awesome-react-components) - React.js 组件目录可能有兴趣包装到 Dash 组件中。
- [Awesome React](https://github.com/enaqx/awesome-react) - 用于开发 React.js 组件的有用资源。
- [达世币核心组件](https://github.com/plotly/dash-core-components)
- [Dash Cytoscape Component](https://dash.plotly.com/cytoscape) - 图形可视化组件，轻松创建
可定制的、高性能的、交互式的、基于网络的网络。
- [Dash DataTable Component](https://dash.plotly.com/datatable) - 支持丰富样式的互动桌子，
条件格式、编辑、排序、过滤等。
- [Dash DAQ Components](https://dash.plotly.com/dash-daq) - 一组使数据集成变得更简单的控件
采集和控制到您的 Dash 应用程序中。
- [达世币 HTML 组件](https://github.com/plotly/dash-html-components)
- [mydcc](https://github.com/jimmybow/mydcc) - Dash 核心组件的扩展。
- [sd-material-ui](https://github.com/StratoDem/sd-material-ui) - StratoDem Analytics 针对 Dash 的material-ui 组件的实现。
- [sd-range-slider](https://github.com/StratoDem/sd-range-slider) - 范围滑块 Dash 组件。
- [dj-plotly-dash](https://github.com/pikhovkin/dj-plotly-dash) - Plotly Dash 的 Django 分叉。
- [dash-flexbox-grid](https://github.com/pikhovkin/dash-flexbox-grid) - Plotly Dash 的 React-Flexbox-Grid 包装。
- [dash-color-picker](https://github.com/vivekvs1/dash-color-picker) - 包装反应颜色。
- [dash-dual-listbox](https://github.com/vivekvs1/dash-dual-listbox) - 围绕反应二元论的包装。
- [dash-bootstrap-components](https://dash-bootstrap-components.opensource.faculty.ai/) - Dash 的引导组件。
- [dash-uploader](https://github.com/np-8/dash-uploader) - 上传 Dash 组件。支持大数据文件。
- [Dash Mantine Components](https://github.com/snehilvj/dash-mantine-components) - 基于 Mantine React Components 库的 40 多个 Dash 组件集合。
- [plotly-resampler](https://github.com/predict-idlab/plotly-resampler) - 绘图图形的包装器，添加数据下采样（聚合）功能，从而实现大型数据集的可视化。
- [dash-vega-components](https://github.com/altair-viz/dash-vega-components) - 用于 Vega-Altair、Vega-Lite 和 Vega 图表的 Dash 组件。

## 应用示例

- [Oil and Gas Explorer](https://dash.gallery/dash-oil-and-gas/) - 通过链接可视化探索一段时间内的石油和天然气生产。 [源代码。](https://github.com/plotly/dash-oil-and-gas-demo)
- [Uber Rides](https://dash.gallery/dash-uber-rides-demo/) - 显示 2014 年纽约市的所有 Uber 行程。后端的 Pandas 过滤 0.5gig 的数据文件。 [源代码。](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-uber-rides-demo)
- [Drug Discovery](https://dash.gallery/dash-drug-discovery/) - 将鼠标悬停在图表中的点上时显示药物的描述。 [源代码。](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-drug-discovery)
- [Live Wind Streaming](https://dash.gallery/dash-wind-streaming/) - 持续查询 SQL 数据库并显示风速和风向的实时图表。 [源代码。](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-wind-streaming)
- [3D Yield Curve](https://dash.gallery/dash-yield-curve/) - 改编自《纽约时报》的优秀文章[预测经济未来的 3D 图表视图：收益率曲线](https://www.nytimes.com/interactive/2015/03/19/upshot/3d-yield-curve-economic-growth.html)。 [源代码](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-yield-curve)
- [Visualize Earthquakes with Plotly Dash](https://dash-earthquakes-production-45eyyotfta-ey.a.run.app/) - 出色的 UI 和 Dash 地理空间分析的使用。包括[基本单元测试](https://github.com/jackdbd/dash-earthquakes/tree/master/tests)。 [源代码](https://github.com/jackdbd/dash-earthquakes)
- [GutenSearch](https://gutensearch.com/) - 看看古腾堡计划的书籍。 [源代码](https://github.com/cordb/gutensearch)
