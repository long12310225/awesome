[Reflex](https://reflex.dev) 是一个开源框架，用于完全使用纯 Python 构建全栈 Web 应用程序。
它允许开发人员仅使用一种语言（Python）创建 Web 应用程序的前端和后端。

借助 Reflex，您可以构建从小型数据科学项目到大型多页网站的任何内容，而无需编写任何 HTML、CSS 或 JavaScript。
