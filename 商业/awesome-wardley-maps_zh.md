# 沃德利地图社区中心 [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> [Wardley Mapping](https://en.wikipedia.org/wiki/Wardley_map) 是一种帮助您检查环境、识别即将发生的变化并正确选择操作的技术。通过检查需要什么、将使用哪些组件、它们的依赖性和特征是什么，您可以构建您的世界的视觉表示、玩假设游戏，并选择您的方向和最佳行动来支持它。

这是有用的 Wardley 绘图资源和示例的列表。短网址：[list.wardleymaps.com](https://list.wardleymaps.com)。欢迎投稿！ [贡献指南](https://list.wardleymaps.com/contributing) 用于向列表中添加内容。

## 应用程序

### 测绘

- [OnlineWardleyMaps](https://github.com/damonsk/onlinewardleymaps) - 在线开源地图工具。如需更新，请关注 Twitter 上的 [@MapsAsCode](https://twitter.com/MapsAsCode)。托管版本可用[此处](https://onlinewardleymaps.com/)。在线文档 [Wardley Maps as Code](https://docs.onlinewardleymaps.com/)。
- [Wardley Maps for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=damonsk.vscode-wardley-maps) - 在 Visual Studio Code 中渲染和编辑 Wardley 贴图。此 VSCode 扩展利用 OnlineWardleyMaps.com 的引擎。在线文档 [Wardley Maps as Code](https://docs.onlinewardleymaps.com/)。
- [Wardley Maps for Obsidian.md](https://github.com/damonsk/obsidian-wardley-maps) - 在 Obsidian.md 中渲染和编辑 Wardley 贴图。该社区插件利用 OnlineWardleyMaps.com 的引擎。在线文档 [Wardley Maps as Code](https://docs.onlinewardleymaps.com/)。
- [Wardley Maps for Jupyter Notebooks (ipywardley)](https://github.com/anjackson/ipywardley) - 用于在 Jupyter Notebooks 中生成 Wardley 地图的插件。
- [MapScript](https://observablehq.com/collection/@ajbouh/mapscript) - 注意 MapScript 是基于 Observable 构建的。阅读他们的[服务条款](https://observablehq.com/terms-of-service)。 MapScript [功能请求](https://mapscript.canny.io/feature-requests)。托管版本可用[此处](https://mapscript.org)。
- [Go-Wardley](https://github.com/DavidGamba/go-wardley) - 从基于 HCL 的地图描述文件生成 SVG 格式的 Wardley 地图。
- [Mapkeep](https://mapkeep.com/) - 只需点击一个按钮即可进行多人沃德利地图。与其他人协作绘制地图。免费托管您的地图。如需更新，您可以关注 [Mastodon 上的@mapkeep](https://mastodon.social/@mapkeep)、[X 上的@mapkeep](https://twitter.com/mapkeep)、[LinkedIn 上的mapkeep](https://www.linkedin.com/company/mapkeep)、订阅 [YouTube 频道](https://www.youtube.com/@mapkeep) 或订阅 [新闻和更新]博客](https://blog.mapkeep.com/)。
- [Glamorous Toolkit](https://gtoolkit.com) - 一个可模制和可编程的环境，其中 Wardley 地图可以与知识管理和多语言笔记本交织在一起。关注 [@feenkcom](https://twitter.com/feenkcom) 了解更多信息。
- [Excalidraw](https://excalidraw.com) - 在线图表工具可快速轻松地创建手绘图表和地图。键盘快捷键“0”，浏览库并添加“Wardley Maps Symbols”库。
- [Flowchart fun](https://flowchart.fun) - 基于文本的工具来生成快速流程图。快速创建价值链的绝佳方式 - 在对话或会议期间非常有用。
- [wardleyToGo](https://owulveryck.github.io/wardleyToGo/demo/) 是一个 SDK，用于执行地图即代码 **和** 高级语言 (`wtg`) 来表达地图即数据。该工具集允许将地图轻松集成到 CI 中，并且在线演示允许在线设计地图。
- [Map](https://map.tranquil.systems) - 本机 macOS 应用程序使用自己的“wmap”语法构建 wardley 地图（仅限 Apple Silicon）。 GPLv3。它包括一个 QuickLook 插件，可以将“wmap”文件预览为图像，并共享以协作编辑地图。
- [Map for Linux](https://map.tranquil.systems/map-for-linux) - GTK4 Linux 应用程序使用“wmap”语法构建沃德利地图。 AGPLv3。
- [wmap](https://map.tranquil.systems/cli) - 一个命令行工具，用于将“wmap”格式的地图渲染为 PNG 或 SVG。

### 发展
- [加入 X/Twitter 上的 Wardley AI 开发小组](https://twitter.com/i/communities/1670015410542985218)
- [An Exploration into Wardley Mapping Language Code](https://medium.com/@mcraddock/an-exploration-into-wardley-mapping-language-mode-aa87763a8b17) - 只需两个简单的步骤即可创建您自己的 Wardley Mapping 代码编辑器。
- [Convert WM format to Many](https://wm2many.streamlit.app/) - 一个简单的 Streamlit 应用程序，用于将 OnlineWardleyMaps OWM 格式转换为 JSON、TOML、JSON 图形和密码文本。
- [cli-owm](https://github.com/monkeypants/cli-owm) - 将 OnlineWardleyMaps DSL 文本渲染为 SVG 的命令行工具。通过管道将 stdin 传输到 stdout，以 npm 包和独立二进制文件的形式提供。使用 OnlineWardleyMaps 解析器。
- [Exploring Prompt Engineering using Wardley Maps](https://medium.com/prompt-engineering) - 关于 AI、Python 和 Wardley Mapping 的一系列博客文章。
- [wiki-plugin-wmap](https://www.npmjs.com/package/wiki-plugin-wmap) - [fedwiki](http://fed.wiki.org/view/welcome-visitors) 的插件，使您能够以“wmap”语法编写地图并从组件创建到其他 wiki 页面的链接。
- [tree-sitter-wmap](https://git.sr.ht/~rbdr/tree-sitter-wmap) - 一个“wmap”语法解析器，使用树保姆和捆绑的 neovim 插件，提供语法突出显示。
- [wmap-parser-swift](https://git.sr.ht/~rbdr/wmap-parser-swift) - 一个 Swift 库，使用“wmap”语法解析 wardley 映射，用于 macOS 和 iOS 开发。
- [wmap-parser-js](https://www.npmjs.com/package/wmap-parser) - 沃德利地图的纯 JavaScript 解析器，使用“wmap”语法，可以在浏览器或节点上运行。
- [wmap-parser-rust](https://crates.io/crates/wmap-parser) - 使用“wmap”语法的 wardley 映射的 rust 解析器。
- [wmap-parser-c](https://git.sr.ht/~rbdr/wmap-parser-c) - 使用“wmap”语法的 wardley 映射的高性能 ANSI C 解析器。
- [wmap-renderer-canvas](https://www.npmjs.com/package/wmap-renderer-canvas) - 一个 JavaScript 渲染器，可在使用“wmap”语法的 wardley 地图的 canvas 元素中创建地图，可专门在浏览器或 NodeJS 上使用。
- [wmap-renderer-svg](https://www.npmjs.com/package/wmap-renderer-svg) - 仅限浏览器的渲染器，可为使用“wmap”语法的 wardley 地图创建 SVG。
- [wmap-renderer-rust](https://crates.io/crates/wmap-renderer) 一个 rust 渲染器，可为使用“wmap”语法的 wardley 地图生成开罗表面、SVG 和 PNG。
