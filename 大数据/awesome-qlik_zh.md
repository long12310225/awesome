# 很棒的 Qlik [<img src="https://webapps.qlik.com/WarsawMarathon/qlik.jpg" width="250"align="right" alt="Qlik 徽标">](https://qlik.com) [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> 精彩 [Qlik](https://qlik.com) 指南、博客、扩展和其他资源的精选列表。

Qlik 是一个[领先](https://www.qlik.com/us/gartner-magic-quadrant-business-intelligence) 商业智能平台，提供数据可视化、分析和报告解决方案。运行在同一引擎上，其主要产品是 [QlikView](https://www.qlik.com/us/products/qlikview)（用于引导分析）和 [Qlik Sense](https://www.qlik.com/us/products/qlik-sense)（用于自助 BI）。

## 社区资源和博客

- [Living QlikView](http://livingqlikview.com/) - Aaron Couron 的博客包含深入的文章、技术提示和技巧，以及第三方应用程序和资源的评论，有助于 QlikView 和 Qlik Sense 的进步。
- [Qlik Community](https://community.qlik.com) - Qlik 社区：Qlik 平台上的产品论坛、资源、服务、活动、群组和博客。
- [QlikCentral](https://qlikcentral.com/) - Richard Pearce 在该博客中分享了他在 QlikView 和 Qlik Sense 中开发的见解和技术示例。
- [QlikFix](http://www.qlikfix.com) - Barry Harmsen 专门介绍 QlikView 和 Qlik Sense 提示、技巧和教程的博客。
- [QlikView Cookbook](https://qlikviewcookbook.com/) - Rob Wunderlich 为 QlikView 开发人员提供的技术“操作方法”博客。
- [Qlik Tips](https://www.qliktips.com) - Stephen Redmond 的一个涵盖所有内容的 Qlik 博客。
- [Quick Intelligence](https://www.quickintelligence.co.uk/blog/) - 该博客由 Steve Dark 领导的熟练 Qlik 开发人员团队提供，提供有关使用 Qlik Sense 和 QlikView 的技巧。
- Reddit 子版块：
  - [/r/Qlik](https://www.reddit.com/r/qlik/) - 关于 Qlik 的 Reddit 子版块。
  - [/r/QlikSense](https://www.reddit.com/r/qliksense/) - 关于 Qlik Sense 的 Reddit 子版块。
  - [/r/QlikView](https://www.reddit.com/r/qlikview/) - 关于 QlikView 的 Reddit 子版块。
