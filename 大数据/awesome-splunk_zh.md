# 很棒的 Splunk [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> Splunk 的精彩应用程序、可视化和其他资源的精选列表。

[*Splunk*](https://splunk.com) 以可搜索的方式捕获、索引和关联实时数据
包含图形、报告、警报、仪表板和可视化的存储库
可以生成。广泛应用于金融、公用事业、
医疗保健和制造的用例，包括安全性、合规性和
IT 服务监控。

### 高级应用程序

Splunk 的高级应用程序。

#### 企业安全

Splunk Enterprise Security 是安全生态系统的神经中枢，使团队能够快速检测和响应内部和外部攻击，简化威胁管理，最大限度地降低风险。

- [ES Home Page](https://www.splunk.com/en_us/software/enterprise-security.html) - Splunk 的企业安全主页。
- [ES Splunkbase Entry](https://splunkbase.splunk.com/app/263/) - 下载页面（如果获得许可）。
- [ES Documentation](https://docs.splunk.com/Documentation/ES/latest) - 企业安全性的 Splunk 文档。
- [Awesome-ES](https://github.com/sduff/awesome-es/) - 企业安全所有内容的精彩列表。

#### IT服务智能

Splunk IT 服务智能 (ITSI) 是一种由 IT 运营人工智能 (AIOps) 提供支持的监控和分析解决方案，可提供关键 IT 和业务服务及其基础设施的运行状况和关键性能指标的可见性。

- [ITSI Home Page](https://www.splunk.com/en_us/software/it-service-intelligence.html) - Splunk 的 IT 服务智能主页。
- [ITSI Splunkbase Entry](https://splunkbase.splunk.com/app/1841/) - 下载页面（如果获得许可）。
- [ITSI Documentation](https://docs.splunk.com/Documentation/ITSI/latest) - ITSI 文档。
- [Awesome-ITSI](https://github.com/sduff/awesome-itsi/) - IT 服务智能的所有内容的精彩列表。

## 可视化

- [Event Timeline Viz](https://splunkbase.splunk.com/app/4370/) - 带有事件标注的交互式时间线。
- [Timeline](https://splunkbase.splunk.com/app/3120/) - 互动时间轴。
- [Halo](https://splunkbase.splunk.com/app/3514/) - 分层的关系饼图。
- [Heat Map](https://splunkbase.splunk.com/app/4460/) - 相关测量值的网格，从该值导出颜色强度。
- [Calendar Heat Map](https://splunkbase.splunk.com/app/3162/) - 按天细分的热图。
- [Punchcard](https://splunkbase.splunk.com/app/3129/) - 打孔卡可视化。
- [Horizon Chart](https://splunkbase.splunk.com/app/3117/) - 地平线图表可视化。
- [Sankey Diagram](https://splunkbase.splunk.com/app/3112/) - 桑基图可视化。
- [WebGL Globe](https://splunkbase.splunk.com/app/3674/) - 旋转地球仪，其中包含与地点相关的事件（华而不实的 C 级养眼之物）。
- [Splunkbase Custom Visualizations](https://splunkbase.splunk.com/apps/#/app_content/visualizations) - 从 Splunkbase 下载其他自定义可视化。
