### 灯光软件

- [Avolites Titan](https://www.avolites.com/software/downloads/titan-pc-suite) - 在 Avolites 桌面上运行的 Titan 软件的 PC 版本。尽管它偶尔会输出随机数据，但在 11.4 版本之前均可免费使用。需要 12 版以上的加密狗。“⚠ 不是免费的”。
- [Chamsys MagicQ](https://chamsyslighting.com/) - MagiQ 是一款专业灯光软件，免费提供多达 64 个 Artnet 宇宙。它包括强大的功能，如 2D/3D 像素映射、效果生成器、媒体服务器和可视化工具。
- [Daslight](https://www.daslight.com/) - 适用于 PC 和 Mac 的 DMX 照明软件。
- [Dot2 software](https://www.malighting.com/downloads/products/dot2/) - 适用于 Dot2 控制台的免费离线软件。包括 Dot2 onPC 和 Dot2 3D。
- [EOS Family](https://www.etcconnect.com/Products/Consoles/Eos-Family/) - 适用于 ETC EOS 系列控制台的软件。 `⚠ 免费增值`。
- [Freestyler](http://www.freestylerdmx.be/) - 免费雷电控制软件
- [GrandMA2 software](https://www.malighting.com/downloads/products/grandma2/) - 适用于 GrandMA2 灯光控制台的免费离线软件。包括 GrandMA2 OnPC 和 MA3D。不免费输出DMX。 `⚠ 免费增值`。
- [GrandMA3 software](https://www.malighting.com/downloads/products/grandma3/) - 适用于**新** GrandMA3 灯光控制台的免费离线软件。包括 GrandMA3 OnPC。不免费输出DMX。 `⚠ 免费增值`。
- [Hog 4 software](https://www.highend.com/products/consoles/) - 适用于 Hog 4 灯光控制台的免费离线软件。包括 Hog 4 件。不免费输出DMX。 `⚠ 免费增值`。
- [LightFactory](https://www.lightfactory.net/) - PC 灯光控制软件也是 NEO 控制台的后端。 `⚠不是免费的`。
- [ONYX](https://obsidiancontrol.com/onyx) - 具有很多功能的现代照明控制系统。
- [QLC+](https://www.qlcplus.org/) - QLC+ 是一款免费的跨平台软件，用于控制 DMX 或模拟照明系统，如摇头灯、调光器、扫描仪等。 [源代码](https://github.com/mcallegari/qlcplus) `✓ 开源`。
- [xLights](https://xlights.org/) - 适用于家庭圣诞灯光秀，但也是在预算内将灯光与音乐同步的好方法。 [源代码](https://github.com/smeighan/xLights) `✓ 开源`。

### 展示台

- [Capture](https://www.capture.se/) - 文档和可视化软件。 Capture 是一款原生 macOS 和 Windows 应用程序，可让您处理灯光、视频、激光、移动风景和水效果。有免费的学生版。 `⚠ 免费增值`。
- [Depence²](https://www.syncronorm.com/products/depence2/overview/) - Depence² 是一款独特的应用程序，它将设计和可视化过程与控制真实表演的潜力结合起来。 `⚠不是免费的`。
- [Ma3D](https://www.malighting.com/downloads/products/grandma2/) - **免费** 适用于 GrandMA 控制台/软件的灯光可视化工具。
- [wysiwyg](https://cast-soft.com/wysiwyg-lighting-design/) - wysiwyg 是一款一体化照明设计软件，具有完全集成的 CAD、绘图、数据、可视化和虚拟显示控制。 `⚠不是免费的`。

### 音频软件

- [Audacity](https://www.audacityteam.org/) - 音频文件编辑器和分析工具，非常适合裁剪音效和音乐曲目。 `✓ 开源`。
- [Ardour](https://ardour.org/) - 完整的实时音频录制和编辑套件。 [源代码](https://github.com/Ardour/ardour) `✓ 开源`。
- [da-Share MultiPlay](https://www.da-share.com/software/multiplay/) - MultiPlay 是一个基于 Windows 的程序，旨在播放音频提示以供剧院或企业使用。它可以免费使用。
- [Linux Show Player](https://www.linux-show-player.org/) - 专为舞台制作设计的免费提示播放器。 [源代码](https://github.com/FrancescoCeruti/linux-show-player) `✓ 开源`。
- [QLab](https://qlab.app/) - QLab 是适用于 macOS 的声音、视频和灯光控制。 `⚠ 免费增值`。
- [REAPER](https://www.reaper.fm/) - REAPER 是一款完整的计算机数字音频制作应用程序，提供完整的多轨音频和 MIDI 录音、编辑、处理、混音和母带处理工具集。 `⚠ 免费增值`。

### 音频工具

- [ARTA](http://www.artalabs.hr/) - 用于音频测量和分析的程序集合。
- [REW](https://www.roomeqwizard.com/) - REW 是免费的室内声学分析软件，用于测量和分析室内和扬声器响应。免费的 SMART 替代品。

## 工具

**[`^回到顶部^`](#Contents)**

- [Livescript](https://github.com/Netlob/livescript) - 插入来自 Google Docs 的音乐/戏剧脚本，并将其用于与网站上的每个人进行实时“自动提示”和滚动。 `✓ 开源`。
- [Ontime](https://github.com/cpvalente/ontime) - 基于浏览器的应用程序，用于管理事件清单、日程安排和提示。在一处规划、跟踪您的日程安排、管理自动化和跨部门展会信息。 `✓ 开源`。
- [stagetimer.io](https://stagetimer.io) - 基于浏览器的远程控制倒计时器。  `⚠ 免费增值`。

### 计算机辅助设计

- [AutoCAD](https://www.autodesk.com/products/autocad) - AutoCAD® 是计算机辅助设计 (CAD) 软件，建筑师、工程师和施工专业人员可用来创建精确的 2D 和 3D 绘图。 `⚠不是免费的`。
- [Vectorworks](https://www.vectorworks.net/) - Vectorworks 是一款多功能的本地应用程序，可提供广泛的 2D 绘图、3D 建模、BIM 和渲染功能，满足您的建筑和景观设计需求。还有一个特别的娱乐版本：[Vectorworks Spotlight](https://www.vectorworks.net/en-GB/spotlight)。 `⚠不是免费的`。

### 时间码

时间码是由定时同步系统定期生成的一系列数字代码。时间码用于视频制作、显示控制和其他需要时间协调或记录记录或动作的应用。

- [El-Tee-See](http://elteesee.pehrhovey.net/) - LTC wav 文件生成器。
- [TCgenerator](https://github.com/dimitriCGNL/TCGenerator) - GrandMA2 TC 发生器的 Reaper 时间标记。 `✓ 开源`。
- [Timecode](https://github.com/MrExplode/Timecode) - ArtNet 和 LTC 时间码生成器，用 Java 制作。 `✓ 开源`。
- [Time MIDI sender](https://github.com/TheGreyDiamond/Time-MIDI-sender) - 用于将时间数据发送到 DasLight（通过 MIDI）的免费工具。 `✓ 开源`。
- [TimelordMTC](https://timelord-mtc.com/) - 先进的时间码管理工具。 `⚠不是免费的`。
- [M2Q](https://github.com/lorenzofattori/M2Q) - Python 应用程序能够通过 MIDI 触发 Chamsys 控制台/软件以进行高级同步表演。 `✓ 开源`。

### 监控

- [AES67 WebMeter](https://github.com/zjstraus/AES67-WebMeter) - 接收 AES67 音频流并呈现带有仪表的网页的服务器。 `✓ 开源`。
- [meters.lv2](https://github.com/x42/meters.lv2) - meters.lv2 是带有 LV2 插件格式 GUI 的音频电平表集合。 `✓ 开源`。
- [Open Sound Meter](https://opensoundmeter.com/) - 干净的开源声表。支持 RTA、幅度、相位和脉冲响应、相干性、群延迟等。 `✓ 开源`。

## 杂项/其他

- [Blender ArtNet](https://github.com/BryanCrotaz/blender-artnet) - Blender 脚本将 ArtNet 数据推送到 Evee 灯光。在视口中以 30fps 的速度运行 Evee 渲染。 `✓ 开源`。
