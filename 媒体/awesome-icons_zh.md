## 一般

通用图标随处使用。

- [Bootstrap Icons](https://github.com/twbs/icons#readme) - 为 Bootstrap 组件定制设计和构建的 SVG 图标。 ([网站](https://icons.getbootstrap.com/))
- [Boxicons](https://github.com/atisawd/boxicons#readme) - 简单的矢量图标集，包含 1000 多个图标和 100 多个徽标。 ([网站](https://boxicons.com/))
- [Byte Size Icons](https://github.com/danklammer/bytesize-icons#readme) - 小型风格控制的 SVG 图标集。 （[网站]（https://danklammer.com/bytesize-icons））
- [css.gg](https://github.com/astrit/css.gg#readme) - 纯 CSS、SVG 和 Figma UI 图标可用于 SVG Sprite、样式组件、NPM 和 API。 ([网站](https://css.gg))
- [Eva icons](https://github.com/akveo/eva-icons#readme) - 包含 480 多个制作精美的开源图标。 ([网站](https://akveo.github.io/eva-icons))
- [Evil icons](https://github.com/evil-icons/evil-icons#readme) - 简单干净的 SVG 图标包，包含支持 Rails、Sprockets、Node.js、Gulp、Grunt 和 CDN 的代码。 ([网站](http://evil-icons.io))
- [Feather](https://github.com/feathericons/feather#readme) - 简单漂亮的开源图标。 ([网站](https://feathericons.com))
- Flaticon - 最大的免费矢量图标数据库。 （[网站](https://flaticon.com)）
- [Fluent System Icons](https://github.com/microsoft/fluentui-system-icons#fluent-system-icons) - 来自 Microsoft 的一系列熟悉、友好且现代的图标。
- [Flowbite Icons](https://flowbite.com/icons/) - 免费开源集合，包含 430 多个专为 Tailwind CSS 和 Figma 构建的实体和轮廓图标。
- [Font Awesome](https://github.com/FortAwesome/Font-Awesome#readme) - 标志性的 SVG、字体和 CSS 工具包。 ([网站](https://fontawesome.com))
- [Heroicons](https://github.com/refactoringui/heroicons#readme) - 一组免费的高品质 SVG 图标供您在 Web 项目中使用。 ([网站](https://heroicons.dev))
- [IconPark](https://github.com/bytedance/IconPark#readme) - IconPark 提供超过 1,400 个高质量图标的访问，并引入了用于自定义图标的界面。 ([网站](https://iconpark.bytedance.com))
- ICONSVG - 快速定制 SVG 图标，轻松定制路径的笔划、末端和连接。 ([网站](https://iconsvg.xyz))
- [Ion icons](https://github.com/ionic-team/ionicons#readme) - 适用于 Ionic 框架和各地 Web 应用程序的高级图标字体。 （[网站](https://ionicons.com)）
- [Material design icons](https://github.com/google/material-design-icons#readme) - 谷歌的材料设计图标。 ([网站](https://fonts.google.com/icons))
- [Octicons](https://github.com/primer/octicons#readme) - 由 GitHub 使用 <3 手工制作的一组可扩展图标。 ([网站](https://octicons.github.com))
- [Picon](https://github.com/yne/picon#readme) - 基于连字的小图标字体和 SVG。 （[网站](https://yne.fr/picon)）
- [Radix Icons](https://github.com/radix-ui/icons) - 由 WorkOS 团队设计的一组清晰的 15×15 图标。 ([网站](https://www.radix-ui.com/icons/))
- [Remix Icon](https://github.com/Remix-Design/RemixIcon#readme) - Remix Icon 是一组为设计师和开发人员提供的开源中性风格系统符号。 （[网站](https://remixicon.com)）
- Shape.so - 可定制的动画图标和插图，可导出为代码。 ([网站](https://shape.so))
- [Super Tiny Icons](https://github.com/edent/SuperTinyIcons#readme) - 您最喜欢的网站和应用程序徽标的微型 SVG 版本，每个版本不到 1KB。
- [Tabler Icons](https://github.com/tabler/tabler-icons) - 一组超过 3200 个免费的 MIT 许可的高质量 SVG 图标，供您在 Web 项目中使用。 ([网站](https://tabler-icons.io))
- [Teenyicons](https://github.com/teenyicons/teenyicons) - 1000 多个 MIT 许可的 SVG 图标，可轻松适应非常小的空间 :pinching_hand: 并保持清晰的外观 :sparkles:。 ([网站](https://teenyicons.com))
- 名词项目 - 一切的图标，超过 200 万个精选图标，由全球社区创建。 ([网站](https://thenounproject.com/))
- Tilda Icons - Tilda Icons 是一个充满登陆页面等免费图标的网站。 ([网站](https://tilda.cc/free-icons))
- [Unicons](https://github.com/iconscout/unicons#readme) - 1000 多个像素完美的矢量图标和图标字体，适合您的下一个项目。 ([网站](https://iconscout.com/unicons))
- [VSCode Icons](https://github.com/microsoft/vscode-icons#readme) - Visual Studio Code 中使用的图标的深色和浅色版本。
- [Weather icons](https://github.com/erikflowers/weather-icons#readme) - 天气主题图标和 CSS。
- Zondicons - 一组免费的高级 SVG 图标，供您在数字产品上使用。 ([网站](http://www.zondicons.com))

## 特别

图标不是通用的或徽标，而是特殊的东西。

- [Flag Kit](https://github.com/madebybowtie/FlagKit#readme) - 漂亮的旗帜图标，可在应用程序和网络上使用。
- [Mapsicon](https://github.com/djaiss/mapsicon#readme) - 世界上每个国家的免费地图集。
- [IconicFonts](https://github.com/iconicFonts/iconic-fonts#readme) - 超过 50 种预先修补的字体，具有 60k 图标作为字形。

## 其他

### 仅CSS

仅由 CSS 制作的图标（不可下载）。

- [cssicon](https://github.com/wentin/cssicon#readme) - 用纯css代码制作的图标集，无依赖，“抓取即走”图标。 ([网站](https://cssicon.space))
- [icono](https://github.com/saeedalipoor/icono#readme) - 纯CSS，基于类的图标包。 ([网站](https://saeedalipoor.github.io/icono))

### 付费

不是开源/免费的，但仍然足够棒。

- [Nucelo Icons](https://nucleoapp.com/premium-icons) - 适用于 iOS、Android 和 Web 项目的高级 SVG 图标库。

### 框架

通常是一个允许您导入其他图标集的库或框架。

- Iconbuddy - 浏览开源 svg 图标的搜索引擎。 ([网站](http://iconbuddy.com))
- [Iconify Deign](https://github.com/iconify/iconify#readme) - 一个框架中最流行的图标集包含超过 200,000 个开源矢量图标。 ([网站](https://iconify.design))
- [React Icons](https://github.com/react-icons/react-icons#reamde) - 专为 React 项目创建的流行图标集。 ([网站](https://react-icons.github.io/react-icons/))
- [Unicon](https://github.com/WebRenew/unicon#readme) - 从 9 个库中浏览 20,000 多个图标，并仅复制您需要的图标，例如 shadcn/ui 图标。 ([网站](https://unicon.webrenew.com))
