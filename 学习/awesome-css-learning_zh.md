## CSS 参考

- [codrops](https://tympanus.net/codrops/css_reference/) - 内容广泛的 CSS 参考，提供比 [MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference) 更多的内容。
- [Can I use](https://caniuse.com) - CSS（和 HTML5）的交互式浏览器支持表。

## CSS 简而言之

- [Introduction to CSS](https://scrimba.com/g/gintrotocss) - 该截屏视频系列将在大约一小时内教您 CSS 基础知识。

## 基本概念

- [The cascade](https://developer.mozilla.org/en-US/docs/Web/CSS/Cascade) - 本文解释了什么是级联以及它如何影响您。
- [Specificity and inheritance](https://www.smashingmagazine.com/2010/04/css-specificity-and-inheritance/) - 了解特殊性和继承对于掌握 CSS 非常重要。这篇文章会有所帮助。
- [CSS Box Model](https://developer.mozilla.org/en-US/docs/Learn/CSS/Introduction_to_CSS/Box_model) - 一篇解释网络布局基础的文章。
- 另请查看有关 [box-sizing](https://css-tricks.com/box-sizing/) 属性的详细信息。
- [Responsive Design](https://web.dev/learn/design/) - 关于响应式设计各个方面的广泛信息，使网站看起来很棒并且适合每个人。

## CSS 单位

- [The Lengths of CSS](https://css-tricks.com/the-lengths-of-css/) - 涵盖绝对和相对单位的概述。
- [Fun with Viewport Units](https://css-tricks.com/fun-viewport-units/) - 传授基础知识并展示一些漂亮的用例。

## 选择器

- [Basic CSS Selectors](https://www.sitepoint.com/css-selectors/) - 介绍您需要了解的非常基本的 CSS 选择器。
- [Advanced CSS Selectors](https://www.smashingmagazine.com/2009/08/taming-advanced-css-selectors/) - 提升你的知识水平。从属性选择器到 CSS3 伪类。
- [CSS Diner](https://flukeout.github.io) - 通过这个有趣的小游戏了解如何使用 CSS 选择器。

## 自定义属性（又名 CSS 变量）

- [CSS Variables: Why Should You Care?](https://developers.google.com/web/updates/2016/02/css-variables-why-should-you-care) - CSS 变量的简短介绍。
- [Locally Scoped CSS Variables: What, How, and Why](https://una.im/local-css-vars/) - 描述局部作用域 CSS 变量的优点。
- [Using CSS variables correctly](https://www.madebymike.com.au/writing/using-css-variables/) - 使用 CSS 变量的模式和反模式。
- [Everything you need to know about CSS Variables](https://medium.freecodecamp.org/everything-you-need-to-know-about-css-variables-c74d922ea855) - 深入的文章使用现实世界的示例超越了 CSS 变量的基础知识。
- [Getting Reactive with CSS](https://www.youtube.com/watch?v=4IRPxCMAIfA) - 令人兴奋的谈论 CSS 变量和 JavaScript 中函数式反应式编程相结合的可能性。

## 布局

- [Learn CSS Layout](http://book.mixu.net/css) - 通过 5 章了解 CSS 布局技术。
- [Layout Land](https://www.youtube.com/channel/UC7TizprGknbDalbHplROtag) - Jen Simmons 关于新的 CSS 布局可能性的视频系列。
- [Laying Out The Future With Grid And Flexbox](https://www.youtube.com/watch?v=hj355PRbwSQ) - 引入了新的布局系统，包括 Flexbox、CSS Grid 和 Box Alignment Module。

### 经典布局

- [Floats](https://tympanus.net/codrops/css_reference/float/) - 有关如何使用（和清除）浮动的深入信息。
- [Positioning Types](https://scotch.io/bar-talk/5-things-you-might-not-know-about-the-css-positioning-types) - 仔细看看CSS定位布局方法相关的几个鲜为人知的东西。
- [inline-block](https://iamsteve.me/blog/entry/inline_block) - 展示在哪些情况下使用显示属性“inline-block”进行布局是有意义的。

### 弹性盒

### 网格

- [A Complete Guide to Grid](https://css-tricks.com/snippets/css/complete-guide-grid/) - 您需要了解的有关 CSS 网格布局的所有信息都在一页上。
- [Grid by Example](https://gridbyexample.com) - 除了如何使用网格的示例之外，该站点还提供其他有用的学习资源。
- [Designing with Grid](https://talks.jensimmons.com/J5VRbA/designing-with-grid) - 讨论 CSS 网格提供的新布局可能性。
- [Grid Garden](https://cssgridgarden.com) - 可爱的游戏，你可以在其中编写 CSS 代码来种植你的胡萝卜花园。
- [GridBugs](https://github.com/rachelandrew/gridbugs) - 社区策划的网格互操作问题及其解决方法列表。
- [Grid Critters](https://www.gridcritters.com) - 通过掌握冒险游戏来学习 CSS 网格布局。
- [CSS Grid](https://cssgrid.io) - 和 Wes Bos 一起学习 CSS 网格 - 免费 4 小时视频课程，25 个视频。
