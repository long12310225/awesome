## 艺术与设计

- [Canon Explains Exposure](http://www.canonoutsideofauto.ca/) - 了解如何为您的照片提供适量的光线。
- [Can't Unsee](https://cantunsee.space/) - 在这款游戏中，您对细节的关注可以为您赢得大量金币。
- [Font Memory Game](https://betterwebtype.com/font-memory-game) - 识别不同风格的字体是排版中最具挑战性的部分之一。
- [Method of Action](https://method.ac/) - 教您设计原理的游戏集合。
  - [The Bézier Game](https://bezier.method.ac/) - 贝塞尔曲线
  - [Color](https://color.method.ac/) - 颜色
  - [Shape Type](https://shape.method.ac/) - 字母塑造
  - [Kern Type](https://type.method.ac/) - 字距调整
- [Pixactly](http://pixact.ly/) - 像素测试来测试您的知识。
- [The Boolean Game](https://boolean.method.ac/) - 了解如何在 Sketch、Figma 和其他矢量编辑器中使用布尔运算。

## CSS

- [CSS Diner](https://flukeout.github.io/) - 这是一个帮助你学习 CSS 选择器的小游戏。
- [CSS Grid Garden](http://cssgridgarden.com/) - 一个学习 CSS 网格布局的游戏。
- [Flexbox Defense](http://www.flexboxdefense.com/) - 与其他塔防游戏不同，您必须使用 CSS 来定位您的塔！
- [Flexbox Froggy](http://flexboxfroggy.com/) - 一个学习 CSS flexbox 的游戏。
- [Flexbox with Webflow](https://www.flexboxgame.com/) - 在 28 个交互式级别中学习和掌握 Flexbox，无需编写代码。

## 编辑

- [Open Vim](https://www.openvim.com/) - 交互式 Vim 教程。
- [Vim Adventures](https://vim-adventures.com/) - 边玩游戏边学习 Vim。
- [Vim Genius](http://vimgenius.com/) - 使用 Vim Genius 提高速度并提高肌肉记忆。

## 语言

- [Drops](https://languagedrops.com/) - 通过有趣的游戏，以您选择的语言学习有用的单词和短语。
- [Duolingo](https://www.duolingo.com/) - 免费学习西班牙语、法语和其他语言。

## 数学与计算机科学

- [Alligator Eggs!](http://worrydream.com/#!/AlligatorEggs) - 一款教授 lambda 演算的儿童纸笔游戏。
- [CS Unplugged](https://csunplugged.org/en/) - 不使用计算机教授计算机科学概念的一系列课堂活动。
- [Finitris](http://www.postcrashgames.com/finitris/) - 查看运行正则表达式会发生什么。
- [Guess The Correlation](http://guessthecorrelation.com/) - 尝试猜测散点图中两个变量的相关性如何。

## 音乐

- [Learning Music](https://learningmusic.ableton.com) - 通过 Ableton 的互动网站探索音乐的基础知识。
- [Learning Synths](https://learningsynths.ableton.com/) - 了解使用合成器（或合成器）的基础知识。
- [Pianu](https://pianu.com) - 了解如何在浏览器中弹钢琴。

## 编程

- [BOX-256](http://box-256.com/) - 使用汇编程序对虚构的 CPU 进行编程以输出图形。
- [Bashcrawl](https://gitlab.com/slackermedia/bashcrawl) - 通过玩简单的文本冒险来学习 Linux 命令。
- [Clips the Game](https://md5crypt.github.io/clipsgame/) - 一款基于剪辑规则引擎的高逻辑游戏。
- [Code Avengers](https://www.codeavengers.com/) - 学习以有趣且有效的方式构建网站、应用程序和游戏。
- [Code School](https://www.pluralsight.com/codeschool ) - 学习在舒适的浏览器中编写代码。
- [Code.org](https://code.org/) - 每所学校的每个学生都应该有机会学习计算机科学。
- [Codecademy](https://www.codecademy.com/) - 免费学习交互式编码。
- [CodeCombat](https://codecombat.com/) - 学习编程最吸引人的编码游戏。
- [CodinGame](https://www.codingame.com/start) - 一个基于挑战的培训平台，您可以在其中探讨最热门的编程主题。
- [Cube Composer](https://david-peter.de/cube-composer/) - 一款受函数式编程启发的益智游戏。
- [Deadlock Empire](https://deadlockempire.github.io/) - 用并发解决难题。
- [Free Code Camp](https://www.freecodecamp.org/) - 帮助您学习编码的开源社区。
- [Learn Git Branching](https://learngitbranching.js.org/) - 强调分支的交互式 Git 教程。
- [Mimo](https://getmimo.com/) - 了解如何成为移动开发人员。
- [Programming Games Wiki](http://programminggames.org/) - 多年来编程游戏的长期目录。
- [RegexOne](https://regexone.com/lesson/introduction_abcs) - 通过简单的交互式练习来学习正则表达式。
- [ReturnTrue](https://alf.nu/ReturnTrue) - 这是一个学习布尔值的游戏，返回 true 即可获胜。
- [Screeps](https://screeps.com/) - 全球首款面向程序员的MMO策略沙盒游戏。
- [SQL Murder Mystery](https://mystery.knightlab.com/) - 在练习数据库技能的同时解决犯罪问题。
- [Swift Playgrounds](https://www.apple.com/swift/playgrounds/) - 以一种非常有趣的方式学习在 iPad 上编码。
- [Untrusted](https://alexnisnevich.github.io/untrusted/) - 在玩 JavaScript 冒险游戏的同时学习。
- [WarriorJS](https://github.com/olistic/warriorjs) - 使用 JavaScript，对战士进行编程，使其爬上高塔、对抗怪物并营救俘虏。
- [Web Audio School](https://mmckegg.github.io/web-audio-school/) - 交互式网络音频教程。
- [XSS game](https://xss-game.appspot.com) - 一款诱骗人们在浏览器中运行代码的游戏。
