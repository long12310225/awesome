## 平台

_awesome🕶️黑客马拉松平台_，我们指的是专门为运行黑客马拉松而设计的网络或移动应用程序，或者已有效适应在此类活动中使用的网络或移动应用程序。它们通常允许组织者宣布日程和主题、注册参与者并记录结果。

### 开源🌳

> 这些可以与云提供商一起运行或自行托管，以完全控制数据。作为开源项目，它们通常采用开放标准。其中一些是由非营利组织和志愿者社区开发的。

- [Civic Tech Exchange](https://github.com/DemocracyLab/CivicTechExchange) - 民主实验室项目的在线平台。
- [DeltaHacks Portal](https://github.com/deltahacks/portal) - DeltaHacks 10（变革黑客马拉松）的与会者和管理门户。
- [Dribdat](https://github.com/dribdat/dribdat) - **基于开放数据和网络标准的影响力黑客马拉松。** `python` `js`
- [Hack the Back](https://github.com/hackthevalley/hack-the-back) - 优雅的后端，具有现成的 GraphQL 和 REST 端点。
- [Hackathon Management System](https://github.com/Teplitsa/hackathon) - [Teplitsa](https://hms.te-st.org/) 的综合 WordPress 插件。 `php`
- [Hackerspace3](https://github.com/govhackaustralia/hackerspace3) - GovHack 黑客马拉松的竞赛管理系统。 `红宝石`
- [Halo](https://github.com/tcnj-acm/halo) - 基于 Django 构建的黑客马拉松管理系统。
- [Hibiscus](https://github.com/HackSC/hibiscus) - 专为 HackSC 创建的一体化、即插即用的黑客马拉松平台。
- [JunctionApp](https://github.com/hackjunction/JunctionApp) - 为组织者提供的一体化黑客马拉松平台，由 Junction 维护。
- [OpenHackathon](https://github.com/kaiyuanshe/OpenHackathon-Web) - 平台具有基于 Next.js 的基于 Git 的云开发环境。
- [Opportunity Hack](https://github.com/opportunity-hack) - 基于技能的社会公益志愿服务，将人员与项目相匹配。
