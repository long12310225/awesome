# 很棒的工作人员 [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)
*或者，很棒的冒险*

针对各种事物的基于研讨会/冒险的教程列表。灵感来自真棒。这些是基于终端的学习新编程概念的指南。以下是 rvagg 的 [learnyounode](https://github.com/workshopper/learnyounode) 的屏幕截图示例：

![Learn You The Node.js For Much Win!](https://raw.github.com/rvagg/learnyounode/master/learnyounode.png)

如果您知道任何其他研讨会/冒险教程，请随意分叉/PR 或打开一个新问题。此列表也将作为类型的愿望列表：如果您想要此处未列出的研讨会人员，请打开拉取请求，我将了解如何启动它。

### 贡献指南
如果添加新源，请确保您的拉取请求遵循以下准则：

* 请为每个建议提出单独的拉取请求（或 Github 问题）。
* 使用以下格式：`| [资源]（链接）|描述 |`
* 保持描述简短。通常，存储库的描述就足够了。
* 按字母顺序对每个类别中的项目进行排序。
* 欢迎新类别或对现有类别的改进。
