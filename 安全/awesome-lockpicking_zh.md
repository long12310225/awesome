# 竞赛、会议和活动

* [Chaos Communications Congress (CCC)](https://www.ccc.de/) - CCC 主办德国开锁锦标赛。
* [DEF CON](https://defcon.org/) - 一年一度的信息安全行业会议，受到许多开锁者的欢迎，其中包括开锁比赛。
* [Hackers On Planet Earth (HOPE)](https://hope.net/) - 年度会议通常以船闸运动比赛和有关物理安全的演讲为特色。
* [OzSecCon](https://ozseccon.com/) - 澳大利亚专注于人身安全的开放社区活动，举办撬锁和逃避手铐比赛。
* [LockCon](https://toool.nl/LockCon) - 一年一度的、仅限受邀人士参加的关于锁具的国际会议，会议对象为受人尊敬的船闸运动组织的成员。

# 兴趣爱好项目

* [3D Printed Lock Picks](http://blog.shop.23b.org/2014/11/3d-printed-lock-picks.html) - 打印由塑料材料制成的开锁工具，虽然很脆弱，但至少可以使用多次。 （[查找更多数字模型](https://www.yeggi.com/q/lockpick/)。）
* [DIY Lock Pick Set from a Windshield Wiper](http://www.itstactical.com/skillcom/lock-picking/how-to-make-a-diy-lock-pick-set-from-a-windshield-wiper/) - 使用挡风玻璃雨刷器制作定制撬锁器和拉力扳手的图解指南。
* [How to Make a Set of Lockpicks! (using street sweeper bristles)](https://www.instructables.com/id/How-to-Make-a-set-of-Lockpicks/) - 使用废弃街道清扫机刷毛作为双面车削工具和镐材料的说明。
* [Making Tension Wrenches/Turning Tools From Old Windshield Wiper Blades](https://www.instructables.com/id/Making-Tension-WrenchesTurning-Tools-From-Old-Wind/) - 将旧挡风玻璃雨刷器改装成各种开锁器类型和车削工具类型的说明。
* [Under Door Tool Kit Instructions](https://web.archive.org/web/20170719053154/http://enterthecore.net/files/CORE_Instructions-Under_Door.pdf) - 使用典型五金店提供的材料制作门下（“杠杆打开”/“Mule”）工具的简短指南。
* [Lock Dice](http://schuylertowne.com/blog/lockpicking-with-dice) - 在这款基于骰子的锁具运动游戏中，作为一名崭露头角的锁具制造商，您已经开始与同行测试您的设计。

# 开锁方法指南

* [Art of Lockpicking](https://art-of-lockpicking.com/) - 带有动画的教育网站，详细介绍了撬锁的过程。
* [Beginner's Guide to Security Pins](https://web.archive.org/web/20171210065243/http://www.ninjacache.com:80/secpins_intro) - 简短的文章指导初学者轻松地撬开挂锁，以了解更复杂的锁。
* [BosnianBill's Lock Picking for Beginners](https://www.youtube.com/playlist?list=PLTSWkYxuSlkXiSBwk3Hvbvx71sg-MH61s) - 精选教学视频的 YouTube 播放列表按从基本原理到特定锁具技术的顺序排序。
* [CIA Lock Picking Field Operative Training Manual](https://archive.org/details/CIA_Lock_Picking_Field_Operative_Training_Manual) - 指南涵盖了大多数弹子锁和圆片弹子锁的术语、工具使用和撬锁技术。
* [Deviant Ollam's Lockpicking Video Presentations and Advisories](http://deviating.net/lockpicking/videos.html) - 来自世界知名专家的针对所有技能水平的物理安全讲座和船闸运动建议的集合。
* [Easy Pickings - Mini Lockpicking Manual](http://index-of.es/Lockpicking/Easy%20Pickings%20-%20Mini%20Lockpicking%20Manual.pdf) - 广泛分发的初学者手册，其中的图表展示了如何撬开各种入门级锁。
* [MIT Lock Picking Guide](https://webunraveling.com/public/mit-lock-picking-guide/index.php) - 工具 Ted 的著名介绍性文本。
* [Lock Picking: Detail Overkill](https://web.archive.org/web/20170730120626/http://ninjacache.com/data/uploads/lockpicking-detail-overkill.pdf) - “对于那些想要的不仅仅是基础知识的人”来说，这是一份非常详尽的指南。
* [LockLab's Lock Picking Course](https://locklab.com/locklab-university/lock-picking-course-2/) - 从基本张紧到专业锁定技术的完整课程。
* [Locksport International Guide to Lock Picking](https://web.archive.org/web/20070222144748/http://locksport.com:80/LSIGuide/lsiguide.pdf) - 漫画书风格的Locksport入门读物，带有照片和对话气泡。

# 锁业协会和组织

* [Fraternal Order of Lock Sport (FOOLS)](http://www.bloomingtonfools.org/) - 开锁俱乐部成立，为印第安纳州布卢明顿及周边地区提供服务。
* [The Open Organization Of Lockpickers (TOOOL)](https://toool.org/) - 全球开锁者协会，设有国家、地区和地方分会。
* [Locksport International](http://locksport.com) - 国际开锁者协会，为创办新的开锁运动俱乐部和团体提供帮助。
* [Sportsfreunde der Sperrtechnik - Deutschland e.V.](https://blog.ssdev.org/) - 德国开锁运动俱乐部，主办德国开锁大师赛，并提供如何开锁的研讨会。

* [Southern Specialities](http://www.lockpicktools.com/) - 零售商，出售多种开锁套件、撬枪等。
* [Sparrows Lock Picks](https://www.sparrowslockpicks.com/) - 专业优质开锁工具制造商；他们的 [King Pin](http://www.sparrowslockpicks.com/product_p/t1.htm) 固定镊子特别有用。
