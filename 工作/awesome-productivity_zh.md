## 工具和应用程序

### 文件管理

- [Box](https://www.box.com) - 对于企业客户来说非常好的云存储解决方案。
- [Dropbox](https://www.dropbox.com) - 简单、优雅、多功能的云存储解决方案。
- [Google Drive](https://www.google.ca/drive/) - 云存储解决方案深度融入Google生态系统。
- [Mega](https://mega.nz/) - 一个非常慷慨（从一开始就有 50G）且高度安全的云存储解决方案。
- [Nextcloud](https://nextcloud.com) - 自托管生产力平台让您掌控一切。
- [OneDrive](https://onedrive.live.com) - 适合 Windows 用户的最佳云存储解决方案。
- [pCloud](https://www.pcloud.com/) - 高度关注安全性的云存储解决方案。
- [Clipboard](https://getclipboard.app) - 易于使用的本地解决方案，用于像便签本一样保存文件、文件夹或文本。

### 笔记管理

- [Evernote](https://evernote.com/) - Evernote 基本上可以成为你的第二大脑，为你记住一切。
- [Google Keep](https://www.google.com/keep/) - 一个漂亮而简单的笔记管理系统，与 Google 产品紧密集成。
- [Inkdrop](https://www.inkdrop.info/) - 专为 Markdown 爱好者打造的跨平台笔记应用程序。
- [Joplin](https://joplinapp.org/) - 具有同步功能的笔记和待办事项应用程序。
- [Notion](https://www.notion.so/) - 一个高度可定制的一体化工作区，用于笔记、任务、wiki 和列表。
- [Yana](https://yana.js.org) - 功能强大的笔记应用程序，具有嵌套文档、全文搜索、富文本编辑器、代码片段编辑器等。
- [OneNote](https://www.onenote.com/) - 非常好的笔记管理解决方案，但最适合 Windows 用户。
- [Preserver](https://github.com/hsbalar/preserver) - 基于 Electron 的笔记管理器桌面应用程序。
- [Simplenote](https://simplenote.com/) - 轻盈、干净、自由。 Simplenote 现已适用于 iOS、Android、Mac、Windows、Linux 和 Web。
- [Standard Notes](https://standardnotes.org/) - 一个简单且私人的笔记应用程序。
- [WorkFlowy](https://workflowy.com/) - 概述嵌套项目符号列表的应用程序。

### 任务管理

- [Airtable](https://airtable.com/) - 电子表格和数据库的混合有助于组织工作。
- [Amazing Marvin](https://www.amazingmarvin.com/) - 马文结合了行为心理学的原理来帮助克服拖延症、感觉一切尽在掌握并完成待办事项清单。
- [Any.do](https://www.any.do/) - 简单的界面，丰富的功能，目前是 [Lifehacker](https://lifehacker.com/5924093/ Five-best-to-do-list-managers) 上最受欢迎的待办事项列表管理器。
- [GitHub Projects](https://github.com/features/project-management/) - GitHub 的一个鲜为人知的功能可以让您轻松地将项目管理流程与代码联系起来。
- [Hitask](https://hitask.com) - 团队的轻松项目和任务管理。
- [KanbanFlow](https://kanbanflow.com) - 采用番茄工作法的看板方法任务板。
- [Microsoft To Do](https://todo.microsoft.com/tasks/) - 基于 Wunderlist 技术的 Microsoft 云任务管理应用程序。
- [OmniFocus](https://www.omnigroup.com/omnifocus) - 适用于 macOS 和 iOS 的基于 Getting Things Done 的任务管理器。
- [Paymo](https://www.paymoapp.com/) - 让您的团队保持一致和参与。从同一平台管理任务、创建团队日程、跟踪工作时间以及向客户开具账单。
- [Pomolectron](https://github.com/amitmerchant1990/pomolectron) - 适用于菜单栏/托盘的番茄工作法应用程序。
- [Remember the Milk](https://www.rememberthemilk.com) - 擅长管理待办事项和基于位置的任务的标签。
- [Taskade](https://taskade.com) - 团队的简单协作任务列表。
- [Taskwarrior](https://taskwarrior.org/) - 一个开源命令行任务管理器。灵活、快速、高效且不引人注目。
- [Timely](https://memory.ai/timely) - 自动时间跟踪软件。获得所有项目和团队时间的完美数字记录，无需手动计时器和记笔记的麻烦。
- [Timing](https://timingapp.com/) - Mac 的自动时间和生产力跟踪。帮助您保持工作正常进行，并确保按小时计费时不会浪费计费时间。
- [Tinido](https://tinido.com/) - 一款美观而简单的任务管理应用程序，还可以通过在类似于 GitHub 个人资料中找到的贡献图表中显示已完成的任务来为您提供见解。
- [Todo.txt](http://todotxt.com/) - 厌倦了使用复杂的应用程序来组织您的任务？ Todo.txt 是一种使用单个“.txt”文件来帮助您完成工作的技术。
- [Todoist](https://todoist.com/) - 强大的跨平台生产力工具。
- [Topydo](https://github.com/topydo/topydo) - 一个强大的待办事项列表应用程序，使用 todo.txt 格式，位于 todo.txt CLI 和 Taskwarrior 之间。
- [Trello](https://trello.com) - 看板方法任务板。
- [Tweek](https://tweek.so) - 超级简单的待办事项列表和每周任务管理。
- [VivifyScrum](https://www.vivifyscrum.com) - 适合交付团队的敏捷项目管理应用程序。可定制的 Scrum 和看板。
- [Bordio](https://bordio.com/) - 用于在一个板上管理任务和事件的每日计划程序。
- [OpenProject](https://www.openproject.org/) - 一个本地开源项目管理解决方案，附带免费社区版本和企业版本。

### 习惯追踪器

- [Emoji Log](https://emojilog.rosano.ca) - 平静的、基于表情符号的跟踪器，用于不使用条纹的任意事物。
- [Everyday](https://everyday.app/) - 通过每天做事来养成新习惯。
- [Habitica](https://habitica.com) - 在角色扮演游戏（以前称为_HabitRPG_）中完成现实生活中的日常目标。
- [HabitBull](https://www.habitbull.com/) - 高级习惯追踪器。
- [Loop Habit Tracker](https://github.com/iSoron/uhabits) - 一款用于养成和保持良好习惯的 Android 应用。
- [Waka Time](https://wakatime.com/) - 开源插件，用于准确了解您花费多长时间进行编码。
- [Conjure](https://conjure.so) - 习惯、目标和时间跟踪平台，具有规则引擎、数据层、API、仪表板等（Web、桌面、iOS、Android）。

### 任务自动化

- [AutoIt](https://www.autoitscript.com/) - 用于 Windows 的小工具，用于自动化 Windows GUI，但它也可用于其他目的。
- [Awesome AutoHotKey](https://github.com/ahkscript/awesome-AutoHotkey) - AutoHotKey 是一款适用于 Windows 的免费开源宏创建和自动化软件，允许用户自动执行重复任务。
- [IFTTT](https://ifttt.com) - 允许您在 Web 服务之间创建条件语句链（称为_recipes_），以使 Web 为您服务并提高您的工作效率。
- [n8n](https://n8n.io) - 用于连接 Web 服务并具有自托管选项的开源工作流程自动化。
- [Qbserve](https://qotoqot.com/qbserve/) - Mac 时间跟踪自动化：自由项目跟踪、时间表、发票和实时生产力反馈。
- [Parabola](https://parabola.io) - 拖放工具可构建自定义报告、工作流程和集成，以自动化您的手动流程。
- [Tasker](https://tasker.dinglisch.net/) - Android 应用程序，可以执行上下文敏感的自定义任务（例如，当您到家时自动打开 wifi）。
- [Zapier](https://zapier.com/) - 自动化工具，允许您连接数百个 Web 服务并在流程之间创建自动化。

### 密码管理器

- [Bitwarden](https://bitwarden.com) - 一款免费、值得信赖的密码管理器，可以在线或通过本机应用程序存储、同步和生成密码。
- [KeePass](https://keepass.info/) - 一款免费、开源、轻量级的密码管理器，支持自动输入、浏览器和插件。
- [LastPass](https://lastpass.com) - 在线存储加密密码的免费增值密码管理器。
- [NordPass](https://nordpass.com/) - 由 NordVPN 团队创建的免费增值密码管理器。存储无限的密码，并通过所有平台上的应用程序从任何地方访问它们。

### 知识管理

- [Obsidian.md](https://obsidian.md/) - 适用于本地 Markdown 文件的知识库工具。它允许您在不同笔记之间创建链接。
- [Scribe](https://scribehow.com/) - 自动为任何流程创建分步指南。只需点击“记录”，Scribe 就会根据您的操作生成详细的指南，并附有屏幕截图，可供与同事、客户和朋友分享。
- [Logseq](https://logseq.com/) - Logseq 是一个隐私优先的开源知识库，可在本地纯文本 Markdown 和 Org 模式文件之上运行。用它来书写、组织和分享您的想法，保留您的待办事项列表，并建立您自己的数字花园。

### 屏幕截图

- [ShareX](https://getsharex.com/) - 屏幕捕获、文件共享和生产力工具。
- [Replayable](https://replayable.io) - 快退屏幕。回放并分享 Replayable 的桌面重放缓冲区到底发生了什么。
- [CleanShot](https://cleanshot.com/) - MacOS 上的屏幕捕获和录制高级工具，具有内置注释和文本识别功能。
