# Google 上的精彩操作  [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

精彩的 AoG（Action on Google）库、博客、开源项目、视频的精选列表。

借助 [Actions on Google](https://developers.google.com/actions/)，您可以轻松接触 Google 上的用户并与之互动。从快速命令到完整对话，操作可帮助您使用 Google 助理将您的内容和服务与用户联系起来。

## 开源代码
- [Canvas Quiz](https://github.com/googlecreativelab/aog-canvas-quiz) - 它是一个入门套件，可供开发人员为 Google Assistant 制作自定义的、支持语音的问答游戏。
- [Interactive Canvas editor](https://github.com/actions-on-google-labs/interactive-canvas-editor-nodejs) - 编辑器，用于为交互式画布创建可立即加载到智能显示设备上的 Web 应用程序。这允许快速构建交互式画布的 Web 应用程序原型。
- [Virtual Pet](https://github.com/actions-on-google-labs/virtual-pet-game-nodejs) - 它是用交互式画布构建的。在这款游戏中，您将像在现实生活中一样收养并照顾仓鼠，只不过是通过 Google Assistant。
- [Spellcast](https://github.com/actions-on-google-labs/dialogflow-spellcast-nodejs) - 适用于 Google Assistant 的实验性交互式画布游戏。
- Silly Name Maker [(Node.js)](https://github.com/actions-on-google/dialogflow-silly-name-maker-webhook-nodejs) [(Java)](https://github.com/actions-on-google/dialogflow-silly-name-maker-webhook-java) - 参数示例（使用 Dialogflow）。
- 有关 Google 的事实 [(Node.js)](https://github.com/actions-on-google/dialogflow-facts-about-google-nodejs) [(Java)](https://github.com/actions-on-google/dialogflow-silly-name-maker-webhook-java) - 自定义实体、上下文和深层链接示例（使用 Dialogflow）。
- 名称 Psychic [(Node.js)](https://github.com/actions-on-google/dialogflow-name-psychic-nodejs) [(Java)](https://github.com/actions-on-google/dialogflow-name-psychic-java) - 权限和 Google 地图静态 API 示例（使用 Dialogflow）。
- Number Genie [(Node.js)](https://github.com/actions-on-google/dialogflow-number-genie-nodejs) [(Java)](https://github.com/actions-on-google/dialogflow-number-genie-java) - 本地化示例。
- [GDG Action](https://github.com/actions-on-google/dialogflow-gdg-nodejs) - 此示例演示了在 Google Assistant 上使用的 Actions on Google 功能，包括本地化（i18n 节点）、对话设计、丰富的响应和 API 集成。
- 说出数字 [(Node.js)](https://github.com/actions-on-google/actionssdk-say-number-nodejs) [(Java)](https://github.com/actions-on-google/actionssdk-say-number-java) - 说出 Actions on Google 的数字 Actions SDK 示例。
- [Smart Home sample](https://github.com/actions-on-google/smart-home-nodejs) - Actions on Google 中的智能家居设备控制 API 示例。
- [PDPU Scheduler](https://github.com/nish17/scheduler) - 这是 PDPU 的 Google Assistant 操作，可以告诉您今天有哪些课程和实验室。
- [karaoke](https://github.com/amdcaruso/karaoke) - [Amanda Cavallaro](https://github.com/amdcaruso) - 使用 GCP、YouTube API 和 Actions on Google 构建您自己的卡拉 OK 的对话式应用程序。
