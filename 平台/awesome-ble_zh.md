# 很棒的 BLE [![Awesome](https://awesome.re/badge-flat.svg)](https://awesome.re)

![](.github/awesome-ble.png)

**蓝牙低功耗 (BLE)**：低功耗无线通信协议可实现设备之间的高效数据交换，非常适合物联网、可穿戴设备和其他电池供电应用。

[🔴intent](https://withintent.com) 提供的很棒的低功耗蓝牙资源协作列表。

## 文档

_有关蓝牙低功耗 (BLE) 协议的官方规范、标准和技术文档。_

- [Android Bluetooth Low Energy](https://developer.android.com/guide/topics/connectivity/bluetooth/ble-overview) - Android 中的 BLE 概述。
- [Developing a Bluetooth Low Energy Application by TI](https://software-dl.ti.com/lprf/simplelink_cc2640r2_sdk/1.35.00.33/exports/docs/ble5stack/ble_user_guide/html/ble-stack/index.html) - TI 提供的有关如何开发 BLE 应用程序的指南。
- [Getting Started with Bluetooth Low Energy (BLE)](https://learn.adafruit.com/introduction-to-bluetooth-low-energy) - 低功耗蓝牙开发入门综合指南。
- [Intro to Bluetooth Generic Attribute Profile (GATT)](https://www.bluetooth.com/bluetooth-resources/intro-to-bluetooth-gap-gatt/) - 关贸总协定简介。
- [Intro to Bluetooth Low Energy](https://www.bluetooth.com/bluetooth-resources/intro-to-bluetooth-low-energy/) - BLE 基础知识介绍。
- [IoT for Beginners](https://microsoft.github.io/IoT-For-Beginners/#/) - Microsoft 的 Azure 云倡导者很高兴提供为期 12 周、24 课时的课程，全部涉及物联网基础知识。
- [nRF51 SDK](https://www.nordicsemi.com/Software-and-tools/Software/nRF5-SDK) - 软件开发套件 (SDK)，用于在 Nordic Semiconductor 的 nRF51 系列芯片上开发蓝牙低功耗应用。
- [RN4870/71 Bluetooth Low Energy Module User's Guide](https://www.microchip.com/wwwproducts/en/RN4870) - Microchip RN4870/71 蓝牙低功耗模块的用户指南。
- [Wikipedia Bluetooth Low Energy](https://en.wikipedia.org/wiki/Bluetooth_Low_Energy) - 低功耗蓝牙 (BLE) 的 wiki 页面。

## 图书馆

_用于将蓝牙低功耗 (BLE) 功能集成到各种平台（包括移动、嵌入式系统和物联网设备）的软件库和框架。_

- [bluepy](https://github.com/IanHarvey/bluepy) - Linux 上蓝牙 LE 的 Python 接口。
- [Bluetooth](https://github.com/PureSwift/Bluetooth) - 蓝牙是一个 Swift 多平台包，具有多个支持蓝牙 SIG 规范的库。
- [bluez](http://www.bluez.org/) - 官方 Linux 蓝牙协议栈。
- [bumble](https://github.com/google/bumble) - Bumble 是一个完全用 Python 编写的全功能蓝牙堆栈。
- [pygatt](https://github.com/peplin/pygatt) - 用于与 BLE 设备交互的 Python 库。
- [SimpleBLE](https://github.com/OpenBluetoothToolbox/SimpleBLE) - 适用于 MacOS、iOS、Windows 和 Linux 的一体化 C/C++/Python/Rust 蓝牙 LE 库。
- [Moddable](https://github.com/Moddable-OpenSource/moddable) - 用于微控制器开发的 JavaScript 库和工具。包括 BLE 功能。

## iOS系统

_用于在 Apple 设备上开发蓝牙低功耗 (BLE) 应用程序的 iOS 库、工具和资源。_

- [AsyncBluetooth](https://github.com/manolofdez/AsyncBluetooth) - 一个小型库，为 CoreBluetooth API 添加并发性。
- [CombineCoreBluetooth](https://github.com/StarryInternet/CombineCoreBluetooth) - CombineCoreBluetooth 是一个桥接 Apple 的 CoreBluetooth 框架和 Apple 的 Joint 框架的库。
- [iOS DFU Library](https://github.com/NordicSemiconductor/IOS-DFU-Library) - DFU 应用程序允许在 DK 上刷新新固件。
- [Rx Bluetooth Kit](https://github.com/Polidea/RxBluetoothKit) - RxBluetoothKit 是一个蓝牙库，使用 RxSwift 可以使与 BLE 设备的交互变得更加愉快。
- [Swifty Sensors](https://github.com/codeinversion/sensors-swift) - 适用于 iOS 和 macOS 的蓝牙 LE 传感器管理器。
- [SpeziBluetooth](https://github.com/StanfordSpezi/SpeziBluetooth) - 使用现代编程范例与蓝牙设备连接和通信。

## 韩国MM

_使用 Kotlin 多平台移动 (KMM) 框架构建蓝牙低功耗 (BLE) 应用程序的资源和库。_

- [BlueFalcon](https://github.com/Reedyuk/blue-falcon) - 适用于 iOS 和 Android 的蓝牙 kotlin 多平台“跨平台”库。
- [Kable](https://github.com/JuulLabs/kable) - Kotlin 异步低功耗蓝牙提供了一个简单的协程驱动的 API，用于与低功耗蓝牙设备交互。

## 安卓

_用于将蓝牙低功耗 (BLE) 功能集成到 Android 应用程序中的 Android 库、工具和资源。_

- [Android BluetoothGatt Sample](https://github.com/android/connectivity-samples/tree/main/BluetoothLeGatt) - 演示如何在 Android 上使用 BluetoothGatt API 的示例应用程序。
- [android-BluetoothChat](https://github.com/googlesamples/android-BluetoothChat) - 一款Android蓝牙聊天应用程序。
- [FastBle](https://github.com/Jasonchenlijian/FastBle) - Android 低功耗蓝牙 (BLE) 快速开发框架。
- [Nordic Semiconductor BLE SDK for Android](https://github.com/NordicSemiconductor/Android-BLE-Library) - Nordic Semiconductor 为 Android 开发提供官方低功耗蓝牙 SDK。
- [RxAndroidBle](https://github.com/dariuszseweryn/RxAndroidBle) - 具有 RxJava3 接口的 Android 低功耗蓝牙 (BLE) 库。
- [SmartGattLib](https://github.com/movisens/SmartGattLib) - SmartGattLib 是一个 Java 库，可简化蓝牙 SMART 设备（又称蓝牙 4.0 中的蓝牙低功耗）的工作。它提供了采用的 GATT 规范的所有 UUID 以及解释特征（例如心率、电池电量）的便捷方法。

## 工具

_低功耗蓝牙 (BLE) 应用的开发和测试工具，包括分析器、调试器和模拟器。_

- [Bettercap](https://github.com/bettercap/bettercap) - 一个易于扩展和移植的框架，供安全研究人员测试 BLE 安全性。
- [nRF Connect](https://github.com/NordicSemiconductor/pc-nrfconnect-core) - 用于开发、测试和配置基于 Nordic Semiconductor 芯片的设备的应用程序框架。
- [BLE Simulator](https://play.google.com/store/apps/details?id=com.withintent.ble.simulator&hl=en) - 一款能够克隆 BLE 设备并编写其交互脚本的移动应用程序。对于调试和快速原型设计很有用。
- [PacketLogger](https://www.bluetooth.com/blog/a-new-way-to-debug-iosbluetooth-applications/) - 一款 macOS 工具，用于从 iOS 设备捕获 BLE 数据包和 HCI 日志，可用于调试蓝牙通信和协议问题。
