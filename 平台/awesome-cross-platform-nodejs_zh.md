# Awesome 跨平台 Node.js [![Awesome](https://awesome.re/badge.svg)](https://github.com/sindresorhus/awesome)

在 Node.js 上编写跨平台代码。

## 目录
- [平台检测](#平台检测)
- [文件系统](#文件系统)
- [路径](#路径)

## 平台检测
- [os](https://nodejs.org/api/os.html) - 内置模块。
- [is-windows](https://github.com/sindresorhus/is-windows) - Windows 检测。
- [is-linux](https://github.com/sindresorhus/is-linux) - Linux 检测。
- [is-macos](https://github.com/sindresorhus/is-macos) - macOS 检测。
- [cross-platform](https://github.com/sindresorhus/cross-platform) - 判断。

## 文件系统
- [fs-extra](https://github.com/jprichardson/node-fs-extra) - 文件操作。
- [graceful-fs](https://github.com/isaacs/node-graceful-fs) - 优雅文件系统。
- [fs-jetpack](https://github.com/szwacz/fs-jetpack) - API 封装。
- [del](https://github.com/sindresorhus/del) - 删除文件。
- [make-dir](https://github.com/sindresorhus/make-dir) - 创建目录。
- [copy-file](https://github.com/sindresorhus/copy-file) - 复制文件。
- [move-file](https://github.com/sindresorhus/move-file) - 移动文件。
- [write-file-atomic](https://github.com/npm/write-file-atomic) - 原子写入。
- [read-chunk](https://github.com/sindresorhus/read-chunk) - 读取块。

## 路径
- [path](https://nodejs.org/api/path.html) - 内置模块。
- [pathe](https://github.com/unjs/pathe) - 规范化路径。
