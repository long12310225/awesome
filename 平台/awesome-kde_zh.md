# 很棒的 KDE [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

[<img src="media/logo-kde-grey.svg"align="right"width="144">](http://www.kde.org)

> 适用于 KDE 桌面环境的_精彩_应用程序、扩展、模块、主题和工具的精选列表。

[KDE](https://kde.org/) 是一个免费软件社区，致力于创建开放且用户友好的计算体验，提供先进的图形桌面、用于通信、工作、教育和娱乐的各种应用程序以及轻松构建新应用程序的平台。

> 标有 📌（图钉）的项目是在 KDE 自己的基础设施上托管并由 KDE 社区开发的项目。

## 应用领域

提示：[官方网站](https://apps.kde.org/) 上列出了最常用的 KDE 社区应用程序的概述。

### 互联网与网络

- [Akregator](https://apps.kde.org/akregator/) - 使您能够关注新闻网站、博客和其他支持 RSS/Atom 的网站 📌。
- [Falkon](https://www.falkon.org/) - 基于 QtWebEngine 的跨平台网络浏览器📌。
- [Firefox KDE](https://build.opensuse.org/package/show/mozilla:Factory/MozillaFirefox) - 来自 mozilla.org 的独立 Web 浏览器，带有 OpenSUSE 补丁，可以更好地与 KDE 集成。
- [GSConnect](https://extensions.gnome.org/extension/1319/gsconnect/) - KDE Con​​nect 的完整实现，特别是针对 GNOME Shell，与 Nautilus、Chrome 和 Firefox 集成。
- [KDE Connect](https://kdeconnect.kde.org/) - 一个跨所有设备进行通信的项目：在桌面计算机上接收手机通知、通过手机控制桌面等📌。
- [KGet](https://apps.kde.org/kget/) - 多功能且用户友好的下载管理器📌。
- [KMail](https://apps.kde.org/kmail2/) - KMail 是 Kontact 的电子邮件组件，Kontact 是 KDE 📌 的集成个人信息管理器。
- [KNetworkManager](https://userbase.kde.org/KNetworkManager) - 后端 NetworkManager 软件的 KDE 客户端。
- [Konqueror](https://konqueror.org/) - 使用 KHTML 渲染引擎（以及可选的 Webkit），因此它支持最新的 Web 标准，例如 HTML5、JavaScript、CSS3 等。
- [Konversation](https://konversation.kde.org/) - 基于 KDE 平台 📌 构建的用户友好的互联网中继聊天 (IRC) 客户端。
- [KRDC](https://apps.kde.org/krdc/) - 客户端应用程序，允许您查看或控制运行兼容服务器（VNC 和 RDP）的另一台计算机上的桌面会话。
- [KTorrent](https://apps.kde.org/ktorrent/) - 一个 BitTorrent 应用程序，允许您使用 BitTorrent 协议下载文件📌。
- [NeoChat](https://apps.kde.org/neochat/) - KDE 📌 的 Matrix 网络客户端。
- [Otter Browser](https://otter-browser.org/) - Otter Browser 旨在使用 Qt5 重新创建经典 Opera (12.x) UI 的最佳方面。
- [Plasma-browser-integration](https://userbase.kde.org/Plasma-browser-integration) - 一系列新功能使 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/plasma-integration/)、[Chrome](https://chrome.google.com/webstore/detail/plasma-integration/cimiefiiaegbelhefglklhhakcgmhkai) 和基于 Chromium 的浏览器可与您的桌面 📌 配合使用。
- [qBittorrent](https://www.qbittorrent.org/) - 一个用 C++ 编写的高级 BitTorrent 客户端，基于 Qt 工具包和 libtorrent-rasterbar。
- [Qt WebBrowser](http://doc.qt.io/QtWebBrowser/) - Qt WebBrowser 是使用 Qt 和 Qt WebEngine 的功能开发的嵌入式设备浏览器。
- [Smb4K](https://apps.kde.org/en/smb4k) - 高级网上邻居浏览器和 Samba 共享安装实用程序📌。
- [Tokodon](https://apps.kde.org/pt-br/tokodon/) - Tokodon 是 Mastodon 的客户。它允许您与 Fediverse 社区互动 📌。

### 图形与成像

- [digiKam](https://www.digikam.org/) - 借助开源的力量进行专业照片管理📌。
- [Gwenview](https://apps.kde.org/gwenview/) - KDE 📌 快速且易于使用的图像查看器。
- [KColorChooser](https://apps.kde.org/kcolorchooser/) - 一个简单的应用程序，用于从屏幕或调色板中选择颜色📌。
- [KolourPaint](https://apps.kde.org/kolourpaint/) - 一个简单的绘画程序，可快速创建光栅图像。它作为修饰工具和简单的图像编辑任务很有用📌。
- [Krita](https://apps.kde.org/krita/) - 专为概念艺术家、插画家、哑光和纹理艺术家以及视觉特效行业📌设计的绘画工具。
- [Okular](https://apps.kde.org/okular/) - 通用文档查看器📌。
- [Spectacle](https://apps.kde.org/spectacle/) - 用于捕获桌面屏幕截图的应用程序📌。

### 多媒体

- [Amarok](https://amarok.kde.org/) - Amarok 是一款适用于 Linux、Unix 和 Windows 的强大音乐播放器，具有直观的界面📌。
- [Elisa](https://elisa.kde.org/) - 由 KDE 社区开发的音乐播放器，力求简单易用📌。
- [Haruna](https://invent.kde.org/multimedia/haruna) - 使用 Qt/QML 和 libmpv 构建的开源视频播放器。
- [Kdenlive](https://www.kdenlive.org/) - 使用 MLT 视频框架的 Linux 非线性视频编辑器。
- [KMix](https://apps.kde.org/kmix/) - 声道混音器和音量控制📌。
- [KWave](https://apps.kde.org/kwave/) - 基于 KDE Frameworks 5 📌 构建的声音编辑器。
- [SMPlayer](https://www.smplayer.info/) - 具有内置编解码器的媒体播放器，几乎可以播放所有视频和音频格式。
- [VLC](https://www.videolan.org/vlc/) - 一个免费开源的跨平台多媒体播放器和框架，可播放大多数多媒体文件和各种流媒体协议。

### 办公与生产力

- [Calligra](https://www.calligra.org/) - 包括高效且功能强大的办公组件📌。
- [GhostWriter](https://ghostwriter.kde.org/) - 没有干扰的 Markdown 编辑器📌。
- [Kate](https://apps.kde.org/kate/) - KDE 📌 的多文档、多视图文本编辑器。
- [Kronometer](https://apps.kde.org/kronometer/) KDE 📌 的秒表应用程序。
- [KWrite](https://apps.kde.org/kwrite/) - KDE 的文本编辑器，基于 KDE 框架编辑器组件📌。
- [RSI Break](https://apps.kde.org/rsibreak/) - RSIBreak 可以用来提醒您时不时休息一下📌。
- [Kraft](https://volle-kraft-voraus.de/) - 卡夫简化了小型企业中发票等文档的处理。

### 系统管理

- [Discover](https://apps.kde.org/discover/) - 适用于终端不是安装应用程序和附加组件或更新的最佳解决方案的用户和发行版。
- [Dolphin](https://apps.kde.org/dolphin/) - KDE 桌面中包含文件管理器📌。
- [KDiskFree](https://apps.kde.org/kdf/) - 显示可用的文件设备（硬盘驱动器分区、驱动器等）及其容量、可用空间、类型和安装点📌信息。
- [KInfoCenter](https://userbase.kde.org/KInfoCenter) - 提供有关计算机系统📌详细信息的实用程序。
- [Konsole](https://apps.kde.org/konsole/) - KDE 桌面中包含终端模拟器 📌。
- [Krusader](https://krusader.org/) - 适用于 KDE 📌 的高级双面板（指挥官风格）文件管理器。
- [KSystemLog](https://apps.kde.org/ksystemlog/) - 系统日志查看工具📌。
- [Kup](https://apps.kde.org/kup/) - KDE Plasma 桌面的备份调度程序 📌。
- [Sweeper](https://apps.kde.org/sweeper/) - Sweeper 有助于清除用户在系统上留下的不需要的痕迹，并通过删除未使用的临时文件来重新获得磁盘空间。
- [Yakuake](https://apps.kde.org/yakuake/) - 基于 KDE 的 Konsole 技术的下拉式终端模拟器 📌。

### 实用程序和工具

- [Android File Transfer](https://whoozle.github.io/android-file-transfer-linux/) - 可靠的 MTP 客户端，具有简约的 UI。
- [Ark](https://apps.kde.org/ark/) - 存档工具📌。
- [dictee](https://github.com/rcspam/dictee) - 使用 KDE Plasma 6 plasmoid、PyQt6 设置向导和托盘图标进行一键通语音听写。 4 个 ASR 后端（Parakeet、Vosk、Whisper、Canary）、后处理管道、翻译。 100% 本地化，超过 25 种语言。
- [fancontrol-gui](https://github.com/Maldela/fancontrol-gui) - 用于风扇控制的 GUI，它是 lm_sensors 的一部分。
- [KDE Wallet Manager](https://invent.kde.org/utilities/kwalletmanager/) - 管理 KDE 系统上的密码。 KDE 钱包子系统允许对与 KDE 钱包集成的每个应用程序进行访问和密码管理。
- [KeePassXC](https://keepassxc.org/) - Keepass 密码管理器的跨平台社区驱动端口。
- [KFind](https://apps.kde.org/kfind/) - 一个独立的搜索工具，由 KRunner 启动或从您的菜单启动。它还作为“工具”菜单中的“查找文件”集成到 Konqueror 中。
- [KRename](https://apps.kde.org/krename/) - 一个重命名多个文件的实用程序📌。
- [KWin-lowlatency](https://github.com/tildearrow/kwin-lowlatency) - 尝试减少 KDE 中使用的流行 KWin 合成器中的延迟和卡顿。
- [KWinFT](https://gitlab.com/kwinft/kwinft) - Wayland 合成器和 X11 窗口管理器。
- [LocalTranslate](https://github.com/terslang/LocalTranslate) - 使用 Qt Quick 构建的离线翻译应用程序，利用 Firefox 的[神经翻译模型](https://github.com/mozilla/firefox-translations-models)。
- [QOwnNotes](https://www.qownnotes.org/) - 具有云集成、网络和移动支持的笔记应用程序。
- [QtPass](https://qtpass.org/) - [pass](https://www.passwordstore.org/) 的多平台 GUI。
- [Wacom GUI](https://github.com/tb2097/wacom-gui) - 适用于 KDE 的 Python/PyQt Wacom GUI。
- [Yin-Yang](https://github.com/daehruoydeef/Yin-Yang) - Linux 的自动夜间模式。

### 开发与编程

- [KDevelop](https://www.kdevelop.org/) - 适用于 C、C++、Python、QML/JavaScript 和 PHP 的跨平台 IDE。
- [Kompare](https://apps.kde.org/kompare/) - 差异/补丁前端📌。
- [Okteta](https://apps.kde.org/okteta/) - 十六进制编辑器📌。
- [Qt Designer](http://doc.qt.io/qt-5/qtdesigner-manual.html) - 用于使用 Qt Widget 设计和构建图形用户界面 (GUI) 的 Qt 工具。
- [Umbrello](https://apps.kde.org/umbrello/) - UML Modeller 是一个基于 KDE 技术📌的统一建模语言（UML）图程序。

### 教育
