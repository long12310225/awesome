## 插件

- [GEmojiSharp](https://github.com/hlaueriksson/GEmojiSharp#gemojisharppowertoysrun) - 查找并复制 GitHub 表情符号。
- [Everything](https://github.com/lin-ycv/EverythingPowerToys) - 使用 Everything 即时查找文件和文件夹。
- [PowerTranslator](https://github.com/N0I0C0K/PowerTranslator) - 使用有道翻译翻译文本。
- [WinGet](https://github.com/bostrot/PowerToysRunPluginWinget) - 使用 WinGet 安装和管理 Windows 软件包。
- [Lorem](https://github.com/dxn-9/prun-lorem) - 生成 lorem ipsum 文本。
- [TOTP](https://github.com/KawaiiZapic/PowertoysRunTOTP) - 复制基于时间的两因素验证码。
- [Scoop](https://github.com/Quriz/PowerToysRunScoop) - 使用 Scoop 安装和管理 Windows 软件包。
- [InputTyper](https://github.com/CoreyHayward/PowerToys-Run-InputTyper) - 输入文本，就像从键盘发送一样。
- [CurrencyConverter](https://github.com/Advaith3600/PowerToys-Run-Currency-Converter) - 使用最新汇率转换传统货币和加密货币。
- [EdgeFavorite](https://github.com/davidegiacometti/PowerToys-Run-EdgeFavorite) - 打开 Microsoft Edge 收藏夹。
- [GitKraken](https://github.com/davidegiacometti/PowerToys-Run-GitKraken) - 打开 GitKraken 存储库。
- [VisualStudio](https://github.com/davidegiacometti/PowerToys-Run-VisualStudio) - 在 Visual Studio 中打开最近的解决方案。
- [ClipboardManager](https://github.com/CoreyHayward/PowerToys-Run-ClipboardManager) - 从剪贴板历史记录中搜索并粘贴。
- [GitHubRepo](https://github.com/8LWXpg/PowerToysRun-GitHubRepo) - 打开 GitHub 存储库。
- [ProcessKiller](https://github.com/8LWXpg/PowerToysRun-ProcessKiller) - 终止 Windows 进程。
- [WebSearchShortcut](https://github.com/Daydreamer-riri/PowerToys-Run-WebSearchShortcut) - 使用预定义的搜索引擎进行搜索。
- [UnicodeInput](https://github.com/nathancartlidge/powertoys-run-unicode) - 使用 Agda 风格的简写查找并复制 Unicode 字符。
- [Timer](https://github.com/CoreyHayward/PowerToys-Run-Timer) - 设置和管理计时器。
- [HexInspector](https://github.com/NaroZeol/PowerHexInspector) - 在基数之间转换数字。
- [LocalLLM](https://github.com/Darkdriller/PowerToys-Run-LocalLLm) - 使用 Ollama 查询本地 LLM 模型。
- [JohnnyDecimal](https://github.com/seguri/PowerToys-Run-JohnnyDecimal) - 浏览您的 JohnnyDecimal 系统。
- [BrowserFavorite](https://github.com/Der-Penz/PowerToys-Run-BrowserFavorite) - 快速打开浏览器书签。
- [Universal Search Suggestions](https://github.com/Fefedu973/PowerToys-Run-Universal-Search-Suggestions-Plugin/) - 键入内容时添加搜索建议。<!--lint 禁用双链接-->
- [Bang](https://github.com/hlaueriksson/Community.PowerToys.Run.Plugins#bang) - 使用 DuckDuckGo !Bangs 搜索网站。
- [DenCode](https://github.com/hlaueriksson/Community.PowerToys.Run.Plugins#dencode) - 编码和解码。
- [Dice](https://github.com/hlaueriksson/Community.PowerToys.Run.Plugins#dice) - 角色扮演骰子滚子。
- [Need](https://github.com/hlaueriksson/Community.PowerToys.Run.Plugins#need) - 重要信息的键值存储。
- [Twitch](https://github.com/hlaueriksson/Community.PowerToys.Run.Plugins#twitch) - 在 Twitch 上浏览、搜索和查看流。<!--lint 启用双链接-->
- [SVGL](https://github.com/SameerJS6/powertoys-svgl) - 通过 svgl 浏览、搜索和复制 SVG 徽标。
- [QuickNotes](https://github.com/ruslanlap/CommunityPowerToysRunPlugin-QuickNotes) - 创建、管理和搜索笔记。
- [SpeedTest](https://github.com/ruslanlap/PowerToysRun-SpeedTest) - 测试您的互联网连接速度。
- [VideoDownloader](https://github.com/ruslanlap/PowerToysRun-VideoDownloader) - 从各个平台下载视频。
- [Definition](https://github.com/ruslanlap/PowerToysRun-Definition) - 查找单词定义、语音和同义词。
- [RandomGen](https://github.com/ruslanlap/PowerToysRun-RandomGen) - 生成各种类型的随机数据。
- [Hotkeys](https://github.com/ruslanlap/PowerToysRun-Hotkeys) - 查找并复制各种应用程序的键盘快捷键。
- [TemplateRunner](https://github.com/Heck-R/PowerToys.Run.Plugin.TemplateRunner) - 使用简单的命令和脚本作为迷你插件。
