## 什么是 Awesome Salesforce？
Awesome 是由 [Sindre Sorhus](https://github.com/sindresorhus) 发起的很棒的项目，他是 [Github](https://github.com/sindresorhus) 上最受欢迎的人。他创建了一个项目来列出机器中添加书签的最酷的链接，并通过在一个地方编译它们来将它们组织在一起。后来，这个项目火了，世界各地的开发者社区开始编写很棒的页面（致力于他们最喜欢的技术）。

# 关于 Repo AWSomeness 的说明

列出的每个存储库至少满足以下要求之一：

* 社区创作的存储库，拥有 100 多颗星
* 社区担保的仓库，星数 < 100

社区仓库的 100+ 星并不是严格要求，它只是作为初始编译的指导方针。  如果您可以保证一个 < 100 颗星的存储库的出色性，并且您可以解释为什么应该列出它，请提交拉取请求。

* [JSForce - JSforce (f.k.a. Node-Salesforce)](https://jsforce.github.io/) - 它将对 Salesforce 提供的各种 API 的访问封装在异步 JavaScript 函数调用中。与其他 Salesforce API 库不同，它旨在为服务器端和客户端应用程序提供集成接口，因此您不必仅为了在不同环境中运行而使用不同的库重写类似的逻辑。此外，它具有有用的命令行界面 (CLI)，它提供了交互式控制台 (REPL)，因此您可以轻松学习用法。
