## 存储库

- [Uno Platform](https://github.com/unoplatform/uno) - Uno 平台的主要存储库，包括 UWP/WinUI 的完整 API 界面，是提交问题和做出贡献的最佳场所。
- [Uno Themes](https://github.com/unoplatform/Uno.Themes) - 适用于 Uno 平台应用程序的 Material 和 Cupertino 主题库。此外，还包括基于 Android/iOS 开发人员指南的高级控件。
- [Uno Toolkit](https://github.com/unoplatform/uno.toolkit.ui) - 多平台优先的 UI 组件库可加速应用程序开发。还为所包含的组件提供 Material 和 Cupertino 样式。
- [Uno Gallery](https://github.com/unoplatform/Uno.Gallery) - 控件和 API 库应用程序。包括大多数内置控件和功能的实时示例以及易于使用的代码片段。
- [Uno Samples](https://github.com/unoplatform/Uno.Samples) - 不断增长的代码示例和片段集合。
- [Uno Core](https://github.com/unoplatform/Uno.Core) - 提供一组可重用的实用程序和帮助程序。由主要 Uno 平台存储库使用。
- [Uno UI Test](https://github.com/unoplatform/Uno.UITest) - 适用于 Android、iOS 和 WebAssembly 的跨平台 UI 测试库。

## 工具
- [Uno Platform extension for Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=unoplatform.uno-platform-addin-2022) - 向 Visual Studio 2022 添加 Uno Platform 项目模板和热重载功能。
- [Uno Platform extension for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=unoplatform.vscode) - 为 Visual Studio Code 添加了一系列生产力增强功能，包括 XAML IntelliSense、热重载和移动目标调试支持。
- [Uno Extensions](https://github.com/unoplatform/uno.extensions) - 一系列 NuGet 包，用于与使用 Uno 平台构建多平台应用程序相关的常用功能。
- [Figma 插件的 Uno 平台](https://www.figma.com/community/plugin/1045528009520465828/Uno-Platform-(Figma-to-XAML)) - Figma 插件，用于从高保真设计生成响应式 XAML。
- [Uno Material Toolkit](https://www.figma.com/community/file/1110792522046146058/Uno-Platform-Material-Toolkit) - Figma 文件与 Uno Figma 插件一起使用。

## 图书馆

### 工具包
- [Windows Community Toolkit for Uno](https://github.com/unoplatform/Uno.WindowsCommunityToolkit) - Microsoft Community Toolkit 的 Uno Platform 分支。
- [Telerik UI for Uno](https://github.com/unoplatform/Uno.Telerik.UI-For-UWP) - 用于 UWP 的 Telerik UI 分叉以支持 Uno 平台。

### MVVM
- [MVx.Observable](https://www.nuget.org/packages/MVx.Observable/) - 一个（大部分）不固执的、轻量级的 ReactiveUI 替代品，作为库（而不是框架）提供。
- [Prism](https://prismlibrary.com/) - 用于构建松散耦合、模块化、可维护和可测试的 XAML 应用程序的框架。
- [Microsoft MVVM Toolkit](https://docs.microsoft.com/en-us/windows/communitytoolkit/mvvm/introduction) - 现代、快速、模块化的 MVVM 库。

### 兼容性
- [SignalR](https://docs.microsoft.com/en-us/aspnet/signalr/overview/getting-started/introduction-to-signalr) - [与 Uno 和 SignalR 的跨平台实时通信](https://ian.bebbs.co.uk/posts/UnoChat) - SignalR 框架的集成示例。
- [Elmish.Uno](https://github.com/xperiandri/Elmish.Uno) - 用于用 F# 编写的应用程序的 Half-Elmish 框架。
