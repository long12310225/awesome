## API开发

* [Hoppscotch](https://hoppscotch.com/) - 测试具有实时协作功能的 API。 [![开源软件][oss]](https://github.com/hoppscotch/hoppscotch)
* [HTTP Toolkit](https://httptoolkit.tech) - 拦截并修改 HTTP 流量。 [![开源软件][oss]](https://github.com/httptoolkit)
* [Insomnia](https://insomnia.rest) - 使用 GraphQL 支持测试 API。 ![开源软件](/assets/opensource.svg)
* [Postman](https://www.getpostman.com/postman) - 使用自动化测试脚本和监控来测试 API。
* [Yaak](https://yaak.app) - 基于 Tauri 的现代轻量级 API 测试应用程序。 [![开源软件][oss]](https://github.com/mountain-loop/yaak)

## 应用程序启动器

* [Flow Launcher](https://flowlauncher.com/) - 搜索文件并启动具有插件扩展性的应用程序。 [![开源软件][oss]](https://github.com/Flow-Launcher/Flow.Launcher) ![star]
* [Keypirinha](https://keypirinha.com/) - 通过模糊搜索和键盘快捷键启动应用程序。
* [kunkun](https://github.com/kunkunsh/kunkun) - 类似 Raycast/Alfred 的启动器，带有沙盒扩展！[oss]
* [Listary](https://www.listary.com/) - 使用高级过滤器和索引搜索文件。
* [Raycast](https://www.raycast.com/) - 高度可扩展、键盘驱动的生产力启动器和命令面板，专为提高效率而设计
* [ueli](https://ueli.app/#/) - 强大的按键启动器。 [![开源软件][oss]](https://github.com/oliverschwendener/ueli)
* [Wox](http://wox-launcher.github.io/Wox/) - 启动具有网络搜索集成的应用程序。 [![开源软件][oss]](https://github.com/Wox-launcher/Wox/)
* [Velocmd](https://yashvardhang.github.io/Velocmd/) - 快如闪电的本机系统启动器。搜索文件、运行命令并无缝导航您的系统。 [![开源软件][oss]](https://github.com/YashvardhanG/Velocmd)

* [Shotcut](https://www.shotcut.org/download/) - 具有本机时间线过滤功能的视频编辑器。 [![开源软件][oss]](https://github.com/mltframework/shotcut)
