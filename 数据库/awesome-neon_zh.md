# 很棒的霓虹灯 [![Awesome](https://awesome.re/badge-flat.svg)](https://awesome.re)

[Neon](https://neon.tech) 是一个很棒的无服务器 Postgres 数据库，是您喜爱的 AWS Aurora 的开源替代品。它是一个无服务器平台，旨在帮助您更快地构建可靠且可扩展的应用程序。
