# 很棒的 TypeDB [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

TypeDB 资源的精选列表。

TypeDB 是一个[多态](https://typedb.com/features#polymorphic-queries)数据库，具有[概念](https://typedb.com/features#conceptual-modeling)数据模型、强大的[子类型](https://typedb.com/features#strong-type-system)系统、符号[推理](https://typedb.com/features#symbolic-reasoning)引擎以及美丽优雅的[类型理论](https://typedb.com/features#modern-language) 语言 TypeQL。

## 使用 TypeDB 的开源项目

- [`TypeDB Studio - IDE for TypeDB`](https://github.com/vaticle/typedb-studio) - TypeDB Studio 是 TypeDB 的集成开发环境，用于执行知识工程。
- [`Blitz-ORM`](https://github.com/Blitzapps/blitz-orm) - Blitz-ORM 是一个用 TypeScript 编写并与 TypeDB 兼容的对象关系映射器。
- [`go-typeql`](https://github.com/CaliLuke/go-typeql) - 适用于 TypeDB 3.x 的 Go ORM，具有类型安全的 CRUD、查询构建、迁移和代码生成。
- [`skills`](https://github.com/CaliLuke/skills) - 代理技能的集合，包括用于代理工作流程的 TypeDB 技能。
- [`TypeDB-ML`](https://github.com/vaticle/typedb-ml) - TypeDB-ML 提供了使用 TypeDB 实现图算法和机器学习的工具。
- [`TypeDB Loader`](https://github.com/typedb-osi/typedb-loader) - 一个开源数据迁移器，用于将数据大规模加载到 TypeDB 中。
- [`TypeDB OSI - Cyber Threat Intelligence`](https://github.com/typedb-osi/typedb-cti) - TypeDB CTI 是一种开源威胁情报平台，使威胁情报专业人员能够将不同的 CTI 信息整合到一个数据库中
并找到有关网络威胁的新见解。
- [`TypeDB OSI - Biomedical Knowledge Graph`](https://github.com/typedb-osi/typedb-bio) - TypeDB Bio 是一个开源生物医学知识图谱，为生物医学研究人员提供了一种直观的方式来查询互连和异构的生物医学
数据集中在一处。
- [`Source++`](https://github.com/sourceplusplus/Assistant) - 具有运行时感知和上下文感知功能的开源自动化编程助手。
- [`Rasa`](https://github.com/RasaHQ/rasa) - 开源机器学习框架，用于自动执行基于文本和语音的对话。
- [`TypeBridge`](https://github.com/ds1sqe/type-bridge) - 适用于 TypeDB 的现代 Python ORM，具有与 TypeDB 类型系统一致的基于属性的 API。
