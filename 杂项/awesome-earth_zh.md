## 减碳

- [1 Click Impact](https://1clickimpact.com/) - 1ClickImpact 是一个领先的平台，可帮助企业和个人使用 API 或无代码集成自动捕获碳和种植树木。每一次碳减排都可以通过经过验证的数据、项目地图和现场种植会议进行实时跟踪，以实现完全透明。它使可测量的碳去除和减少成为您的产品或可持续发展战略的无缝组成部分。
- [1% for the Planet](https://www.onepercentfortheplanet.org/) - 面向企业和个人的会员计划，允许他们将年毛利润或工资的 1% 捐赠给环保非营利组织，以创造一个更健康的地球。
- [Beam](https://www.beamproject.co/) - 当您在线购物时自动进行小额捐款，以支持试图改变世界的清洁技术初创公司。多年来，清洁技术投资资金已经枯竭，因此众包正在成为许多人的唯一选择，并且通过 Chrome 扩展，您可以在购买其他东西时投入一美元。
- [Carbon Pricing Dashboard](https://carbonpricingdashboard.worldbank.org/) - 有关区域、国家和地方碳定价举措的地图和统计数据。
- [Climate Action Tracker](https://climateactiontracker.org/countries/) - 了解哪些国家有望达成《巴黎协定》或类似协议，哪些国家却遭遇严重失败。提示：除了摩洛哥和冈比亚之外的所有国家。
- [ClimateWorks](https://www.climateworks.org/) - 一个与慈善家合作应对气候危机的非政府组织。他们力求确保投资产生最大的影响。
- [Climworks](http://climeworkspioneers.refr.cc/philsturgeon) - Climeworks 直接空气捕获技术从空气中捕获二氧化碳，将二氧化碳与地热发电厂使用的水混合，然后将其泵入地下深处。通过自然过程，二氧化碳与玄武岩发生反应，并在几年内永久安全地变成石头。冰岛是世界上为我们的开拓进程提供理想条件的几个地点之一。每月订阅以抵消一点您的足迹，并支持这项早期技术的发展。
- [Cool Earth](https://www.coolearth.org/) - 该慈善机构的成立是为了通过减少雨林损失造成的碳排放来应对气候变化。
- [Ecologi](https://ecologi.com/?r=5d01f2ed12ae7a358b75fdd7) - 坐在沙发上帮助地球，每月只需花费不到一瓶便宜的葡萄酒。每月花费 6.50 美元/6 欧元，观看您自己的森林变得生机勃勃，并抵消您的整个足迹。然后让您的朋友、家人、企业，甚至您的狗都报名参加！
- [Flight Free (UK)](https://flightfree.co.uk/) - 当我们迫切需要减少温室气体排放时，航空业是增长最快的温室气体排放源。承诺2020年免费飞行。

- [Giki](https://gikibadges.com/) - Giki 应用程序提供可访问、独立、透明的信息，帮助人们改变习惯，实现更可持续的生活。我们的 14 枚徽章涵盖可持续发展、健康和公平。

- [Patch](https://www.usepatch.com) - Patch 是负排放的 API。开发人员可以使用 Patch 的 API 通过各种负排放方法（例如直接空气碳捕获、生物油地质封存、增强风化、重新造林等）使任何交易实现碳中和。
- [Possible](https://www.wearepossible.org/) - 将个人和地方行动与更大的系统性变革相结合，我们以进取的态度和乐趣来面对气候恐惧。解决气候危机的可能方法是鼓励人们在 5 个关键领域采取行动——饮食和购买、旅行、能源、自然和对话。
