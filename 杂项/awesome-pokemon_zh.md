## 开发项目
#### API
- [PokeAPI](https://github.com/PokeAPI/pokeapi/) - Pokémon 的 RESTful API (http://pokeapi.co/)。
- [pgoapi](https://github.com/tejado/pgoapi) - Pokémon GO API 库/演示。
- [pokemon-API](https://github.com/Purukitto/pokemon-api) - 初学者友好的 API 端点，以 JSON 格式返回 Pokémon 数据 (https://app.pokemon-api.xyz)。
- [Pokemon-GO-node-api](https://github.com/Armax/Pokemon-GO-node-api) - Pokémon GO API Node.js 库。
- [Pokemon-Go-Rocket-API](https://github.com/FeroxRev/Pokemon-Go-Rocket-API) - C# 语言的 Pokémon GO 客户端 API 库。
- [Pokemon GO Java API](https://github.com/Grover-c13/PokeGOAPI-Java) - 适用于 Java 的 Pokémon GO API。
- [Pokemon Name API](https://github.com/sindresorhus/pokemon) - 获取 JSON 格式的 Pokémon 名称列表。
- [PokemonGO-Pokedex](https://github.com/Biuni/PokemonGO-Pokedex) - JSON 格式的 Pokémon GO 图鉴。
- [PokeTypes](https://github.com/fbosch/poke-types) - 了解神奇宝贝的类型、弱点和优势。
- [TCGdex](https://github.com/tcgdex/cards-database) - 多语言 Pokémon TCG API。
- [Kotlin-Pokedex](https://github.com/mrcsxsiq/Kotlin-Pokedex) - 使用 Kotlin 构建的 Pokedex 应用程序。
- [graphql-pokemon](https://github.com/lucasbento/graphql-pokemon) - 使用 GraphQL 获取 Pokémon 的信息。

#### 应用程序
- [Pokemon-Showdown](https://github.com/Zarel/Pokemon-Showdown) - 神奇宝贝战斗模拟器。
- [iPokeMon](https://github.com/Kjuly/iPokeMon) - iOS 上的 Pokémon 类游戏具有基于位置的服务。
- [pokemon-online](https://github.com/po-devs/pokemon-online) - 在线神奇宝贝战斗模拟器。
- [Shuffle-Move](https://github.com/Loreinator/Shuffle-Move) - 用于识别和显示 Pokémon Shuffle 游戏的最佳下一步动作的程序。
- [pokecry](https://github.com/fent/pokecry) - 从它们的叫声中猜出神奇宝贝。
- [Pokedex](https://github.com/skydoves/Pokedex) - Android Pokedex 使用 Dagger、Hilt、Motion、Coroutines、Flow 和 Jetpack。
- [Who's That Pokémon?](https://github.com/serena2341/whos-that-pokemon) - 从神奇宝贝的轮廓猜猜它们。

#### 机器人
- [Pokemon Go Bot](https://github.com/Hrithikd/Pokemon-Go-Bot-Working-Hack-API) - Pokémon GO 机器人，为 Pokestops 和 Pokémon 进行耕种。
- [slack-pokemon](https://github.com/rvinluan/slack-pokemon) - 用于在 Slack 中进行神奇宝贝战斗的机器人。
- [PokeSlack](https://github.com/timwah/pokeslack) - 提醒您附近的神奇宝贝。
- [TwitchPlaysPokemon](https://github.com/sunshinekitty/TwitchPlaysPokemon) - 模仿 Twitch 玩 Pokémon 的机器人。
- [PokemonGo-Bot](https://github.com/PokemonGoF/PokemonGo-Bot) - Pokémon GO 机器人脚本，与社区一起烘焙。
- [Poketrainer](https://github.com/j-e-k/poketrainer) - 一个快速且功能齐全的 Python 机器人。
- [PokeBot](https://github.com/akbaryahya/PokeBot) - 自动神奇宝贝速通机。
- [PokemonGoBot](https://github.com/jabbink/PokemonGoBot) - 玩 Pokémon GO 的机器人。
- [Bot Builder for PokemonGoBot](https://github.com/shilch/pogobot-builder) - 只需构建您的 Pokémon GO 机器人即可。
- [PokeBuddy](https://github.com/finndev/PokeBuddy) - Pokémon GO 机器人。

#### 杂项
- [alfred-pokedex](https://github.com/vutran/alfred-pokedex) - Alfred 3 查找 Pokémon 信息的工作流程。
- [hyper-pokemon](https://github.com/hyper-pokemon/hyper-pokemon) - 为您的超级终端量身定制奇妙的神奇宝贝主题。
- [Pokemon-Go-Controller](https://github.com/kahopoon/Pokemon-Go-Controller) - 安全地或在不可用的区域玩 Pokémon GO。
- [pokecrystal](https://github.com/pret/pokecrystal) - 神奇宝贝水晶的分解。
- [x-pokemon](https://github.com/passy/x-pokemon) - 用于显示 Pokémon 的 Web 组件。
- [pokered](https://github.com/pret/pokered) - 神奇宝贝红/蓝的拆解。
- [pokemon-reverse-engineering-tools](https://github.com/pret/pokemon-reverse-engineering-tools) - 神奇宝贝红和神奇宝贝水晶的编译和反编译工具。
- [pokemon-go-mitm-node](https://github.com/rastapasta/pokemon-go-mitm-node) - Pokémon GO MITM 代理。
- [PokeMMO](https://github.com/maierfelix/PokeMMO) - 带实时编辑器的 WebGL Pokémon MMO 引擎。
- [pogo-optimizer](https://github.com/justinleewells/pogo-optimizer) - 向您显示确定哪些神奇宝贝被磨成糖果所需的 IV 和信息。
- [PoGoTools](https://github.com/nelsyeung/PoGoTools) - 一组 CLI 工具，用于获取 IV、转移 Pokémon、进化 Pokémon 和丢弃物品。
- [slack-pokemon-emoji](https://github.com/fraserxu/slack-pokemon-emoji) - 一个将所有神奇宝贝上传到 Slack 表情符号的工具。
- [slack-emoji-pokemon](https://github.com/Templarian/slack-emoji-pokemon) - Slack 的神奇宝贝表情符号列表。
- [PokemonGO-IV-Renamer](https://github.com/Boren/PokemonGO-IV-Renamer) - 自动将您的神奇宝贝重命名为其 IV 统计数据。
- [pokemongo-genymotion](https://github.com/jlobos/pokemongo-genymotion) - 从 Genymotion 设备玩 Pokémon GO。
- [pokemongo-game-master](https://github.com/BrunnerLivio/pokemongo-game-master) - Pokémon GO Game Master 版本合集。
- [Pokemon-Terminal](https://github.com/LazoCoder/Pokemon-Terminal) - iTerm2 的 Pokémon 终端主题。
- [pokemon-cli](https://github.com/sindresorhus/pokemon-cli) - 用于获取 Pokémon 名称的 CLI。
- [pokesprite](https://github.com/msikma/pokesprite) - 来自核心系列游戏的神奇宝贝精灵数据库，包括定制的闪亮版本。
- [PokéPalettes](https://github.com/BarryMode/pokepalettes) - 代表 Pokémon 的 16 位调色板（基于 Pokémon GO 的模型）。
- [PokemonRedExperiments](https://github.com/PWhiddy/PokemonRedExperiments) - 通过强化学习玩神奇宝贝红。
