# 很棒的堆栈
[![StackShare](https://img.shields.io/badge/tech-stack-0690fa.svg?style=flat)](https://stackshare.io/stackshare/awesome-stacks)

> 用于构建不同应用程序和功能的技术堆栈。

## 内容

- [Front-end](#front-end)
- [全栈](#full-stack)
- [Back-end](#back-end)
- [Mobile](#mobile)

Awesome Stacks 是社区策划的技术堆栈列表，用于构建不同的应用程序和功能。它是开源的，灵感来自原始的[很棒的列表](http://awesome.re/)。

列表中的每个堆栈都有名称、描述以及一些关键工具和技术的列表。或者，它还可以链接到教程、入门工具包或样板文件，以便轻松上手。

有一个您认为是构建某些东西的好方法的堆栈吗？请编辑此文件并添加它！查看 [CONTRIBUTING.md](CONTRIBUTING.md) 了解更多信息。

#### 浏览堆栈

可以在两个地方浏览堆栈：自述文件和 [awesomestacks.dev](https://awesomestacks.dev/)。该网站由 Gatsby 和 React 构建，通过从 GitHub 和 StackShare API 提取数据来显示 README 中列出的每个工具的徽标和指标。
