太棒了 Ti*k*Z
============

[![](https://cdn.rawgit.com/sindresorhus/awesome/master/media/badge.svg)](http://awesome.es)

[Ti*k*Z](https://en.m.wikipedia.org/wiki/PGF/TikZ) 是一个非常棒的 TeX/LaTeX/ConTeXt 图形绘制包。与其他 TeX 包不同，TikZ 本身非常庞大，它的手册包含大约 1100+ 页、100+ 章。数十个其他软件包和多个 GUI 应用程序都基于 TikZ。

该项目包含一系列精彩的 TikZ 文档、库和资源。

书籍
-----

- [PGF Manual](http://mirrors.ctan.org/graphics/pgf/base/doc/pgfmanual.pdf) - 大约1200页，其中300页包含核心概念。
- [PGFPlots Manual](http://mirrors.ctan.org/graphics/pgf/contrib/pgfplots/doc/pgfplots.pdf) - 约560页。
- [VisualTikZ](https://www.ctan.org/pkg/visualtikz) - TikZ 的视觉帮助基于具有最少文本的图像。
- [LaTeX/PGF/TikZ](https://en.wikibooks.org/wiki/LaTeX/PGF/TikZ) - 来自维基教科书的简短教程。
- [MinimalTikZ](http://cremeronline.com/LaTeX/minimaltikz.pdf) - 对 TikZ 的简单介绍。

画廊
-------

- [TeXample](http://www.texample.net/tikz/) - TikZ 制作的精彩示例画廊。
- [walmes/Tikz](https://github.com/walmes/Tikz) - Walmes M. Zeviani 教授收藏的 TikZ 艺术作品包含约 200 个 TikZ 人物，其中大部分是为了教授统计学而制作的。
- [MartinThoma/LaTeX-examples](https://github.com/MartinThoma/LaTeX-examples/tree/master/tikz) - 包含来自 Martin 的约 350 个 TikZ 示例。托马.
- [Nice Scientific Pictures](https://tex.stackexchange.com/questions/158668/nice-scientific-pictures-show-off) - 包含一些非常好的 TikZ 示例，以及其他 TeX 朋友绘制的图片。
- [Love heart with LaTeX](https://tex.stackexchange.com/questions/139733/can-we-make-a-love-heart-with-latex) - 包含用 LaTeX/TikZ 制作爱心的可爱代码。
- [PetarV-/TikZ](https://github.com/PetarV-/TikZ) - 由 Petar Veličković 创作的可供出版的 PGF/TikZ 人物合集。
- [alemelis/tikz_drawings](https://github.com/alemelis/tikz_drawings) - Alessandro Melis 在 TikZ 中为各种出版物制作的插图源代码。
- [LatexDiagrams](https://github.com/FriendlyUser/LatexDiagrams) - 该仓库包含许多为学术用途而生成的图表，包括算法-数据结构、人工智能、博客图表、DSP、工程软件设计等。
- [tikz-network](https://github.com/hackl/tikz-network) - 在 LaTeX 中可视化复杂网络的工具。
- [TikZ.net](https://tikz.net/) - LaTeX 中的 TikZ 图形。

- [sane_tikz](https://github.com/negrinho/sane_tikz) - 使用 Python 创建 TikZ 人物。
- [quiver](https://github.com/varkor/quiver) - 导出到 tikz-cd 的交换图编辑器。
- [TikZiT](https://tikzit.github.io/) - 专为图形和字符串图设计的 GUI 编辑器。
