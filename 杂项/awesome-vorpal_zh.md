## 项目

*使用 Vorpal 构建的很棒的项目和应用程序。*

- [Cash](https://github.com/dthree/cash) - 跨平台 Linux 命令，毫无问题。
- [Raptor](https://developer.mozilla.org/en-US/Firefox_OS/Automated_testing/Raptor) - 面向 Firefox 开发人员的性能测试 CLI。
- [iTunes-remote](https://github.com/mischah/itunes-remote/) - 在 CLI 上控制 iTunes。
- [phunt](https://github.com/Kristories/phunt) - 在 CLI 上访问 [Product Hunt](https://www.producthunt.com/)。 *（你猜对了）Product Hunt 上的趋势。*
- [trelew](https://github.com/websitesfortrello/trelew) - 在 CLI 上访问 [Trello](https://trello.com/)。
- [wat](https://github.com/dthree/wat) - 针对每种语言、平台和库的社区构建文档。
- [metronome](https://github.com/AljoschaMeyer/metronome-cli) - CLI 上的实时节拍器。
- [fastack](https://github.com/fastack/cli) - 一个零配置工具，可以轻松开发客户端应用程序。
- [cbcluster](https://www.npmjs.com/package/cbcluster) - Couchbase 服务器集群的 CLI。
- [quotr](https://github.com/andrerpena/quotr) - 查看证券交易所报价。
- [Vantage](https://github.com/dthree/vantage) - 用于您的实时 Node 应用程序的“CLI”+“SSH”+“REPL”。
- [birdknife](https://github.com/vanita5/birdknife) - 功能齐全的 [Twitter](https://twitter.com/) CLI。
- [tv-remote-cli](https://github.com/Glavin001/tv-remote-cli) - CLI 用于远程控制您的智能电视。
- [launch](https://github.com/NewSpring/meteor-launch) - 自动将 Meteor 构建到 AppStore、TestFlight、Hockey、Google Play 等
- [lifxsh](https://github.com/ristomatti/lifxsh) - 用于控制 [LIFX](http://www.lifx.com) 智能灯的交互式 shell。

## 扩展

*扩展添加到 Vorpal 的功能 - 以编程方式或实时方式。*

#### POSIX 实现

- [less](https://github.com/vorpaljs/vorpal-less) - 执行“less”命令。
- [grep](https://github.com/vorpaljs/vorpal-grep) - 执行“grep”命令。

#### 开发工具

- [repl](https://github.com/vorpaljs/vorpal-repl) - 将您的 CLI 放入应用程序上下文中的 REPL 中。
- [tour](https://github.com/vorpaljs/vorpal-tour) - 为您的 Vorpal 应用程序构建交互式游览。
- [watch](https://github.com/vantagejs/vantage-watch) - 实时更新您的实时 Vorpal 扩展。
- [log](https://github.com/AljoschaMeyer/vorpal-log) - 高级日志记录实用程序。
- [set-or-print](https://github.com/AljoschaMeyer/vorpal-setorprint) - 使用命令轻松设置/获取参数。

#### 实时

- [use](https://github.com/vorpaljs/vorpal-use) - 实时导入 Vorpal 扩展：在您的应用程序运行时。

#### 乐趣

- [hacker-news](https://github.com/vorpaljs/vorpal-hacker-news) - 从黑客新闻中提取最热门的内容。

#### 工具箱
- [rc](https://github.com/subk/vorpal-rc) - 简单的 .rc 文件支持。
- [comment](https://github.com/subk/vorpal-comment) - 类似 Bash 的评论支持。
- [as-default](https://github.com/ialpert/vorpal-as-default) - 如果 Vorpal 任务在没有任何特定定义的任务的情况下运行，则触发该任务。
