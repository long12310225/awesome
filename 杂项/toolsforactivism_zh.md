# 激进主义的数字工具

![digital tools for activism](https://raw.githubusercontent.com/drewrwilson/toolsforactivism/master/digital-tool.jpg "digital tools for activism")

[![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

以下是一系列很棒的激进数字工具。这些是为个人、特设团体或正式组织提供的工具，用于为社会变革开展活动、组织或沟通。其中一些工具是开源项目，需要一些技术知识才能自行设置和托管。有些是软件即服务工具，您可以通过浏览器访问网页来使用。请随意分享建议。

# 名单

## 开源自行托管：

 * [Contact Congress (Open Source)](https://github.com/EFForg/contact-congress) - 向美国国会议员发送电子邮件。这是由 EFF 作为开源项目开发的。他们将其用于自己的内部活动。还有一个由 Fight for the Future 运行的托管版本（见下文）。
 * [Call Congress](https://github.com/fightforthefuture/call-congress) - 协调与美国国会、总统或当地或企业目标的通话日。来自 Fight for the Future 和 Taskforce.is 的原始项目，但有来自 [Mozilla](https://github.com/mozilla/call-congress) 和 [EFF](https://github.com/effOrg/call-congress/tree/refactor/master) 的大量开发。前端看起来像这样，[https://call.mozilla.org](https://call.mozilla.org/)，后端看起来像[http://callpower.org](http://callpower.org/easy-to-use/)。托管版本可用，代码是 AGPL。
 * [Twilio.org's Rapid Response Kit](https://github.com/Twilio-org/rapid-response-kit) - 使用 Twilio 的服务收集与电话相关的工具。套件包括：自动回复（入站语音/短信应答器）、广播（向列表广播文本/呼叫的简单方式）、会议线路、转发器、响铃（呼叫列表中的人员，如果第一个人没有应答，则继续下一个）、帮助热线（按 1 进行...，按 2 进行...）
 * [Ushahidi](https://www.ushahidi.com/) - 数据管理和可视化平台。其他工具包括 CrowdMap、CrisisNET（有关世界危机的数据）、Ping（用于在紧急情况后检查某人是否安然的工具）、SMSsync（双向短信网关）。查看他们的 [github 存储库](https://github.com/ushahidi)。
 * [Crowdring](https://github.com/therules/CrowdRing) - 网络应用程序，世界各地的人们可以拨打免费的未接来电以签署请愿书
 * [Self Starter](https://github.com/lockitron/selfstarter) - 推出您自己的众筹网站
 * [TakeCharge.io](https://github.com/controlshift/prague-server) - 由 Stripe 提供支持的开源或 SaaS 在线捐赠。
 * [Countdown](https://github.com/drewrwilson/countdown) - 一个快速简单的工具，用于制作您自己的网页，并为即将举行的活动提供倒计时。
 * [CiviCRM](https://civicrm.org/) - 适用于非营利/活动人士部门的联系人管理系统，包括捐赠、会员资格、活动、短信、批量邮件和完整的 CRM。
 * [Spoke](https://github.com/Elizabeth-Warren/Spoke) - 供组织动员支持者和成员采取行动的文本分发工具。 Spoke 允许您上传电话号码、自定义脚本并分配志愿者与支持者沟通，同时允许组织管理该流程。

## 软件即服务：

 * [Loomio](https://www.loomio.org/) - 小组协作决策的工具。注意：Loomio 的[代码也是开源的](https://github.com/loomio/loomio)，但它通常用作托管服务。
* [Change.org](https://www.change.org/) 请愿网站。选择一个目标，提出请愿书，并将其发送给您的朋友。如果他们受到关注，Change.org 的员工可以提供支持。 （他们还有组织帐户，允许团体创建品牌页面并导出签名者的联系信息。）
 * [Contact Congress (FFTF)](http://congress.fightforthefuture.org/) - 用于向美国国会议员发送电子邮件。这是“为未来而战”的开源联系国会的托管版本。非常适合向国会发送大量电子邮件但没有可以安装和维护自己版本的开源软件的技术人员的组织。
 * [ControlShift](https://www.controlshiftlabs.com/) - 为进步倡导组织分发请愿书、活动和当地团体。帮助提高活动参与度的工具。
 * [Riseup](http://riseup.net/) - 为活动人士提供通信工具的提供商，包括免费网络邮件、聊天和 VPN
 * [Action Network](https://actionnetwork.org) - 适用于进步组织的非营利性数字组织工具集。包括请愿、宣传、筹款、群发电子邮件等免费或廉价活动。
* [Democracy.io](https://democracy.io) ([code](https://github.com/sinak/democracy.io)) - 向美国参议院和众议院议员发送消息。
 * [MoveOn's Petition Platform](https://petitions.moveon.org/) - 在 MoveOn 的托管平台上发起您自己的请愿。如果您的请愿书获得关注，MoveOn 工作人员可能会帮助支持您。
 * [CiviWiki](https://civi.wiki/) - 一个开源、非营利社区，致力于开发民主参与网络系统。
 
## 操作系统：

 * [Tails](https://tails.boum.org/) - 一种基于 Linux 的实时媒体操作系统，旨在允许在不留下数字足迹的情况下使用计算机。
 
## 浏览器：

 * [Tor Browser](https://github.com/TheTorProject/gettorbrowser) - 用于实现匿名通信的免费软件。它可以防止监视您的互联网连接的人了解您访问的网站，它可以防止您访问的网站了解您的物理位置，并且可以让您访问被阻止的网站。
 
* [Brave Browser](https://github.com/brave/brave-browser) - [Brave](https://brave.com/) 是由 Brave Software, Inc. 基于 Chromium 网络浏览器开发的免费开源网络浏览器。它可以阻止广告和网站跟踪器，并为用户提供一种以基本注意力令牌的形式向网站和内容创建者发送信息贡献的方式。


## 需要关注的项目（即开发中的项目）

这些都是正在进行中的项目。未完成，但值得一看。

 * [Where@](https://github.com/the-learning-collective/whereat-macroid) - 一款帮助抗议者在游行期间实时沟通的移动应用程序
* [Stencil Validator](https://drewrwilson.com/stencilvalidator/) ([code](https://github.com/drewrwilson/stencilvalidator)) - 一个设计工具，用于确定您的作品是否是有效的模板
 * [Day of Action](https://github.com/handsupwalkout/handsupwalkout.github.io) - 用于协调高中生一天行动的非常粗糙的工具。运行一天行动活动的更通用工具的起点
 * [Umbrella App](https://github.com/securityfirst/Umbrella_android) - 一款开源 Android 应用程序，包含有关数字和物理安全主题的课程和清单，范围从发送安全邮件到处理物理监视和安全参加抗议。还包括用于响应安全事件和源的表格，以确保用户及时了解其所在区域的最新安全信息。还有西班牙语和中文版本。
 * [Code Corps](https://github.com/code-corps) - Code corps 希望提供工具，将人们与具有社会影响的项目联系起来，促进众筹，并为项目工作人员提供支持

## 墓地

这些是曾经存在但不再存在的项目或工具
 * [Thunderclap](http://thunderclap.it) - 使用它来协调同时在社交媒体上共享相同消息的群体。您写一条推文，开展一项活动，并要求人们承诺支持您的活动。当您达到一定数量的承诺者的目标时，您的推文将自动发布到每个承诺者的社交媒体上。


## 我如何对此列表做出贡献

您知道应该在此列表中添加一个工具吗？添加方法如下：
* **熟悉 github？** 发送带有您建议的拉取请求。如果您不确定如何做到这一点，请不要担心。见下文。
* **不熟悉 github？** 您可以通过单击右侧的“Issues”选项卡并添加新问题（就像评论一样）在此页面上留下评论。仅供参考，您需要创建一个 github 帐户。
