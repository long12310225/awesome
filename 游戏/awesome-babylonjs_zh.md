# 很棒的 Babylon.js [![Awesome](https://awesome.re/badge-flat.svg)](https://awesome.re)

<a href="https://www.babylonjs.com/"><img alt="Babylon.js logo" src="media/logo.svg" align="right" width="150" /></a>

> 与 Babylon.js 游戏引擎相关的_精彩_内容的精选列表。

[Babylon.js](https://www.babylonjs.com/) 是一个用 TypeScript 编写的开源游戏和渲染引擎，旨在强大、美观、简单和开放。它支持通过 WebGL、WebGPU 和 Babylon Native 运行时进行跨平台游戏开发。
