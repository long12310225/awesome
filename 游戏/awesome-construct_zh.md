## 出口

### 构造3

- [Chowdren](https://mp2.dk/chowdren/) - Clickteam Fusion 和 Scirra Construct 的运行速度极快。
- [Pipelab](https://pipelab.app/) - 全自动出口管道。

### 构造2

- [AppPlayer-Electron](https://github.com/Floflis/AppPlayer-Electron) - 在 Electron 预构建的 exe 中预览应用程序/游戏 - 就像 Electron 是普通浏览器一样！目前仅适用于 Windows，但很快将适用于 Mac 和 Linux。
- [Fresh NW](https://github.com/kckarnige/FreshNWjsForC2) - 轻松设置最新版本 NW.js 的导出，替换过时的“NW.js for Construct 2”安装程序。
- [website Reinarte](https://reinarte.com.br/) - 插件、行为和示例。

## 学习

### 系列/课程

#### 构造3

- [Marcos Game Dev: Plataforma Gostosinho - Técnicas de Polimento](https://www.youtube.com/playlist?list=PLuDpKE2ECBovsRiINMYss6iOHJYCN8MNe) - 这一切都始于一个问题：“你能在不添加新艺术的情况下完善一款游戏吗？” “Plataforma Gostosinho”系列通过将基本平台游戏转变为《Construct 3》中有趣的游戏来回答这个问题！
- [Rafael Trigo's Tutorials](https://www.construct.net/en/tutorials/construct-tutorials-2525) - Construct 3 教程列表。

- [English Acorn: Editor](https://www.youtube.com/playlist?list=PLusL6SZZRaWr4zqm6GrQHmOuShwtypRJ5) - 有关编辑器界面的一般教程。
- [English Acorn: Plugins and Objects](https://www.youtube.com/playlist?list=PLusL6SZZRaWqP6UmVPs9Ml8BRo2t-wvnT) - 有关插件及其在编辑器中如何工作的教程。

- [John Maksym's Tutorials](https://www.youtube.com/playlist?list=PLq6aUvz66jtfnuyOKnPXO5dZly6CB8naA) - 一般教程。

- [Lee Stemkoski: Breakout Clone](https://www.youtube.com/playlist?list=PL59F92017DA9887DB) - 了解如何创建 Brakeout 游戏。

- [Let's Make Games: PONG](https://www.youtube.com/watch?v=cvItNHEMB80) - 有关如何创建 Pong 克隆的教程。
- [Let's Make Games: ASTEROIDS](https://www.youtube.com/watch?v=nYULTk2BpKY) - 有关如何创建小行星克隆的教程。
- [Let's Make Games: Flappy Bird](https://www.youtube.com/watch?v=eBccn1qMB_U) - 有关如何创建 Flappy Bird 克隆的教程。

- [LividProductions: Platformer Series](https://www.youtube.com/playlist?list=PLz66pfeeD4TcPN9kO7JlqiITpgi_WWgn7) - 本系列将教您如何制作平台游戏。

- [Nicholas Dingle's Tutorials](https://www.youtube.com/playlist?list=PLsJBMeqEdtggBJBiuX8bnF10ewHjdE20e) - 大量教程涵盖您可以创建的不同游戏和不同技巧。

- [Marcos Game Dev: Curso Metroidvania 1.0](https://www.youtube.com/playlist?list=PLuDpKE2ECBou-kcov-1DdbnNHLjxGiwDp) - 2018 年录制的《银河恶魔城》课程 1.0 版现已免费！按照课程在《Construct》中创建您自己的银河恶魔城！
- [Marcos Game Dev: Curso Beat'em Up](https://www.youtube.com/playlist?list=PLuDpKE2ECBovMW6iHOWwa0zy5_g_7KL2l) - 在《Construct》中通过连击、本地 4 人多人游戏、多个敌人等创造令人惊叹的清版游戏！不推荐初学者使用。
- [Marcos Game Dev: Criando Roguelike](https://www.youtube.com/playlist?list=PLuDpKE2ECBosjwzkBzyTCOGr8WDLXflY6) - 学习通过随机关卡生成、多个敌人和完整的平台游戏角色来创建 Roguelike！不推荐初学者使用。
- [Marcos Game Dev: Super Mario Bros.](https://www.youtube.com/playlist?list=PLuDpKE2ECBou7QX1vZa7dFt6n3mTbnYAD) - 了解如何在《Construct》中创建《超级马里奥兄弟》机制，忠实再现 NES 经典！
- [Marcos Game Dev: Recriando Megaman X](https://www.youtube.com/playlist?list=PLuDpKE2ECBovmH0TG_ye_Wrg9CATAiB1p) - 学习创造 Megaman X 机制，如标志性的 X-buster、激烈的 Boss 战斗以及 Construct 中的特殊能力！详细的中级视频（带有一些高级内容）。
- [Marcos Game Dev: Criando Menu Principal](https://www.youtube.com/playlist?list=PLuDpKE2ECBoulD0kDJNLmFc32nphXvY3U) - 学习创建适用于几乎所有游戏的主菜单（重点是桌面游戏）。包括全屏/窗口模式、控制映射、语言切换、音频设置等。

- [thenewboston: Game Development w/ Construct](https://www.youtube.com/playlist?list=PL6gx4Cwl9DGAfHxsK6bji7trLS-N0nKF-) - thenewboston Game Development 的官方播放列表。
- [thenewboston: Platform Game Development](https://www.youtube.com/playlist?list=PL6gx4Cwl9DGAp287UuTE0-K7Ty-b8rGAX) - newboston 平台游戏开发的官方播放列表。
- [thenewboston: Physics Puzzle Game Development](https://www.youtube.com/playlist?list=PL6gx4Cwl9DGDexNbWi0uPBP6buduUZO3Q) - 新波士顿物理益智游戏开发教程的官方播放列表。

- [TodaysTuts: Behaviors](https://www.youtube.com/playlist?list=PLO3K3VFvlU6B1InGyrx7Iz7w_MDizWlXK) - 有关行为的教程。
