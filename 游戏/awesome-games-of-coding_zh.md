## 独立于语言

- [Artifacts MMO](https://artifactsmmo.com/) - Artifacts 是一款沙盒 MMORPG，您可以使用任何编程语言通过我们的 API 来控制您的角色。
- [Lambda Spellcrafting Academy](https://www.bittwiddlegames.com/lambda-spellcrafting-academy/) - 一款可视化编程/益智游戏，您可以在其中构建咒语来解决难题。涵盖入门编程以及一些数据结构和算法。该语言的灵感来自 Lisp，但游戏侧重于适用于大多数语言的编程基础知识。
- [Codingame](https://www.codingame.com/home) - 一个学习和竞争的平台。提出各种多人游戏和竞赛以及优化挑战和单人谜题。您也可以贡献自己的力量。选择您的语言，享受图形。
- [CodeCombat](https://codecombat.com) - 选择你的英雄，并在基斯加德地牢的食人魔巡逻队、熔岩坑和激光束中编写你的道路。升级、赚取宝石并掠夺魔法物品以解锁新的编程能力。加入数百万玩家的行列，在这款开源编程游戏中从一个完全的初学者成长为使用 Python、JavaScript、CoffeeScript 或 Lua 编写高级多人 AI。
- [Vindinium](https://www.codingame.com/multiplayer/bot-programming/vindinium) - 玩家用自己选择的编程语言（可以选择使用众多入门包之一）编写脚本，通过 HTTP 请求控制 16 位英雄，引导他收集金币，与其他玩家战斗，甚至访问酒馆。
- [RealTimeBattle](http://realtimebattle.sourceforge.net) - 一种由程序控制的机器人互相战斗的编程游戏。目标是消灭敌人，使用雷达检查环境并使用大炮射击。
- [Battlesnake](https://play.battlesnake.com) - 加入来自世界各地的开发者社区，在 Battlesnake 中争夺名声和荣耀，Battlesnake 是一项基于经典游戏“蛇”的多人编程竞赛！您所需要玩的只是一个实时网络服务器，该服务器实现了 Battlesnake API 和您选择的语言，以开始战斗寻找食物，避开其他 Battlesnake，并尽可能长时间地生存。
- [Coder One](https://www.gocoder.one) - 年度人工智能编程锦标赛以类似于“炸弹人”的多人游戏、现金奖励和决赛现场直播为特色。创建一个 AI 玩家，可以在 2D 网格世界中导航，收集能量并放置炸药，以击倒对手。使用我们的 Python 或 TypeScript 入门工具包之一以 Docker 映像的形式提交您的 AI（社区贡献的入门工具包也提供 Go、C++ 和 Rust 版本）。

## 蟒蛇

- [Robocode](https://github.com/turkishviking/Python-Robocode) - 目标是开发一种机器人作战坦克来与其他坦克作战。机器人战斗在屏幕上实时进行。
- [Robot Rumble](https://robotrumble.org/) - 一个免费开源的人工智能竞赛平台。为在 1x1 网格竞技场中战斗的机器人团队编写算法，并与网站上的其他人测试您的技能。机器人可以向四个方向之一移动和攻击，100回合后机器人较多的一方获胜。支持 Python 和 JavaScript，并附带基于桌面的命令行工具。

## JavaScript

- [Programming Game](https://programming-game.com) - 通过 websocket 玩的实时生存 MMORPG。浏览器中有一个超级简单的演示[演示](https://programming-game.com/demo)，展示了如何玩。
- [AdventureLand](https://adventure.land/) - Adventure Land 是一款实验性独立 MMORPG。任何人都可以编写最多 4 个字符，让 JavaScript 来完成繁琐的工作，而您则可以做一些富有成效的事情！
- [Screeps](https://screeps.com) - 不再有任何意义和点击！您编写真正的 JavaScript 来自主控制您的单元。随时随地，即使您处于离线状态。
- [JavaScript Fight Club](https://jsfight.club) - 红王与蓝王交战了！决定胜利者需要你的 JavaScript 技能：你能编写一个有效的 AI 来获得荣誉和荣耀吗？
- [Elevator Saga](http://play.elevatorsaga.com) - 直接在浏览器中编写 JavaScript 对电梯进行编程，使其达到其效率目标。从简单的“入门”脚本开始，然后逐渐发展到具有许多楼层和多部电梯的更复杂的情况。
- [Untrusted](https://alexnisnevich.github.io/untrusted) - 该游戏为您提供了一个类似 roguelike 的游戏环境和一个控制台窗口，其中包含生成每个级别的 JavaScript 代码。加载后，每个关卡都是无与伦比的，并且大部分 JavaScript 都被阻止进行编辑。挑战在于仅使用留给您的有限工具打开通往下一个级别的道路。
- [DeepestWorld](https://deepestworld.com/) - 一款浏览器 MMORPG，您可以在其中使用 JavaScript 编写角色。探索一个没有加载屏幕的无边无际、持续开放的世界，其中众多的生物群落和生物聚集在一起，形成一个生机勃勃的世界。
- [BitBurner](https://github.com/bitburner-official/bitburner-src) - 一款基于编程的闲置增量角色扮演游戏，玩家在黑暗的反乌托邦世界中扮演一名不知名的黑客。游戏可以在浏览器中玩，也可以通过Steam本地安装。

## 斯卡拉

- [Scalatron](http://scalatron.github.io) - 程序员让机器人程序（用 Scala 编写）相互竞争。它是为想要更多地了解 Scala 编程语言或想要磨练 Scala 编程技能的程序员团体或个人提供的教育资源。


## 杂项

- [Generals IO](http://generals.io) - 一款快节奏的策略游戏，您可以扩展自己的土地并与敌人作战。当你的将军被俘虏时你就输了，但捕获对手的将军可以让你控制他们的整个帝国。
- [Robo Dojo](http://robodojo.club) - 你建造的机器人可以建造、破坏和摧毁其他机器人，试图消灭所有竞争对手。
- [Corewars](http://www.corewars.org) - 战士与战士之间的游戏，由两个对手编程，并放置在虚拟“环”中进行殊死搏斗。
- [TIS-100](http://www.zachtronics.com/tis-100) - 由 SpaceChem 和 Infinifactory 的创建者 Zachtronics 开发的开放式编程游戏，您可以在其中重写损坏的代码段来修复 TIS-100 并解开其秘密。这是您从未要求过的汇编语言编程游戏！
- [Shenzhen I/O](http://www.zachtronics.com/shenzhen-io) - 您可以使用来自不同制造商的各种组件（例如微控制器、存储器和逻辑门）构建电路，然后使用紧凑且功能强大的汇编语言编写代码，其中每条指令都可以有条件地执行。
- [MHRD](http://store.steampowered.com/app/576030) - 硬件设计游戏，您可以在其中使用硬件描述语言设计各种硬件电路。在创建功能齐全的 CPU 设计之前，您设计的硬件电路会变得越来越复杂。
- [Colobot](https://colobot.info) - 3D 视频游戏，您可以对机器人进行编程，使其能够骑行、飞行、寻找、携带、加油和破坏。游戏使用一种称为 CBOT 的语言，类似于 C++ 和 Java。 Colobot 提供教程式活动，任务难度逐渐增加，或者您也可以玩场景来挑战您的技能。
- [Code Games](http://codegames.io/en) - 实时编程游戏，您可以通过对太空飞船的行为进行编码来与其他玩家竞争。编程语言简单易学，游戏超级有趣。订阅我们的一项锦标赛并争夺奖品。
- [Mudlet](http://www.mudlet.org) - 允许您玩 MUD（纯文本 MMORPG）的游戏客户端。您可以在 Mudlet 中编写 Lua 代码来帮助您玩 MUD。
- [CSS Diner](http://flukeout.github.io) - CSS 项目排列游戏，教授 CSS 选择器和布局的概念和使用。您修改 CSS 来排列表示餐桌上的盘子和便当盒的 HTML 元素。
- [Guido van Robot](http://gvr.sourceforge.net) - 编程语言和免费软件应用程序旨在向初学者介绍编程基础知识。 GvR 可在 Windows、Macintosh 和 GNU/Linux 上以多种语言运行！它在课堂和家庭中都非常适合向人们介绍编程的基本概念。
- [CSSBattle](https://cssbattle.dev) - 在线 CSS 代码高尔夫战场。在这里，来自世界各地的玩家尝试用尽可能小的 CSS 代码直观地复制“目标”，并争夺排行榜的榜首。
- [The Command Line Murders](https://github.com/veltman/clmystery) - 终点站城发生了一起谋杀案，TCPD 需要您的帮助。
- [SQL Murder Mystery](https://mystery.knightlab.com/) - SQL 谋杀之谜旨在成为学习 SQL 概念和命令的自学课程，也是经验丰富的 SQL 用户解决有趣犯罪的有趣游戏。
- [S3 Game](http://s3game-level1.s3-website.us-east-2.amazonaws.com/) - 你是宝藏猎人。你的任务是获取尽可能多的密码并打开尽可能多的宝箱。
- [Learn Git Branching](https://learngitbranching.js.org/) - 在网络上学习 Git。
- [Oh my Git](https://ohmygit.org/) - 一款关于学习 Git 的开源游戏！
- [YouBrokeProd](https://youbrokeprod.com) - 通过交互式终端模拟调试生产事件，涵盖磁盘已满、DNS 故障、Kubernetes 崩溃循环等。
