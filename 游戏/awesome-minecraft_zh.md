[Minecraft](https://minecraft.net/) 是 Mojang 开发的一款沙盒视频游戏。该游戏允许玩家在 3D 程序生成的世界中使用各种不同的块进行建造，这需要玩家的创造力。游戏中的其他活动包括探索、资源收集、制作和战斗。

## 网站

### 维基百科

- [Official Minecraft Wiki](https://minecraft.wiki/) - Minecraft Wiki 是 Minecraft 的完整资源，包括游戏玩法、方块、制作、更新、事件、模组、运行服务器和资源包。

## 模组加载器

### 技术

- [Tinkers Construct](https://minecraft.curseforge.com/projects/tinkers-construct) - Tinkers' Construct 是一个模组，它以多种方式将工具组合在一起，然后对其进行修改，直到它们变成其他东西。
- [Immersive Engineering](https://minecraft.curseforge.com/projects/immersive-engineering) - 沉浸式工程是一种具有一定魅力的技术模组，基于达米安·哈扎德 (Damien Hazard) 的想法和概念，并拥有大部分资产。
- [Botania](https://github.com/Vazkii/Botania) - Botania 本质上是一个科技模组。主题是自然魔法。
- [Thermal Foundation + Dynamics + Expansion](https://minecraft.curseforge.com/projects/thermalexpansion) - 服务器友好且内容丰富的魔法与技术的融合！标题是三个不同的模组名称，但属于同一系列并且通常一起玩。
- [Railcraft](https://github.com/Railcraft/Railcraft) - 极大地扩展和改进了《我的世界》中的矿车系统。
- [Actually Additions](https://www.curseforge.com/minecraft/mc-mods/actually-additions) - 实际上，Additions 是一个技术模块，添加了各种实用程序，如多功能工具、物品磁铁、原子重建器等。
- [Applied Energistics 2](https://github.com/AppliedEnergistics/Applied-Energistics-2) - 一个关于物质、能量并利用它们征服世界的 Minecraft 模组。主要集中在物品存储上。
- [Draconic Evolution](https://www.curseforge.com/minecraft/mc-mods/draconic-evolution) - 最好的技术模组之一，但在游戏后期确实过于强大。添加诸如龙语能量核心之类的东西，目前还没有人能够填充......
- [Ender IO](https://www.curseforge.com/minecraft/mc-mods/ender-io) - 一个大型末影主题模组，增加了许多新的有趣的东西。
- [Galacticraft](https://micdoodle8.com/mods/galacticraft) - 该模组添加了火箭和新行星供探索。可通过插件进行扩展。
- [OpenComputers](https://ocdoc.cil.li) - OpenComputers 是一个在游戏中添加计算机和机器人的 mod，可以使用 Lua 5.3 进行编程。
- [ComputerCraft](https://computercraft.cc) - 将基于 Lua（5.2 和 5.3）的计算机添加到 Minecraft 中。
- [WarpDrive](https://www.curseforge.com/minecraft/mc-mods/warpdrive) - 将《星际迷航》和其他系列（如船舶、武器、运输车等）添加到游戏中。还添加了新的行星（维度）。

### 农业与食品

- [Pam's HarvestCraft](https://minecraft.curseforge.com/projects/pams-harvestcraft) - Pam's HarvestCraft 是 Minecraft 的一个模组，目前添加了多种农作物和食物。
- [AgriCraft](https://www.curseforge.com/minecraft/mc-mods/agricraft) - 该模组添加了更复杂的作物系统，例如基因突变的能力，并且还添加了更多作物。
- [Bonsai Trees](https://www.curseforge.com/minecraft/mc-mods/bonsai-trees) - 本质上增加了一个街区大的迷你林场。

### 能源

- [RFTools](https://minecraft.curseforge.com/projects/rftools) - RFTools 是一个包含与 RF (Redflux) 或一般技术相关的各种工具的模组。
- [Extreme Reactors](https://www.curseforge.com/minecraft/mc-mods/extreme-reactors) - 添加带有虚构燃料“钇”的核反应堆。原始大型反应堆的继承。
- [Zetta Industries](https://www.curseforge.com/minecraft/mc-mods/zetta-industries) - 一个添加了大电池的主多块的模组。
- [Applied Energistics 2](https://github.com/AppliedEnergistics/Applied-Energistics-2) - 一个关于物质、能量并利用它们征服世界的 Minecraft 模组。

### 魔法

- [AbyssalCraft](https://www.curseforge.com/minecraft/mc-mods/abyssalcraft) - AbyssalCraft 是一个以探索为中心的魔法模组。
- [Astral Sorcery](https://www.curseforge.com/minecraft/mc-mods/astral-sorcery) - 这是一个基于星星的魔法模组，利用星光和星座的力量。
- [Avaritia](https://www.curseforge.com/minecraft/mc-mods/avaritia) - Avaritia 是一个有着非常长的进度树的模组，但在游戏后期却非常强大。它的灵魂目的是将残局游戏玩法拖到极限。
- [Blood Magic](https://www.curseforge.com/minecraft/mc-mods/blood-magic) - 这是一个基于血液的魔法模组。当心，你会死很多次。

### 公用事业

- [TNTUtils](https://github.com/ljfa-ag/TNTUtils) - 更好地控制 Minecraft 中的爆炸。
- [Akashic Tome](https://www.curseforge.com/minecraft/mc-mods/akashic-tome) - 一个基于 Morph-o-Tool 的 mod，主要提供合并不同 mod 文档书籍的功能。
- [AppleSkin](https://www.curseforge.com/minecraft/mc-mods/appleskin) - 该模组不会改变任何现有的机制，而只是使现有的饥饿饱和度和疲惫程度在默认 GUI 中可见。
- [Attribute Fixer](https://www.curseforge.com/minecraft/mc-mods/attributefix) - 这个模组修复了《我的世界》中处理护甲点之类的属性系统，因为许多其他模组无意中受到了该系统中的限制的影响。
- [Diet Hopper](https://www.curseforge.com/minecraft/mc-mods/diet-hoppers) - 此 mod 修改了 Minecraft 计算料斗碰撞盒的方式，以便您可以访问其后面的任何可见块。
- [FoamFix](https://www.curseforge.com/minecraft/mc-mods/foamfix-optimization-mod) - 修复了 Minecraft 代码的各个方面，总体上使游戏速度更快。
- [WTHIT](https://www.curseforge.com/minecraft/mc-mods/wthit) - 显示当前查看的块（Hwyla 的新版本）。
- [Hwyla](https://www.curseforge.com/minecraft/mc-mods/hwyla) - 显示当前查看的块（Waila 的新版本）。
- [WorldInfo](https://www.curseforge.com/minecraft/mc-mods/worldinfo) - 显示当前查看的块（Rift 的 Waila）。
- [Waila](https://www.curseforge.com/minecraft/mc-mods/waila) - 显示当前查看的区块（原始 Waila 项目）。
- [Block Helper](https://www.curseforge.com/minecraft/mc-mods/block-helper) - 显示当前查看的区块（Waila 的旧版替代品，适用于早于 1.6 的 Minecraft 版本）。

### 一般

- [ArmorPlus](https://www.curseforge.com/minecraft/mc-mods/armorplus) - 这个模组确实为游戏添加了大量的盔甲，以及不同的制作方法。
- [Baubles](https://www.curseforge.com/minecraft/mc-mods/baubles) - 该模组在玩家库存中添加了七个新插槽，可以安装其他模组添加的各种物品，如护身符、腰带、戒指等。

### 探索

- [Aroma1997's Dimensional World](https://www.curseforge.com/minecraft/mc-mods/aroma1997s-dimensional-world) - 这个模组增加了一个额外的维度，其预期目的是采矿，这意味着主世界将不再被诸如 Buildcraft 采石场之类的东西填满。
- [Twilight Forest](https://www.curseforge.com/minecraft/mc-mods/the-twilight-forest) - 添加了新的暮光森林维度，包含新的结构、头目和需要完成的任务。

### 建筑

- [Carpenter's Blocks](https://www.curseforge.com/minecraft/mc-mods/carpenters-blocks) - 该模组添加了新的块，例如倾斜和角，可以使其看起来像游戏中的任何其他块！对于艺术构建非常有用。 （对于 1.12+，请参阅 BlockCraftery）
- [Chisel](https://www.curseforge.com/minecraft/mc-mods/chisel) - 该模组为游戏中的大量方块添加了多种新纹理，添加了各种各样的纹理。 （要编辑小于块的单位，请参阅凿子和钻头。）
- [WorldEdit](https://github.com/EngineHub/WorldEdit) - WorldEdit 是一款开源游戏内地图编辑器，适用于 Bukkit、Forge、MinecraftEdu 和许多其他平台。
- [FAWE](https://intellectualsites.github.io/download/fawe.html) - 代表快速异步世界编辑。基于 WorldEdit，它速度更快，并且添加了更多工具（即画笔）可供使用。

## 服务器插件

- [AntiCheatReloaded](https://github.com/Rammelkast/AntiCheatReloaded) - AntiCheatReloaded 通过监控和分析玩家的行为，帮助服务器管理员轻松识别和阻止恶意用户。

- [CraftBook](https://github.com/EngineHub/CraftBook) - CraftBook 可让您创建神奇的延伸桥、紧凑的红石电路、复杂的矿车机制等等——所有这些都无需客户端模组，并且可以由服务器完全自定义。
- [Connect](https://connect.minekube.com/) - 打开共享 Minecraft 网络，用于宣传公共和本地主机服务器。
- [LuckPerms](https://luckperms.net/) - 管理权限。
- [CoreProtect](https://github.com/PlayPro/CoreProtect) - 记录玩家所做的一切。确切地说是1984年。

- [Legendary-Java-Minecraft-Geyser-Floodgate](https://github.com/TheRemote/Legendary-Java-Minecraft-Geyser-Floodgate) - PaperMC 服务器使用 docker 提供开箱即用的 Geyser 和 Floodgate 支持。
- [minecraft-docker](https://github.com/tywil04/minecraft-docker) - [Legendary-Java-Minecraft-Geyser-Floodgate](https://github.com/TheRemote/Legendary-Java-Minecraft-Geyser-Floodgate) 的分支，嵌入了一个不和谐的机器人，可以轻松启动和停止服务器。
- [Minestom](https://minestom.net/) - 从头开始的轻量级服务器实现。
- [Mohist](https://mohistmc.com/) - 实现 Forge 和 Bukkit API 的服务器。
- [Paper](https://papermc.io/) - Spigot 和 BungeeCord 的各种补丁提供了更好的性能和更多的配置选项。
- [Spigot](https://www.spigotmc.org/) - CraftBukkit 的后继者提供对 Bukkit API 的支持。
- [Sponge](https://www.spongepowered.org/) - Minecraft Server 提供对 Sponge API 的支持，并可扩展至 Forge API 和 Bukkit API。
- [Thermos](https://cyberdynecc.github.io/Thermos/install) - 为 1.7.10 实现 Forge 和 Bukkit API 的服务器。
- [Gate](https://github.com/minekube/gate) - 用 Go 编写的高性能并行 Minecraft 代理。

### 图书馆

- [MCProtocolLib](https://github.com/Steveice10/MCProtocolLib) - 一个用 Java 编写的简单库，用于与 Minecraft 客户端/服务器进行通信。
