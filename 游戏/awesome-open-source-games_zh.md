## 基于浏览器


### 桌游

- [3D Hartwing Chess Set](https://github.com/juliangarnier/3D-Hartwig-chess-set) - 用 HTML/CSS/JavaScript 完成的 3D 国际象棋游戏。
- [c4](https://github.com/kenrick95/c4) - 四连子游戏，玩家与人工智能对战。
- [Desperate Gods](https://github.com/David20321/FTJ) - 免费的在线棋盘游戏，其设计就像现实生活中的棋盘游戏一样：计算机不强制执行任何规则。
- [Green Mahjong](https://github.com/danbeck/green-mahjong) - 用 HTML/CSS/JavaScript 完成的纸牌麻将游戏。
- [Kriegspiel](https://github.com/binarymax/kriegspiel) - 不完美信息游戏，Kriegspiel 国际象棋的变体。

- [Lichess](https://github.com/ornicar/lila) - 使用 HTML5 和 Websockets 的免费国际象棋游戏，使用 Scala、Play 2.8、MongoDB 和 Elasticsearch 构建。
- [Wolfcha](https://github.com/oil-oil/wolfcha) - AI驱动的狼人（黑手党）社交推理游戏，每个玩家都由顶级法学硕士控制。使用 Next.js 和 TypeScript 构建。
