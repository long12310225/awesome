## 软件

- [Bloxstrap](https://github.com/bloxstraplabs/bloxstrap) - Roblox 的替代引导程序，具有许多额外功能。
- [BTRoblox](https://github.com/AntiBoomz/BTRoblox) - 增强 Roblox 网站外观和功能的浏览器扩展。
- [Roblox Studio Mod Manager](https://github.com/MaximumADHD/Roblox-Studio-Mod-Manager) - Roblox Studio 的开源自定义引导程序，允许您覆盖 Roblox Studio 目录中的文件、选择加入 Roblox 的开发分支并尝试快速标志。
- [Vinegar](https://github.com/vinegarhq/vinegar) - 一个开源、可配置、快速的引导程序，用于在 Linux 上运行 Roblox Studio。

## 模块

### 行政管理

- [Adonis](https://github.com/Epix-Incorporated/Adonis) - 服务器审核和管理系统。
- [Centurion](https://github.com/paradoxuum/centurion) - 一个灵活且可扩展的 roblox-ts 命令框架。
- [Cmdr](https://github.com/evaera/Cmdr) - 为开发人员提供的可扩展命令控制台。
- [Conch](https://github.com/alicesaidhi/conch) - 现代且可扩展的开发者控制台，具有类似 shell 的图灵完备脚本语言。
- [Kohl's Admin](https://github.com/kohls-admin/kohls-admin) - 体验管理的一体化解决方案。
- [Nexus-Admin](https://github.com/TheNexusAvenger/Nexus-Admin) - Roblox 的管理系统适用于普通玩家。

### 背包

- [Neobar](https://github.com/ImAvafe/NeoHotbar) - Roblox 的现代、可定制热栏。
- [Purse](https://github.com/RyanLua/Purse) - Roblox 默认背包的 CoreGui 解耦版本。
- [Satchel](https://github.com/RyanLua/Satchel) - Roblox 默认背包的现代替代品。

### 顶栏

- [TopbarPlus](https://github.com/1ForeverHD/TopbarPlus) - 构建动态且直观的顶栏图标。

### 用户界面组件

- [Lydie](https://github.com/7kayoh/Lydie) - Fusion 的漂亮 UI 组件。
- [OnyxUI](https://github.com/loneka/onyx-ui) - 适用于 Fusion 的快速、可定制的组件。

## 图书馆

### API 包装器

- [noblox.js](https://github.com/noblox/noblox.js) - Roblox 的 Node.js API 包装器。
- [Openblox](https://github.com/cameronpcampbell/openblox) - Roblox 的 API 包装器，用 TypeScript 编写。完全类型安全、可选缓存、使用任何 http 客户端。
- [ro.py](https://github.com/ro-py/ro.py) - Roblox API 的现代异步 Python 3 包装器。

### 数据存储

- [DataKeep](https://github.com/noahrepublic/DataKeep) - 一个基于 Promised 的自动保存 DataStore 库。
- [DocumentService](https://github.com/anthony0br/DocumentService) - 一个强大的、系统化的 Roblox 数据存储库。
- [Lapis](https://github.com/nezuo/lapis) - Roblox 的 DataStore 抽象库。
- [Lyra](https://github.com/paradoxum-games/lyra) - 通过简单的 API 进行高级玩家数据管理。
- [ProfileStore](https://github.com/MadStudioRoblox/ProfileStore) - 具有会话锁定的定期数据存储保存解决方案。

### 实体组件系统

- [ECR](https://github.com/centau/ecr) - Luau 的基于稀疏集的 ECS 库。
- [Jecs](https://github.com/Ukendio/jecs) - 一个快速、可移植的 Luau 实体组件系统。
- [Matter](https://github.com/matter-ecs/matter) - Roblox 的现代 ECS 库。
- [Planck](https://github.com/YetAnotherClown/planck) - ECS 的不可知调度程序。
- [Replecs](https://github.com/PepeElToro41/replecs) - 快速、精细控制的 JECS 复制库。

### 框架

- [Flamework](https://github.com/rbxts-flamework/core) - roblox-ts 的可扩展游戏框架。
- [Prvd 'M Wrong](https://github.com/prvdmwrong/prvdmwrong) - Roblox 游戏框架采用现代标准构建，提供类型安全的 API、依赖项解析和无与伦比的 DX。

### 网络

- [Blink](https://github.com/1Axen/blink) - 用 Luau 编写的 IDL 编译器，用于 ROBLOX 缓冲区网络。
- [ByteNet](https://github.com/ffrostfall/ByteNet) - 先进的、现代的网络库。
- [RbxNet](https://github.com/roblox-aurora/rbx-net) - 先进的多语言网络框架。
- [Red](https://github.com/red-blox/red) - 简单、快速且功能强大的网络库。
- [Zap](https://github.com/red-blox/zap) - 闪电般快速的网络解决方案。

### 开放云

- [rbxcloud](https://github.com/Sleitnick/rbxcloud) - Roblox 开放云 API 的 CLI 和库。

### 光线投射

- [ZonePlus](https://github.com/1ForeverHD/ZonePlus) - 构建动态区域以有效地确定其边界内的玩家和部分。

### 用户界面

- [fluid](https://github.com/ffrostfall/fluid) - Roblox 的声明式 UI 框架。
- [Fusion](https://github.com/dphfox/Fusion) - 专为 Roblox 和 Luau 构建的现代反应式 UI 库。
- [Iris](https://github.com/Michael-48/Iris) - Roblox 的即时模式 GUI 库，基于 [Dear ImGui](https://github.com/ocornut/imgui)。
- [React Lua](https://github.com/jsdotlua/react-lua) - 将上游 ReactJS 17.x 全面但不详尽地翻译为 Lua。
- [Vide](https://github.com/centau/vide) - 用于创建 UI 的反应式 Luau 库。

### 实用性

- [Janitor](https://github.com/howmanysmall/Janitor) - 轻量级、灵活的对象，用于清理连接、实例等。
- [Nevermore](https://github.com/Quenty/NevermoreEngine) - ModuleScript 加载程序具有可重用且简单的统一服务器客户端模块，可加快 Roblox 上的游戏开发速度。
- [RbxUtil](https://github.com/Sleitnick/RbxUtil) - Roblox 实用程序模块的集合。
- [t](https://github.com/osyrisrblx/t) - Roblox 的运行时类型检查器。

## 工装

### 编译器

- [darklua](https://github.com/seaofvoices/darklua) - 使用规则转换 Lua 5.1 和 Roblox Lua 脚本。
- [Lune](https://github.com/lune-org/lune) - 独立的 Luau 运行时。
- [roblox-ts](https://github.com/roblox-ts/roblox-ts) - Roblox 的 TypeScript 到 Luau 编译器。

### 文档生成器

- [Moonwave](https://github.com/evaera/moonwave) - 用于根据 Lua 源代码中的注释生成文档的工具。

### 语言服务器

- [Luau Language Server](https://github.com/JohnnyMorganz/luau-lsp) - Luau 语言服务器实现。
- [Roblox Luau Language Server](https://github.com/NightrainsRbx/RobloxLsp) - sumneko 基于 Lua 的 Roblox 语言服务器。

### 短绒/格式化程序

- [selene](https://github.com/Kampfkarren/selene) - 用 Rust 编写的速度极快的现代 Lua linter。
- [StyLua](https://github.com/JohnnyMorganz/StyLua) - 一个固执己见的 Lua 代码格式化程序。

### 开放云

- [Mantle](https://github.com/blake-mealey/mantle) - Roblox 的基础设施即代码和部署工具。

### 包管理器

- [pesde](https://github.com/pesde-pkg/pesde) - Luau 编程语言的包管理器。
- [Wally](https://github.com/UpliftGames/wally) - 受 Cargo 启发的 Roblox 项目的现代包管理器。

### 同步工具

- [Argon](https://github.com/argon-rbx/argon) - 用于 Roblox 开发的全功能工具。
- [Lync](https://github.com/Iron-Stag-Games/Lync) - 文件同步工具。
- [Rojo](https://github.com/rojo-rbx/rojo) - 使 Roblox 开发人员能够使用专业级软件工程工具。

### 工具链管理器

- [Foreman](https://github.com/Roblox/foreman) - Roblox 项目的工具链管理器。
- [Rokit](https://github.com/rojo-rbx/rokit) - Roblox 项目的下一代工具链管理器。
