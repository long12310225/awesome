## 二维

* [2D Rope System (Paid)](https://assetstore.unity.com/packages/tools/sprite-management/2d-rope-system-17722) - 用于在编辑器中或运行时创建任何类型的 2D 绳索的脚本。
* [Ferr2D Terrain Tool (Paid)](https://assetstore.unity.com/packages/tools/level-design/ferr2d-terrain-tool-11653) - 快速创建手工制作的 2D 景观和关卡。
* [Pixel Camera 2D](https://github.com/RyanNielson/PixelCamera2D) - 一个简单的像素完美相机，带有 2D 游戏的缩放选项。
* [Spine (Paid)](http://esotericsoftware.com) - 带有 Unity 库的骨骼动画编辑器。
* [Tiled2Unity](http://www.seanba.com/tiled2unity) - 获取您的 [Tiled](http://www.mapeditor.org) 文件并从中创建可轻松放入 Unity 场景中的 Unity 预制件。 Unity 的 PolygonCollider2D 类支持复杂碰撞。
* [Unity Anima2D](https://assetstore.unity.com/packages/2d/characters/anima2d-no-longer-supported-replaced-by-2d-animation-79840) - 高级骨骼动画编辑器，支持每个对象和蒙皮网格动画，并带有集成的编辑器内蒙皮工具。
* [UnityTiled](https://github.com/nickgravelyn/UnityTiled) - [Tiled](http://www.mapeditor.org) 地图的导入器。

## 人工智能

* [A* Pathfinding Project](http://arongranberg.com/astar/) - 通过高度优化的算法和大量的功能集实现闪电般的快速寻路。
* [Apex Path (Paid)](https://assetstore.unity.com/packages/tools/ai/apex-path-17943) - Apex Path 处理动态寻路，包括局部回避转向和动态障碍物。

* [Crystal AI](https://github.com/igiagkiozis/CrystalAI) - Crystal 是一个快速、可扩展且可扩展的基于实用程序的 C# 和 Unity AI 框架。

## 增强现实和虚拟现实
* [ARToolKit](http://artoolkit.org/documentation/doku.php?id=6_Unity:unity_about) - 增强现实 SDK，包括库、实用程序和示例。
* [Google VR SDK](https://developers.google.com/vr/unity) - 脚本和预制件可帮助开发适用于 Android 和 iOS 的 Google Daydream 和 Cardboard 应用程序。
* [SteamVR Unity Toolkit](https://assetstore.unity.com/packages/tools/integration/vrtk-virtual-reality-toolkit-vr-toolkit-64131) - 脚本和很好的例子来抽象 Unity 中 VR 控制器动作的使用。
* [Virtual Reality Toolkit](http://github.com/thestonefox/vrtk) - 虚拟现实框架，可实现强大的交互、运动和视觉效果。
* [Vuforia](https://vuforia.com/) - 增强现实 SDK 具有图像和对象识别、智能地形和扩展跟踪功能。

## 相机

* [UFPS (Paid)](https://assetstore.unity.com/packages/templates/systems/ufps-ultimate-fps-2943) - 提供用于 FPS 游戏的相机、控制器和其他效果。

## 角色控制器

* [CharacterController2D](https://github.com/prime31/CharacterController2D) - 2D 控制器，其行为与 Unity 的 CharacterController 组件非常相似。

## 框架

* [Fungus](https://github.com/snozbot/fungus) - 易于使用的 Unity 3D 库，用于创建插图互动小说游戏。
* [StrangeIoC](http://strangeioc.github.io/strangeioc/) - Strange 是一个超轻量级且高度可扩展的控制反转 (IoC) 框架，专为 C# 和 Unity 编写。
* [uFrame (Paid)](https://assetstore.unity.com/packages/tools/visual-scripting/uframe-game-framework-14381) - 比以往更快、更好、更稳定、更一致地创建可维护的游戏。

## 输入

* [InControl](https://github.com/pbhogan/InControl) - 驯服的输入管理器使处理程序跨平台。控制器输入简单。
* [InputBinder](https://github.com/RyanNielson/InputBinder) - 通过代码将游戏输入绑定到方法或使用检查器将事件驱动的输入处理添加到您的项目中。
* [TouchKit](https://github.com/prime31/TouchKit) - 可以轻松识别手势和其他触摸输入。
* [TouchScript](https://github.com/TouchScript/TouchScript) - 使在任何触摸表面上处理复杂的手势交互变得更加容易。

## 造型
* [SabreCSG](http://sabrecsg.com/) - 一组 [CSG](https://en.wikipedia.org/wiki/Constructive_solid_geometry) 关卡设计工具，用于在 Unity 中快速构建复杂关卡。

## 货币化
* [Unity Monetization](https://assetstore.unity.com/packages/add-ons/services/unity-monetization-66123) - Unity Ads 是一个视频广告网络，使用常规广告和选择性广告进行快速无缝集成。

## 网络

* [Nakama](https://assetstore.unity.com/packages/tools/network/nakama-81338) - 使用开源[分布式服务器](https://github.com/heroiclabs/nakama)构建社交和实时游戏。
* [Photon Bolt (Paid)](https://assetstore.unity.com/packages/tools/network/photon-bolt-free-127156) - 构建网络游戏，无需了解网络细节或编写任何复杂的网络代码。
* [Photon Unity Networking](https://assetstore.unity.com/packages/tools/network/photon-unity-networking-classic-free-1786) - 即插即用的云网络也适用于本地托管。最多 20 个并发用户免费。

## 脚本编写
* [Easy Save 2 (Paid)](https://assetstore.unity.com/packages/tools/input-management/easy-save-the-complete-save-load-asset-768) - 在 Unity 支持的所有主要平台上保存和加载数据的快速而简单的方法。
* [UniStorm (Paid)](https://assetstore.unity.com/packages/tools/particles-effects/unistorm-volumetric-clouds-sky-modular-weather-and-cloud-shadows-2714) - 可定制的动态昼夜天气系统，可创建逼真的风暴和天气。

## 服务
* [Unity Analytics](https://docs.unity3d.com/Manual/UnityAnalyticsSetup.html) - 提供包含指标的仪表板，以帮助跟踪活跃玩家、会话、保留率和收入。

## 补间

* [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676) - 补间任何数字属性或字段（包括向量、矩形等）以及一些非数字属性或字段（如字符串）。这是 HOTween 的后续。
* [GoKit](https://github.com/prime31/GoKit) - 一个开源的轻量级补间库，旨在使补间对象变得非常简单。
* [iTween](https://assetstore.unity.com/packages/tools/animation/itween-84) - 一个简单易用的动画系统。
* [LeanTween](https://assetstore.unity.com/packages/tools/animation/leantween-3595) - FOSS，也是 Unity 最轻量级的补间库。允许您补间您可以通过 .value() 方法访问的任何值。

## 用户界面

* [NGUI (Paid)](https://assetstore.unity.com/packages/tools/gui/ngui-next-gen-ui-2413) - 强大的UI系统和事件通知框架。
* [TextMesh Pro](https://docs.unity3d.com/Packages/com.unity.textmeshpro@1.3/manual/index.html) - 通过使用距离场技术在 uGUI 中渲染文本的替代方法，该技术允许在任何比例下呈现清晰的字体。这是最近被Unity购买的，将来会集成到引擎中。

## 公用事业

* [Consolation](https://github.com/mminer/consolation) - 游戏内调试控制台显示“Debug.Log”的输出。
* [GitHub for Unity](https://unity.github.com/) - 新的 GitHub for Unity 扩展将 GitHub 工作流程等引入 Unity，通过 Git LFS 和文件锁定提供对大文件的支持。
* [Grouping Tool](https://assetstore.unity.com/packages/tools/utilities/grouping-tool-147552) - 轻松将对象分组在一起
* [Scene View Bookmarks](https://github.com/mminer/scene-view-bookmarks) - 编辑器扩展可添加书签并在以后调用场景视图。
* [SnazzyGrid (Paid)](https://assetstore.unity.com/packages/tools/level-design/snazzygrid2-19245) - 通过易于使用的捕捉工具和更多功能来改进场景创建工作流程，可以轻松管理场景中资产的位置。
* [UniMerge (Paid)](https://assetstore.unity.com/packages/tools/version-control/unimerge-9733) - 用于合并场景和预制件的编辑器扩展，还与 VCS 集成。
* [UniRx](https://github.com/neuecc/UniRx) - UniRx（Unity 反应式扩展）是 .NET 反应式扩展的重新实现。 Rx 在没有 async/await 的情况下治愈了“异步忧郁症”。
* [UnityToolbag](https://github.com/nickgravelyn/unitytoolbag) - Unity 5.0 的各种开源脚本和帮助程序的集合。

## 视频

* [Vimeo Unity SDK](https://github.com/vimeo/vimeo-unity-sdk) - 轻松将您的 Vimeo 视频流式传输到 Unity 中或录制并发布到 Vimeo。

## 可视化脚本

* [Playmaker (Paid)](https://assetstore.unity.com/packages/tools/visual-scripting/playmaker-368) - 快速制作游戏原型，A.I.使用有限状态机的行为、动画图、交互式对象等。

# 资源

## 提示和技巧

* [Editor Tips](http://imgur.com/a/2w7zd) - gif 形式的提示显示了更有效地使用编辑器的几种方法。
* [Unity Labs' Super Science](https://github.com/Unity-Technologies/SuperScience) - 用于用户教育的 Unity Labs Gems。
* [Unity Tips](https://unity3d.com/learn/tutorials/topics/tips) - 短片展示了使用 Unity 时的一些实用技巧。

## 教程

* [2D Splatter Effects Using the Stencil Buffer](http://nielson.dev/2015/12/splatter-effects-in-unity-using-the-stencil-buffer) - 使用 Unity 中的模板缓冲区在表面上绘制飞溅效果。这可用于油漆或血迹。
* [A Gentle Introduction to Shaders in Unity3D](http://www.alanzucconi.com/2015/06/10/a-gentle-introduction-to-shaders-in-unity3d) - 本系列文章将向您介绍着色器编码，并且面向对着色器知之甚少或一无所知的开发人员。
* [Amit’s Game Programming Information](http://www-cs-students.stanford.edu/~amitp/gameprog.html) - 通用游戏编程内容的精彩集合。
* [Catlike Coding](http://catlikecoding.com/unity/tutorials/) - 专为学习 Unity 的 C# 脚本方面而设计的教程。
* [Fixing Gaps Between Sprites](http://nielson.dev/2015/10/fixing-gaps-between-sprites-better-2d-in-unity-part-2) - 关于消除相邻精灵之间有时出现的小间隙的简短教程。
* [Game Programming Patterns](http://gameprogrammingpatterns.com/contents.html) - 无论是否使用 Unity 制作游戏，许多出色的游戏开发模式都很有用。
* [Modern GUI Development in Unity 4.6](https://www.youtube.com/playlist?list=PLt_Y3Hw1v3QTEbh8fQV1DUOUIh9nF0k6c) - 视频教程深入解释 Unity 的新 UI 系统。
* [Official Video Tutorials](http://unity3d.com/learn/tutorials/modules) - 有关脚本、动画、音频以及几乎所有 Unity 相关内容的官方教程。
* [Ray Wenderlich's Tutorials](http://www.raywenderlich.com/category/unity) - 初级和中级教程侧重于学习 Unity 功能或创建小示例游戏。
* [Unity in HoloLens](https://developer.microsoft.com/en-us/windows/holographic/unity_development_overview) - Microsoft 提供的用于创建 Microsoft HoloLens 应用程序的官方教程。
* [Unity Virtual Reality](http://docs.unity3d.com/Manual/VROverview.html) - Unity 有关开发虚拟现实应用程序的官方文档。
