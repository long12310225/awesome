此外，数据挖掘（DM）是发现数据集（或数据集）中模式的过程，涉及机器学习、统计和数据库系统的方法； DM 专注于提取数据集的信息_Han (2011)_。

## 应用程序编程接口

API 是“一组功能和过程，允许创建访问操作系统、应用程序或其他服务的功能或数据的应用程序”（Google）。

- [Battle.net](https://develop.battle.net/) - [暴雪](https://www.blizzard.com) 开发的游戏合集。
- [Battlefield 4 Stats](https://bf4db.com/stats) - 球员的统计数据、排名和进步。
- [BoardGameGeek](https://boardgamegeek.com/wiki/page/BGG_XML_API2) - 棋盘游戏数据。
- [Counter-Strike](https://github.com/pnxenopoulos/csgo) - 《反恐精英：全球攻势》(CSGO) 的数据解析。
- [Giant Bomb](https://www.giantbomb.com/api/) - 游戏数据和玩家评论。
- [IGDB](https://www.igdb.com/api) - 来自任何平台的游戏的一般信息。
- [Marvel Developer](https://developer.marvel.com/) - 有关 Marvel 庞大漫画库的信息。
- [metacritc](https://www.metacritic.com/game) - 游戏评论和评价。
- [NEXARDA](https://www.nexarda.com/api) - 游戏和价格数据。
- [OpenCritic](https://www.opencritic.com) - 游戏评论聚合器。
- [OpenDota](https://www.opendota.com/) - 提供Dota 2数据的平台。
- [PandaScore](https://pandascore.co/) - 实时电子竞技数据。
- [PixelGamesHub AI Games Index](https://pixelgameshub.com/ai-games/index) - 具有开放 JSON 和 CSV 端点的人工智能浏览器游戏的精选索引。
- [PokéAPI](https://pokeapi.co/) - 所有世代的神奇宝贝数据。
- [Riot Games](https://developer.riotgames.com/) - 活跃比赛、比赛历史记录和排名统计数据。
- [smm-course-search](https://github.com/leomaurodesenv/smm-course-search) - 从《超级马里奥创客》游戏中搜索课程。
- [smm-course-viewer](https://github.com/leomaurodesenv/smm-course-viewer) - 阅读《超级马里奥制造》保存的课程。
- [smm-maker-profile](https://github.com/leomaurodesenv/smm-maker-profile) - 从 Super Mario Maker 游戏中获取用户个人资料。
- [Steam Database](https://github.com/SteamDatabase) - Steam 数据系列工具。 [网站](https://steamdb.info/)。
- [Steam Spy](https://github.com/topics/steamspy) - 来自 Steam 用户个人资料的游戏统计。
- [Steam Web API](https://developer.valvesoftware.com/wiki/Steam_Web_API) - 来自 Steam 的查询工具。
- [TGDB](https://github.com/TheGamesDB/TheGamesDB/) - 来自任何平台的游戏的一般信息。 [网站](https://thegamesdb.net/)。
- [TrendingNow.games](https://trendingnow.games/data-feeds) - 实时趋势 Steam 游戏数据，包含每小时更新的免费 CSV、RSS 和 JSON 源。
- [vgchartzScrape](https://github.com/GregorUT/vgchartzScrape) - 来自 [VGChartz](http://www.vgchartz.com) 的爬虫。
- [WhatoPlay](https://whatoplay.com) - 游戏评论和评级聚合器，以及发现游戏的推荐器。
- [Xbox LIVE API](https://github.com/xboxapi) - 游戏、应用程序、用户统计数据和消息。 [论文](https://doi.org/10.1145/1943552.1943569)。
