## 适合初学者

### YouTube 播放列表
* [Khan Academy Algorithms](https://www.khanacademy.org/computing/computer-science/algorithms) - 算法课程由 Tomas Cormen 和 Devin Balkcom 主持。
* 免费代码营。算法和数据结构：[5 小时](https://www.youtube.com/watch?v=8hly31xKli0)、[49 小时](https://youtu.be/xwI5OBEnsZU?si=W3CduIEmUehJERHk) - 完整的初学者友好的算法和数据结构教程，带有令人兴奋的动画。

### 预订
* [Real World Algorithms: A Beginner's Guide](https://mitpress.mit.edu/books/real-world-algorithms) - 为没有高等数学或计算机科学背景的读者介绍算法。
* [Levitin. Introduction to Algorithms](https://www.amazon.com/Introduction-Design-Analysis-Algorithms-3rd/dp/0132316811) - 以谜题的形式非常容易理解的算法说明。无需编程经验！

### 互动学习与可视化
* [VisuAlgo](http://visualgo.net) - 通过动画可视化数据结构和算法。
* [See Algorithms](https://see-algorithms.com/) - 可视化算法并允许用户创建和共享自定义图形和二叉树。
* [Recursive](https://recursive.oilater.com/en) - 通过调用树、变量跟踪和行突出显示逐步可视化代码执行。支持 Python 和 JavaScript/TypeScript。

## 编程竞赛

### 网站
* [CP-Algorithms](https://cp-algorithms.com/) - 算法和数据结构在竞争性编程领域尤其受欢迎。

### 预订
- [史蒂文·哈利姆.竞争性编程书籍](https://cpbook.net/)

### 竞赛网站
* [HackerEarth](https://www.hackerearth.com/) - 适合初学者的教程。
* [Code Chef](https://www.codechef.com/contests) - 初学者可以解决的问题。
* [Codeforces](http://codeforces.com/) - 具有挑战性的问题。
* [TopCoder](https://www.topcoder.com/) - 由科技公司赞助。
* [Leet Code](https://leetcode.com/) - 面试问题。
* [UVA](https://uva.onlinejudge.org/) - 存档之前的 ACM-ICPC 地区决赛和世界总决赛。
* [ProjectEuler](https://projecteuler.net/) - 数学导向的问题。

### 博客
* [An awesome list for competitive programming!](https://codeforces.com/blog/entry/23054) - 很棒的博客，包含所有资源以及书籍和算法列表。
* [Algorithms Weekly](https://petr-mitrichev.blogspot.com/) - Petr Mitrichev 写的一个很好的博客，主要是 Java 语言。
* [Algorithm Tutorials by Tanuj Khattar](https://tanujkhattar.wordpress.com/) - Tanuj Khattar 的优秀博客。涵盖一些有趣的数据结构的教程以及要解决的示例问题。

## 理论与基础知识

### 预订
* [Introduction to Algorithms](https://mitpress.mit.edu/9780262046305/introduction-to-algorithms/) - 适合任何人的标准经典。
* [The Art of Computer Programming](http://www-cs-faculty.stanford.edu/~uno/taocp.html) - 高德纳的经典之作。
* [The Algorithm Design Manual](http://www.algorist.com/) - 易于阅读并充满现实世界的例子。

### 在线课程
* [MIT - 6-006](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-fall-2011/video_galleries/lecture-videos/) - 很好解释的算法。
* [MIT - 6-046j](https://ocw.mit.edu/courses/6-046j-introduction-to-algorithms-sma-5503-fall-2005/video_galleries/video-lectures/) - 与上一个类似，但算法不同。
* [MIT - 6-00sc](http://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-00sc-introduction-to-computer-science-and-programming-spring-2011/index.htm) - 简单易懂的算法介绍。

### YouTube 播放列表
* [Tim Roughgarden. Algorithms Illuminated](https://www.algorithmsilluminated.org/) - 计算机科学先驱蒂姆·拉夫加登 (Tim Roughgarden) 为他的书提供了一个配套教材，其中包含讲座视频和幻灯片。

## 生产体系

* [Performance Engineering of Software Systems. MIT OCW](https://ocw.mit.edu/courses/6-172-performance-engineering-of-software-systems-fall-2018/) - 基于项目的实践介绍如何构建可扩展的高性能软件系统。主题包括性能分析、高性能算法技术、指令级优化和缓存优化。
* [Cloud and Distributed Computing: Algorithms and Systems by Misra & Patel](https://www.wileyindia.com/cloud-and-distributed-computing-algorithms-and-systems.html) - 它教授云计算的内部结构以及分布式系统的概念如何在云中工作。

## 数据库

* [PostgreSQL Query Optimization: The Ultimate Guide to Building Efficient Queries](https://www.oreilly.com/library/view/postgresql-query-optimization/9781484268858/) - 它介绍了不同类别的查询和适合每种查询的优化技术，例如索引的使用和特定的连接算法。

## 专题

* [Fabulous Adventures in Data Structures and Algorithms](https://www.manning.com/books/fabulous-adventures-in-data-structures-and-algorithms) - 它教授鲜为人知的算法方法，例如不可变的数据结构——堆栈、队列、双端队列，以及休斯列表等不寻常的结构。
