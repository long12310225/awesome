### 算法书籍

| ☆ |名称 |描述 |
| --- | --- | --- |
| ★★★ | [算法简介](https://mitpress.mit.edu/books/introduction-algorithms)，作者：Thomas H. Cormen、Charles E. Leiserson、Ronald L. Rivest 和 Clifford Stein |本书也称为 CLRS（取自名字首字母），通常被称为算法和数据结构的“圣经”。它是大学算法课程最受欢迎的教科书之一。本书非常详细地介绍了各种算法和数据结构。写作更加严谨，对某些人来说可能很难。 |
| ★★☆ | [算法设计](http://amzn.to/VjhioK)，作者：Jon Kleinberg 和 Éva Tardos |本书围绕设计算法的技术展开。它组织良好，以清晰易懂的语言编写。每一章都配有实际示例和有用的练习。关于网络流量的章节受到了很多人的好评。 ... 教科书随附的讲座幻灯片可在其[官方网站](http://www.cs.princeton.edu/~wayne/kleinberg-tardos/) 上找到。 |
| ★★☆ | [算法设计手册](https://www.amazon.com/Algorithm-Design-Manual-Steven-Skiena/dp/1848000693)，作者：Steven S. Skiena |这本书是用更易读的文字写成的。有些人认为它比其他书更全面。您还可以在其[官方网站](http://www.algorist.com)上找到一些不错的资源（包括作者自己的视频讲座）。 |
| ★★★ | [算法](https://www.amazon.com/Algorithms-4th-Robert-Sedgewick/dp/032157351X)，作者：Robert Sedgewick 和 Kevin Wayne |这本书分类清晰，配有详尽的解释和精美的插图。它被一些IOI训练营用作教材。 |
|     | [高级算法和数据结构](https://www.manning.com/books/advanced-algorithms-and-data-structs)，作者：Marcello La Rocca |本书提供了一种不同的算法方法，从更实用的角度平衡理论，每章都有一个部分重点介绍如何将算法应用于日常工作或竞争性编程中可以找到的实际用例；它还融合了经典、先进和新算法。 |
| ★★★ | [算法](https://jeffe.cs.illinois.edu/teaching/algorithms/)，作者：Jeff Erickson |自行出版的教科书的免费电子版，经 CC 4.0 许可。这是一本写得很好的书，来自伊利诺伊大学理论计算机科学课程的讲义。涵盖了回溯、动态规划、贪婪的主要范例，特别是深入的图。 |
|     | [Grokking 数据结构](https://www.manning.com/books/grokking-data-structs)，作者：Marcello La Rocca | 《Grokking 数据结构》仔细指导您从最基本的数据结构（如数组或链表）一直到强大的结构（如图形）。它非常适合初学者，你只需要高中数学就可以开始。您遇到的每个数据结构都有其自己的完整 Python 实现，因此您可以立即开始尝试您所学到的内容。 |
