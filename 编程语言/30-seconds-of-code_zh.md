# 30秒的代码

> 编写文章来提高您的开发技能

* 访问【网站】(https://30secondsofcode.org)查看所有文章。
* 使用名称、标签、语言或描述搜索适合您需求的文章和集合。
* 浏览每个主题的所有[文章](https://30secondsofcode.org/snippets/p/1) 或单个[集合](https://30secondsofcode.org/collections/p/1)。
* 单击每张卡片可查看整篇文章，包括代码片段、说明和示例。
* 如果您喜欢该项目，请给它一颗星。这意味着很多。
