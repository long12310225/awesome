[CircuitPython](https://learn.adafruit.com/welcome-to- Circuitpython/what-is- Circuitpython) 是 Adafruit 的 MicroPython 分支，旨在简化低成本微控制器的实验和教育。由于不需要预先下载桌面软件，因此比以往任何时候都更容易开始原型设计。使用 CircuitPython，您可以编写干净、简单的 Python 代码来控制硬件，而不必使用复杂的低级语言，如 C 或 C++（Arduino 用于编程的语言）。这对于初学者来说非常棒！

## CircuitPython.org

[![CircuitPython.org](./assets/circuitpython-org.png)](https://circuitpython.org/)

[CircuitPython.org](https:// Circuitpython.org/)是信息蛇高速公路的第一站。在这里，您可以查看所有[支持的主板并下载](https:// Circuitpython.org/downloads)每个主板的特定固件，包括测试版、最新版和以前的版本。下载可以按以下方式排序：制造商、功能、名称等。目前 CircuitPython 系列支持超过 500 个开发板！

## 贡献

随时欢迎您的贡献和建议！请从您自己的分支发出拉取请求以修改 Awesome CircuitPython。

这里详细解释了工作流程：[Adafruit：使用 Git 和 GitHub 为 CircuitPython 做出贡献](https://learn.adafruit.com/contribute-to- Circuitpython-with-git-and-github/overview)。

### 贡献工作流程概述

1. 在 GitHub 上分叉 [此存储库](https://github.com/adafruit/awesome- Circuitpython)。
1. 将分叉的存储库克隆到您的驱动器。
1. 使用所有者 adafruit 和原始 URL 添加远程。
1. 拿起遥控器。
1. 合并 adafruit 或 main.
1. 创建分支。
1. 做出改变。
1. 提交并推送到您的存储库。
1. 在官方存储库上创建拉取请求。
1. 一旦接受，请重复从官方存储库合并到您的分支的主控，然后重新分支。

问题？我们很高兴[帮助您在 Discord 上做出贡献](https://discord.com/invite/5FBsBHU)。
