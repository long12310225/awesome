> 精选的 Imba 框架、库、软件和资源列表

Imba 是一种受 Ruby 和 Python 启发的编程语言，通过内置转译器和 webpack 加载器编译为高性能且可读的 JavaScript。

## 资源

#### 官方资源

#### 工具包和框架

- [Start Imba](https://github.com/athif23/start-imba) - 用于引导 Imba 保护的工具，灵感来自 Create React App 和 Vue CLI。
- [Hello World Imba](https://github.com/imba/hello-world-imba) - 官方 Imba“Hello World”示例。您也可以将其用作 GitHub 模板。
- [Imba Router](https://github.com/somebee/imba-router) - Imba 的实验路由器。
- [UXA](https://github.com/somebee/uxa) - Imba 的实验框架。
- [Parcel Plugin Imba](https://github.com/imba/parcel-plugin-imba) - [Parcel Bundler](https://parceljs.org/) 插件启用
Imba 文件的处理。
- [Material Imba](https://github.com/nathanjohnson320/material-imba) - 用 Imba 编写的 Material Design 组件。
- [Imba Inertia Adapter](https://github.com/haikyuu/imba-inertia-adapter) - Imba [Inertia](https://inertiajs.com) 适配器。
- [Formidable](https://www.formidablejs.org/) - 用于快速 API 开发的 Imba 框架。
- [Imba Shell](https://github.com/donaldp/imba-shell) - Imba 的交互式调试器和 REPL。

#### 编辑器插件和 IDE

- [Atom](http://github.com/somebee/language-imba) - Atom 的 Imba 插件。
- [Sublime Text](http://github.com/somebee/sublime-imba) - Sublime 的 Imba 插件。
- [VSCode](http://github.com/somebee/vscode-imba) - VSCode 的 Imba 插件。


#### 指南和教程

- [官方指南](https://imba.io/guides)
- [Imba 更深入指南](https://imba.github.io/imba-guide/)
- [Hello World Imba Tutorial](https://github.com/jiggneshhgohel/hello-world-imba-tutorial) - 这是一个详细的教程，记录了使用 Node.js、Imba 和 Webpack 构建“Hello World”应用程序所采取的步骤。
- [Why is Imba so fast?](https://scrimba.com/p/pJkZsB/c6B9rAM) - 了解 Imba 为什么很快、它是如何工作的，然后尝试一下。
- [Quick introduction to Imba Router](https://scrimba.com/playlist/pMvYcg) - 了解 imba-router 在 Imba 中的工作原理。
- [Creating a todo-list app in Imba](https://scrimba.com/p/pDzDSZ/cRvRMSB) - 了解如何使用 Imba 更快、更轻松地创建应用程序。


#### 文章

- [Imba 简介：与 JavaScript 兼容的语言，可实现闪电般快速的 DOM 更新 - freeCodeCamp](https://medium.freecodecamp.org/introduction-to-imba-the-alternative-to-javascript-e2aa1e3d1769)
- [React 基于 JavaScript 的新 Web 语言挑战 - ADTMag](https://adtmag.com/articles/2016/01/14/imba-web-language.aspx)


## 使用 Imba 构建

#### 现实世界的应用程序

- [Scrimba](http://scrimba.com) - 交互式屏幕广播平台 Scrimba.com 的前端和后端都是用 Imba 编写的。
- [Iceland Fish Auction Market](https://rsf.is) - 冰岛鱼市的网站。
- [GitSpeak](https://gitspeak.com) - 面向开发人员的项目管理工具。


#### 工具

- [html2imba](https://github.com/konsumer/html2imba) - [在线查看](http://konsumer.js.org/html2imba/) - 一个将HTML转换为Imba标签源的工具。
- [JSON pretty printer](https://github.com/taw/imba-json-beautifier) - [在线查看](https://taw.github.io/imba-json-beautifier) - JSON 更漂亮。
- [Monoalphabetic Cipher Decoder](https://github.com/taw/imba-monoalphabetic) - [在线查看](https://taw.github.io/imba-monoalphabetic) - 单字母密码的解码器。
- [RegExp Explorer](https://github.com/taw/imba-regexp-explorer) - [在线查看](https://taw.github.io/imba-regexp-explorer) - 正则表达式工具。


#### 应用程序/网站

- [Imba Todo MVC](https://github.com/somebee/todomvc-imba) - Imba TodoMVC 示例。
- [Other Todo MVC](https://github.com/shapkarin/imba-todo) - [在线查看](https://shapkarin.github.io/imba-todo/) - TodoMVC 实现遵循 [tastejs/todomvc](https://github.com/tastejs/todomvc) 规​​则。
- [Simple Todo](https://github.com/taw/imba-todo-list) - [在线查看](https://taw.github.io/imba-todo-list) - 更基本的待办事项列表应用程序。
- [Imba Realworld App](https://github.com/cartonalexandre/imba-realworld-example-app) - [在线查看](https://imba-realworld-example-app.netlify.com/) - 创建项目是为了演示一个完全成熟的全栈应用程序，该应用程序遵守 [RealWorld](https://github.com/gothinkster/realworld) 规范和 API。
- [Imba HackerNews Clone](https://github.com/SamirHodzic/imba-capacitor-hn) - 用于构建 Hacker News 克隆的 Imba 和 Capacitor 演示。
- [2048 Game](https://github.com/taw/imba-2048) - [在线查看](https://taw.github.io/imba-2048) - 2048游戏。
- [Calculator](https://github.com/taw/imba-calculator) - [在线查看](https://taw.github.io/imba-calculator) - 计算器。
- [Emoji Memory Game](https://github.com/taw/imba-emoji-memory) - [在线查看](https://taw.github.io/imba-emoji-memory) - 记忆游戏。
- [Eyes](https://github.com/taw/imba-eyes) - [在线查看](https://taw.github.io/imba-eyes) - 一个很酷的交互网站。
- [Fifteen Puzzle](https://github.com/taw/imba-fifteen) - [在线查看](https://taw.github.io/imba-fifteen) - 十五个谜题。
- [Game of Life](https://github.com/taw/imba-game-of-life) - [在线查看](https://taw.github.io/imba-game-of-life) - 生命游戏。
- [Matrix Rain](https://github.com/taw/imba-matrix-rain) - [在线查看](https://taw.github.io/imba-matrix-rain) - 酷炫的类似矩阵的雨动画。
- [Quote Machine](https://github.com/taw/imba-quote-machine) - [在线查看](https://taw.github.io/imba-quote-machine) - 生成随机报价的应用程序。
- [ROT-N](https://github.com/taw/imba-rotn) - [在线查看](https://taw.github.io/imba-rotn) - ROT-N 应用程序。
- [Snake Game](https://github.com/taw/imba-snake) - [在线查看](https://taw.github.io/imba-snake) - 贪吃蛇游戏。
- [Weather App](https://github.com/taw/imba-weather) - [在线查看](https://taw.github.io/imba-weather) - 天气应用程序。
- [Top Down Shooter](https://github.com/CassianoSF/imba-top-down-shooter) - [在线查看](https://cassianosf.github.io/imba-top-down-shooter/dist/index.html) - 2D僵尸射击游戏。
- [Ping CRM](https://github.com/haikyuu/perfect-stack) - CRUD 应用程序，具有登录、帐户、组织和联系人管理功能。使用 Inertia、Edgedb 和 Imba 构建
