## 入门
- [GNU Emacs 手册](https://www.gnu.org/software/emacs/manual/) - 官方 Emacs 手册。
- [Emacs Wiki](https://www.emacswiki.org/) - Emacs 社区维基。
- [Emacs 入门指南](https://masteringemacs.org/) - Mastering Emacs。

## 包管理器
- [MELPA](https://melpa.org/) - Emacs 包仓库。
- [ELPA](https://elpa.gnu.org/) - GNU Emacs Lisp 包存档。
- [use-package](https://github.com/jwiegley/use-package) - 声明式包配置。

## 编辑增强
- [Evil](https://github.com/emacs-evil/evil) - Emacs 的 Vim 键绑定层。
- [Magit](https://magit.vc/) - Emacs Git 客户端。
- [Helm](https://github.com/emacs-helm/helm) - 增量完成和选择框架。
- [Ivy](https://github.com/abo-abo/swiper) - 完成框架。
- [Company](https://company-mode.github.io/) - 补全框架。
- [Yasnippet](https://github.com/joaotavora/yasnippet) - 代码片段引擎。

## 组织模式
- [Org 模式](https://orgmode.org/) - Emacs 笔记和任务管理器。
- [Org-roam](https://org-roam.org/) - 双向链接笔记系统。
- [Org-journal](https://github.com/bastibe/org-journal) - 日记模式。

## 主题
- [Doom Emacs 主题](https://github.com/hlissner/emacs-doom-themes) - Doom 主题。
- [Solarized](https://github.com/bbatsov/solarized-emacs) - Solarized 配色。
- [Spacemacs 主题](https://github.com/nashamri/spacemacs-theme) - Spacemacs 主题。
