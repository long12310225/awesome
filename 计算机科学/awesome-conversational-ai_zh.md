## 书籍和论文

- [An Annotated Reading List of Conversational AI](https://medium.com/x8-the-ai-community/a-reading-list-and-mini-survey-of-conversational-ai-32fceea97180) - 一份带注释的列表，主要包含有关对话式 AI 的所有主题的学术论文。创建于 2018 年。
- [Designing Bots](https://www.oreilly.com/library/view/designing-bots/9781491974810/) - 本实用指南向您展示如何设计和构建出色的对话体验和令人愉快的机器人，以帮助人们提高工作效率。
- [Designing Voice User Interfaces](https://www.oreilly.com/library/view/designing-voice-user/9781491955406/) - 无论您是设计移动应用程序、玩具还是家庭助理等设备，这本实用的书籍都会指导您了解基本的 VUI 设计原则，帮助您选择正确的语音识别引擎，并向您展示如何衡量 VUI 的性能并对其进行改进。

## 客户

- [Alexa](https://developer.amazon.com/) - 将您的聊天机器人或虚拟助手作为一项技能部署到亚马逊基于云的语音服务 Alexa。
- [LINE](https://developers.line.biz/) - 将您的聊天机器人连接到 LINE，并利用他们的机器人设计器来满足原型设计需求。
- [Messenger](https://developers.facebook.com/docs/messenger-platform/) - 通过 Messenger 添加聊天机器人，增强您的 Facebook 主页体验。
- [MS Teams](https://developer.microsoft.com/microsoft-teams) - 利用 MS Teams 机器人 SDK 和框架将您的聊天机器人变成 MS Teams 应用程序。
- [Slack](https://api.slack.com/) - 通过将聊天机器人部署到 Slack 来接触最终用户的工作地点。
- [Telegram](https://core.telegram.org/) - 利用 Telegram Bot API 轻松将您的聊天机器人连接到 Telegram。
- [WhatsApp](https://developers.facebook.com/docs/whatsapp/) - 使用 WhatsApp Business Platform API 将机器人与客户联系起来。

## 对话式用户体验

- [Conversation Design Institute](https://www.conversationdesigninstitute.com/) - 据我所知，这是第一个提供对话式设计课程、认证和年度会议的机构。
- [Creating AI Conversations Panel Series](https://www.youtube.com/playlist?list=PL_kEIZJfQ6lIALEhtaFYuIsPtzeKtJ4_h) - 创建人工智能对话是一系列小组讨论，深入探讨围绕人工智能实体与人的个性、对话和互动的思考、研究和过程。由斯坦福 MediaX 组织。
- [SAP Conversational Experience Design](https://experience.sap.com/conversational-ux/) - SAP Conversational Experience Design (SAP CxD) 是一种用于企业会话产品的设计语言。它提供了用于设计跨业务应用程序无缝集成的对话产品的指南和工具。
- [Voicebot.ai](https://voicebot.ai/) - 在一处提供最重要的语音技术新闻、评论、研究和分析。
- [Voiceflow](https://www.voiceflow.com/) - 一款对话设计工具，供团队设计、原型设计和启动语音和聊天助手。

## 自然语言理解

- [Awesome Deep Learning for Natural Language Processing](https://github.com/brianspiering/awesome-dl4nlp) - 专门用于自然语言处理深度学习的精选资源列表。
- [Awesome Machine Learning](https://github.com/josephmisiti/awesome-machine-learning) - 精彩的机器学习框架、库和软件的精选列表（按语言）。
- [Awesome Natural Language Generation](https://github.com/accelerated-text/awesome-nlg) - 专门用于自然语言生成的精选资源列表。
- [Awesome NLP](https://github.com/keon/awesome-nlp) - 专门用于自然语言处理的精选资源列表。
- [Awesome NLP with Ruby](https://github.com/arbox/nlp-with-ruby) - 专用于 Ruby 自然语言处理和文本处理的精选资源列表。
- [The NLP Index](https://index.quantumstat.com/) - NLP Index 是一款免费的开源代码发现工具，包含 3000 多个代码存储库。它包括 arxiv 研究论文 PDF、ConnectedPapers 链接及其 GitHub 存储库。

## 平台

- [Amazon Lex](https://aws.amazon.com/lex/) - 一项 AWS 服务，用于为使用语音和文本的应用程序构建对话界面。
- [Chai](https://chai.ml/) - 开源平台，供开发人员使用 Python 构建聊天机器人并将其部署到 Chai 移动应用程序。
- [Dasha](https://dasha.ai/) - 对话式人工智能平台即服务，重点关注语音。
- [Dialogflow](https://cloud.google.com/dialogflow) - 逼真的对话式人工智能与谷歌最先进的虚拟代理。
- [OpenDialog](https://opendialog.ai/) - OpenDialog 使您能够设计、开发和部署对话应用程序。提供开源版本和企业版。
- [Rasa](https://rasa.com/) - 大规模创建虚拟助手的平台。提供开源版本和企业版。
- [SAP Conversational AI](https://cai.tools.sap/) - SAP 的端到端聊天机器人构建平台。附带免费的社区版和企业版。
