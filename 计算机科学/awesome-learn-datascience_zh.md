# 适合初学者的数据科学教程和资源 [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

*如果您想了解更多关于数据科学的信息，但不知道从哪里开始，这个列表适合您！* :chart_with_upwards_trend:

不需要任何基础知识，但 Python 和统计基础知识肯定会派上用场。这些资源已被我当地的数据科学学生小组 [ML-KA](http://ml-ka.de/) 的许多初学者成功使用。

## 什么是数据科学？

- [“什么是数据科学？”在 Quora 上](https://www.quora.com/What-is-data-science)
- [Explanation of important vocabulary](https://www.quora.com/What-is-the-difference-between-Data-Analytics-Data-Analysis-Data-Mining-Data-Science-Machine-Learning-and-Big-Data-1?share=1) - 大数据的差异化、机器学习、数据科学。
- [Data Science for Business (Book)](https://amzn.to/2voPJUi) - 介绍数据科学及其作为商业资产的用途。
- [Data Science Process: A Beginner’s Comprehensive Guide](https://www.scaler.com/blog/data-science-process/) - 数据科学的技术技能：这强调整个数据科学过程所需的实践技能。

## 通用算法和程序

- [Supervised vs unsupervised learning](https://stackoverflow.com/questions/1832076/what-is-the-difference-between-supervised-learning-and-unsupervised-learning) - 两种最常见的机器学习算法类型。
- [9个重要的数据科学算法及其实现](https://nbviewer.jupyter.org/github/jakevdp/PythonDataScienceHandbook/blob/master/notebooks/05.05-Naive-Bayes.ipynb)
- [Cross validation](https://nbviewer.jupyter.org/github/jakevdp/PythonDataScienceHandbook/blob/master/notebooks/05.03-Hyperparameters-and-Model-Validation.ipynb) - 评估您的算法/模型的性能。
- [Feature engineering](https://nbviewer.jupyter.org/github/jakevdp/PythonDataScienceHandbook/blob/master/notebooks/05.04-Feature-Engineering.ipynb) - 修改数据以更好地进行模型预测。
- [10 个重要数据科学算法的科学介绍](http://www.cs.umd.edu/%7Esamir/498/10Algorithms-08.pdf)
- [Model ensemble: Explanation](https://www.analyticsvidhya.com/blog/2017/02/introduction-to-ensembling-along-with-implementation-in-r/) - 将多个模型合并为一个以获得更好的性能。

## 使用 Python 进行数据科学
此列表仅涵盖 Python，因为许多人已经熟悉这种语言。 [使用 R 的数据科学教程](https://github.com/ujjwalkarn/DataScienceR)。

### 一般

- [O'Reilly Data Science from Scratch (Book)](https://amzn.to/2GSjjrK) - 数据处理、实施和可视化以及示例代码。
- [Coursera Applied Data Science](https://www.coursera.org/specializations/data-science-python) - 使用 Python 的在线课程涵盖了大多数相关工具包。

### 学习Python

- [senddex 的 YouTube 教程系列](https://www.youtube.com/watch?v=oVp1vrfL_w4&list=PLQVvvaa0QuDe8XSftW-RAxdo6OmaeL85M)
- [交互式 Python 教程网站](http://www.learnpython.org/)

### 麻木
[numpy](http://www.numpy.org/) 是一个 Python 库，它提供大型多维数组和对其进行快速数学运算。

- [DataCamp 上的 Numpy 教程](https://www.datacamp.com/community/tutorials/python-numpy-tutorial#gs.h3DvLnk)

### 熊猫
[pandas](http://pandas.pydata.org/index.html)为Python提供了高效的数据结构和分析工具。它是建立在 numpy 之上的。

- [熊猫简介](http://www.synesthesiam.com/posts/an-introduction-to-pandas.html)
- [DataCamp pandas foundations](https://www.datacamp.com/courses/pandas-foundations) - 付费课程，但创建帐户后有 30 天免费时间（足以完成课程）。
- [Pandas cheatsheet](https://github.com/pandas-dev/pandas/blob/master/doc/cheatsheet/Pandas_Cheat_Sheet.pdf) - 快速概述最重要的功能。

### scikit学习
[scikit-learn](http://scikit-learn.org/stable/) 是 Python 中最常见的机器学习和数据科学库。

- [简介及首个模型应用](https://nbviewer.jupyter.org/github/jakevdp/PythonDataScienceHandbook/blob/master/notebooks/05.02-Introducing-Scikit-Learn.ipynb)
- [选择估算器的粗略指南](http://scikit-learn.org/stable/tutorial/machine_learning_map/)
- [Scikit-learn 完整用户指南](http://scikit-learn.org/stable/user_guide.html)
- [模型集成：Python 实现](http://machinelearningmastery.com/ensemble-machine-learning-algorithms-python-scikit-learn/)

### Jupyter笔记本
[Jupyter Notebook](https://jupyter.org/) 是一个用于轻松数据可视化和代码演示的 Web 应用程序。

- [下载并运行第一个 Jupyter 笔记本](https://jupyter.org/install.html)
- [用于数据探索的示例笔记本](https://www.kaggle.com/sudalairajkumar/simple-exploration-notebook-instacart)
- [Seaborn data visualization tutorial](https://elitedatascience.com/python-seaborn-tutorial) - 与 Jupyter 配合良好的绘图库。


### 各种其他有用的工具和资源

- [用于组织数据科学项目的模板文件夹结构](https://github.com/drivendata/cookiecutter-data-science)
- [Anaconda Python distribution](https://www.continuum.io/downloads) - 包含数据科学的大部分重要 Python 包。
- [Spacy](https://spacy.io/) - 用于处理基于文本的数据的开源工具包。
- [LightGBM gradient boosting framework](https://github.com/Microsoft/LightGBM) - 成功用于许多 Kaggle 挑战。
- [Amazon AWS](https://aws.amazon.com/) - 租用云服务器进行更耗时的计算（r4.xlarge 服务器是一个很好的起点）。


## 初学者的数据科学挑战
按复杂性递增排序。

- [Walkthrough: House prices challenge](https://www.dataquest.io/blog/kaggle-getting-started/) - 演练有关房价的简单挑战。
- [Blood Donation Challenge](https://www.drivendata.org/competitions/2/warm-up-predict-blood-donations/) - 预测捐赠者是否会再次捐赠。
- [Titanic Challenge](https://www.kaggle.com/c/titanic) - 预测泰坦尼克号上的生存情况。
- [Water Pump Challenge](https://www.drivendata.org/competitions/7/pump-it-up-data-mining-the-water-table/) - 预测非洲水泵的运行状况。
