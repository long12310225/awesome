# 令人敬畏的量子计算 [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

[<img src =“https://raw.githubusercontent.com/desireevl/awesome-quantum-computing/master/logo.png”align =“right”width =“110”>](https://en.wikipedia.org/wiki/Quantum_computing)

[量子计算](https://en.wikipedia.org/wiki/Quantum_computing)利用纠缠和叠加等量子力学现象来操纵量子比特在量子计算机上执行计算。目前可用的工具是在公共可用的量子计算机上创建和运行程序的工具以及了解它们的资源。

这是有关学习和开发量子计算机的最新资源的精选列表。目标是建立一个分类的社区驱动的最新、高质量资源集合。

## 学习

- [An Interactive Introduction to Quantum Computing](https://davidbkemp.github.io/QuantumComputingArticle/) - David Kemp 的量子门计算交互式学习。
- [Black Opal](https://q-ctrl.com/black-opal) - 用于学习量子计算基础知识的交互式平台。
- [CNOT](https://cnot.io/) - 通俗易懂，逐步介绍量子计算概念。
- [Chris Ferrie](https://csferrie.medium.com/) - 澳大利亚悉尼大学教授，《婴儿量子计算》（以及更多其他书籍）的作者，以及 Medium 上优秀的量子计算讲座的作者。
- [Documentation for Forest and pyQuil](http://pyquil.readthedocs.io/en/latest/) - Rigetti 计算 SDK 教程。
- [Documentation for Strawberry Fields](https://strawberryfields.readthedocs.io/en/latest/) - 有关量子计算的光子连续变量方法的背景信息，以及 Xanadu SDK 的教程。
- [D-Wave Leap](https://www.dwavesys.com/take-leap) - 演示和教育资源以及 D-Wave 量子计算机的访问权限。
- [IBM Q Full User Guide](https://quantumexperience.ng.bluemix.net/qx/tutorial?sectionId=full-user-guide&page=introduction) - 简短的教程简要介绍了量子计算和 IBM Q。
- [Introduction to Quantum Computing](https://home.cern/news/announcement/computing/online-introductory-lectures-quantum-computing-6-november) - CERN（欧洲核研究组织）关于量子计算的在线介绍讲座。
- [John Preskill's Notes on Quantum Computation](http://www.theory.caltech.edu/~preskill/ph219/index.html#lecture) - John Preskill 的加州理工学院量子计算课程物理 219/计算机科学 219 笔记。
- [Nielsen and Chuang](https://workedproblems.wordpress.com/category/nielsenchuang/) - 关于量子算法问题的工作示例。
- [Qiskit Tutorials](https://github.com/Qiskit/qiskit-tutorial) - Jupyter Notebook 简单演示了如何使用 Qiskit。
- [Quantum Algorithm Implementations for Beginners](https://arxiv.org/pdf/1804.03719.pdf) - 使用 qiskit 对 20 种不同的量子算法进行了简要调查。
- [Quantum Algorithm Zoo](http://math.nist.gov/quantum/zoo/) - 量子算法的综合目录。
- [Quantum Algorithms](https://quantumalgorithms.org/) - 关于量子算法的开源讲座笔记，重点是机器学习和数据分析。
- [Quantum Computer Programming](https://cs269q.stanford.edu/syllabus.html?fbclid=IwAR09_JNstMi4WVU4oMHDpWR6xWaSISlrYPjWTUTnhcRdEQhzpoOTRgQN8LI) - 向没有量子力学背景的人教授量子计算的斯坦福大学课程。
- [Quantum Computing Foundations](https://docs.microsoft.com/en-us/learn/paths/quantum-computing-fundamentals/) - Microsoft Learn 学习路径涵盖 Azure Quantum 和量子计算元素。
- [Quantum Computing Playground](http://www.quantumplayground.net/#/home) - 3D 量子态可视化工具能够模拟多达 22 个量子位。
- [Quantum Computing UK](https://quantumcomputinguk.org/) - 免费 Qiskit 教程和代码存储库。
- [Quantum Computing for the Very Curious](https://quantum.country/qcvc) - Michael Nielsen 和 Andy Matuschak 撰写的介绍量子计算的论文。
- [Quantum Computing with Amazon Braket SDK](https://www.manning.com/liveprojectseries/quantum-computing-with-amazon-braket-sdk-ser) - Manning liveProject 系列介绍了 Braket API 并指导学习者实现几种简单的算法。
- [Quantum Computing with Microsoft QDK](https://www.manning.com/liveprojectseries/quantum-computing-with-five-projects) - Manning liveProject 系列指导学习者使用 Microsoft QDK 实施各种协议和算法。
- [Quantum Inspire Knowledge Base](https://www.quantum-inspire.com/kbase/introduction-to-quantum-computing) - 易于阅读的知识库，包含丰富的基本量子计算概念。
- [Quantum in the Cloud](http://cnotmz.appspot.com/#) - 四量子位光子量子模拟器和计算机。
- [Quantum Katas](https://github.com/Microsoft/QuantumKatas/) - 用于学习量子计算和 Q# 的编程练习。
- [Quantum Machine Learning for Data Scientists](https://arxiv.org/pdf/1804.10068.pdf) - 量子机器学习算法的解释。
- [Quirk](http://algassert.com/quirk) - 基于浏览器的拖放式量子电路模拟器，可实时反应、模拟和制作动画。
- [QWorld's Bronze-Qiskit](https://gitlab.com/qworld/bronze-qiskit) - 非常容易理解的面向计算/编程的教程。它从矩阵基础知识开始一直到 Grover 算法。还有解释教程的随附视频。
- [The Quantum Quest](https://www.quantum-quest.nl/) - 专为高中生设计的量子计算原理介绍性网络课程。

## 开发工具

- [Amazon Braket](https://aws.amazon.com/braket/) - 完全托管的服务提供在量子模拟器和计算机上运行量子电路的开发环境。
- [Blueqat](https://github.com/Blueqat/Blueqat) - 用于量子门计算的 Python 软件开发套件。
- [Boulder Opal](https://q-ctrl.com/boulder-opal) - 用于自动化和优化量子硬件性能的 Python 工具集。
- [Cirq](https://github.com/quantumlib/Cirq) - 用于编写、操作和优化 NISQ 电路以在量子计算机上运行的 Python 库。
- [Covalent](https://github.com/AgnostiqHQ/covalent) - 从 CPU 到 GPU 到量子计算机的异构基础设施上的分布式计算框架。
- [Fire Opal](https://q-ctrl.com/fire-opal) - 用于提高量子算法结果质量的 Python 包。
- [IBM Quantum Experience](https://quantum-computing.ibm.com) - 在线量子作曲家在真实的量子计算硬件上运行实验。
- [Mitiq](https://mitiq.readthedocs.io/) - 用于在量子计算机上实施错误缓解技术的 Python 工具包。
- [NISQAI](https://github.com/quantumai-lib/nisqai) - 用于在近期量子计算机上执行量子人工智能的库。
- [Ocean](https://docs.ocean.dwavesys.com/en/latest/overview/install.html) - D-Wave 的 SDK，用于使用 Python 在其量子计算机上进行开发。
- [Orquestra](https://www.zapatacomputing.com/orquestra/) - Zapata 计算的统一量子操作环境，允许支持量子的工作流程。
- [Paddle Quantum](https://github.com/PaddlePaddle/Quantum) - 百度的量子机器学习Python工具包。
- [PennyLane](http://github.com/XanaduAI/pennylane) - 集成了各种其他平台的量子计算和量子机器学习的开源框架。
- [Project Q](http://projectq.ch/) - 在 Python 中实现量子计算的框架。
- [pyQuil](https://github.com/rigetticomputing/pyquil) - 使用 Rigetti 的 Quil 进行量子编程的 Python 库。
- [pytket](https://cqcl.github.io/pytket/build/html/index.html) - 用于与剑桥量子计算的 t|ket> 连接的 Python 模块；电路优化和量子位分配的工具。
- [QCL](http://tph.tuwien.ac.at/~oemer/qcl.html) - 较旧的、类似 C 的量子计算机语言。只有模拟器和调试器。
- [Qiskit SDK](https://qiskit.org) - IBM 的软件开发套件，用于在模拟器和真实硬件上编写和运行量子算法。
- [Qiskit.js](https://github.com/QISKit/qiskit-js) - IBM 开发的 Qiskit for JavaScript。
- [Qrack](https://vm6502q.readthedocs.io) - LGPL 许可的高性能 C++ 量子模拟器库、文档和测试代码。
- [Quantum++](https://github.com/vsoftco/qpp) - 高性能现代 C++11 量子计算库。
- [Quantum Inspire](https://www.quantum-inspire.com/) - 在模拟器或量子硬件上运行量子算法的平台 - 由 QuTech 提供。
- [Quantum Programming Studio](https://quantum-circuit.com/) - 基于 Web 的量子编程 IDE 和模拟器。
- [Quipper](https://www.mathstat.dal.ca/~selinger/quipper/) - 用于量子计算的嵌入式、可扩展、函数式编程语言。
- [Qurry](https://github.com/LSaldyt/Qurry) - 基于函数和概率范式的量子概率编程语言。
- [QuTiP](http://qutip.org/docs/latest/index.html) - Python 中的量子工具箱，用于模拟开放量子系统的动力学。
- [Q#](https://docs.microsoft.com/en-gb/quantum/?view=qsharp-preview) - Microsoft 量子开发套件和 Q# 编程语言。
- [Strangeworks Platform](https://quantumcomputing.com/) - 与硬件无关的平台和接口允许专注于开发而不是特定的硬件解决方案构建。
- [TensorFlow Quantum](https://www.tensorflow.org/quantum) - 一个量子机器学习库，将 Cirq 与 TensorFlow 集成，用于经典或量子数据的混合量子经典模型的原型设计。
- [Tequila](https://github.com/aspuru-guzik-group/tequila) - 由 Alan Aspuru-Guzik 小组（多伦多大学）开发的可扩展量子信息和学习架构。

## 图书
- [An Introduction to Quantum Computing](https://www.amazon.com/Introduction-Quantum-Computing-Phillip-Kaye/dp/019857049X/) - 在可访问性和数学严谨性之间取得了良好的平衡。它适合本科生。
- [Classical and Quantum Computation](https://books.google.com.au/books/about/Classical_and_Quantum_Computation.html?id=TrMposZZ0MQC&redir_esc=y) - 介绍经典和量子计算的基础知识。

- [Learn Quantum Computation using Qiskit](http://community.qiskit.org/textbook) - 一本开源教科书，涵盖量子算法并展示如何使用 Qiskit 在真实硬件上运行它们。还涵盖先决条件。
- [Learn Quantum Computing with Python and Q#](https://www.manning.com/books/learn-quantum-computing-with-python-and-q-sharp) - 介绍使用 Python 和 Q#（微软用于量子编程的新语言）的量子计算。
- [Problems and Solutions in Quantum Computing](https://www.worldscientific.com/worldscibooks/10.1142/6077#) - 通过详细的解决方案轻松解决高级量子计算和信息问题。
- [Programming Quantum Computers: Essential Algorithms and Code Samples](https://www.amazon.com/Programming-Quantum-Computers-Essential-Algorithms/dp/1492039683) - 量子计算的实践介绍，重点关注概念和编程示例（多种语言）。
- [Quantum Computation and Quantum Information](http://mmrc.amss.cas.cn/tlb/201702/W020170224608149940643.pdf) - 适合具有数学、计算机科学和物理学基础知识的人士的综合教科书。
- [Quantum Computing: An Applied Approach](https://www.springer.com/gp/book/9783030239213) - 量子计算的实践介绍，解释了量子计算的基础和量子系统背后的数学。
- [Quantum Computing: A Gentle Introduction](http://mmrc.amss.cas.cn/tlb/201702/W020170224608150244118.pdf) - 仅需要基本的大学数学知识即可解释量子计算。
- [Quantum Computing Explained](https://www.amazon.com/Quantum-Computing-Explained-David-McMahon/dp/0470096993) - 使用有效的解决方案解释量子计算的对话方法。
- [Quantum Computing for Computer Scientists](https://www.amazon.com/Quantum-Computing-Computer-Scientists-Yanofsky/dp/0521879965) - 量子计算使用计算机科学本科生可以使用的方法进行解释。
- [Quantum Computing for Everyone](https://www.amazon.com/Quantum-Computing-Everyone-Mit-Press/dp/0262539535) - 为普通读者介绍量子位、纠缠和量子隐形传态等主题。
- [Quantum Computing for the Quantum Curious](https://link.springer.com/book/10.1007%2F978-3-030-61601-4) - 面向高中生、本科生和公众的免费量子计算教科书。
- [Quantum Computing in Action](https://www.manning.com/books/quantum-computing-in-action) - 适合希望尽早开始量子计算的各个级别的 Java 开发人员。
- [Understanding Quantum Technologies](https://www.oezratty.net/wordpress/2021/understanding-quantum-technologies-2021/) - 优秀的书籍，提供了涵盖所有维度的 360 度量子技术方法。
