# 很棒的 TensorFlowJS [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

TensorFlow.js 是[一个开源软件库](https://github.com/tensorflow/tfjs)，用于在 JavaScript 中开发机器学习模型，并使用机器学习
直接在浏览器或 Node.js 中（训练和推理）。

该列表收集了最好的 [TensorFlow.js](https://www.tensorflow.org/js) 工具、资源、视频和闪亮的东西。

## 📖 学习

### 📃 文档

- [官方文档](https://www.tensorflow.org/js/guide)
- [API参考](https://js.tensorflow.org/api/latest/)


### 📝 教程

- [TensorFlow.js gallery](https://github.com/tensorflow/tfjs/blob/master/GALLERY.md) - TensorFlow.js 项目、教程、视频等的集合。
- [Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course/) - Google 快节奏、实用的机器学习介绍。
- [Google Codelabs](https://codelabs.developers.google.com/) - Google Developers Codelabs 提供指导式、教程式、实践编码体验。
- [Beginning ML with TensorFlow.js](https://academy.infinite.red/p/beginning-machine-learning-with-tensorflow-js) - 为 Web 开发人员提供进入 TF.js 的完整在线课程。
- [Effective TensorFlow.js](https://effectivemachinelearning.com/TensorFlow.js) - TensorFlow.js 教程和最佳实践。


### 🤖 模型/项目

- [Official tfjs models on TensorFlow.js repo](https://github.com/tensorflow/tfjs-models) - TensorFlow.js 的预训练模型。
- [Official tfjs models on TensorFlow.js website](https://www.tensorflow.org/js/models) - 探索可立即用于任何项目的预训练 TensorFlow.js 模型。
- [@tensorflow-models on NPM](https://www.npmjs.com/search?q=%40tensorflow-models) - NPM 上 TensorFlow.js 的各种预训练模型。
- [Official tfjs examples repo](https://github.com/tensorflow/tfjs-examples) - 使用 TensorFlow.js 构建的示例！
- [Official tfjs demo website](https://www.tensorflow.org/js/demos) - 查看使用 TensorFlow.js 构建的示例和现场演示！


### 📎 论文

- [TensorFlow.js: Machine Learning for the Web and Beyond](https://arxiv.org/abs/1901.05350) - 本文介绍了 TensorFlow.js 的设计、API 和实现，并重点介绍了一些有影响力的用例。
- [JSDoop and TensorFlow.js: Volunteer Distributed Web Browser-Based Neural Network Training](https://arxiv.org/abs/1910.07402) - 实验结果表明，在分布式网络浏览器中训练神经网络是可行且准确的，具有较高的可扩展性，是一个有趣的研究领域。


### 🎥 视频

- [Official TensorFlow.js video series](https://www.youtube.com/playlist?reload=9&list=PLs6AluHXaQnjeI6jzDkpKXvbPj31i4GgF) - YouTube 内容重点关注 TensorFlow 生态系统更新。
- [TensorFlow.js: ML for the web and beyond](https://youtu.be/iH9CS-QYmZs) - 2020 年 TensorFlow 开发峰会。
- [Rock Paper Scissors!](https://www.youtube.com/watch?v=y4pfTQJaUJU) - 在您的机器上训练剪刀石头布，并使用网络摄像头观看结果！
- [Color Blind Tensors](https://www.youtube.com/watch?v=X55m9eS5UFU) - 操纵图像张量来感知色盲、实时编码。


### 📚 书籍

- [Deep Learning with JavaScript](https://www.manning.com/books/deep-learning-with-javascript) - 学习使用 TensorFlow.js 构建直接在浏览器中运行的深度学习模型，掌握深度学习的所有基础知识并探索高级概念，例如 Shanqing Cai、Stanley Bileschi、Eric D. Nielsen 和 Francois Chollet 重新训练现有模型以进行迁移学习和图像生成。

- [Hands-On Machine Learning with TensorFlow.js](https://www.amazon.com/Hands-Machine-Learning-TensorFlow-js-applications/dp/1838821732) - Kai Sasaki 编写的使用 TensorFlow.js 库构建与 Web 技术集成的 ML 应用程序的指南。

- [Learning Tensorflow.js: Powerful Machine Learning in JavaScript](https://amzn.to/3dR3vpY) - 在本指南中，作者 [Gant Laborde](https://github.com/gantman) 为包括数据科学家、工程师、Web 开发人员、学生和研究人员在内的广大技术受众提供了 TensorFlow.js 基础知识的端到端实践方法。

- [Practical TensorFlow.js](https://www.apress.com/gp/book/9781484262726) - Web 应用程序开发中的深度学习，作者：Juan De Dios Santos Rivera。

- [Practical Machine Learning in JavaScript](https://www.apress.com/gp/book/9781484264171) - Charlie Gerard 为 Web 开发人员提供的 TensorFlow.js。


### 📰 博客和帖子

- [Official TensorFlow.js blog](https://blog.tensorflow.org/search?label=TensorFlow.js&max-results=20) - 涵盖在 JavaScript 中构建模型以及在浏览器或 Node.js 中使用它们。
- [走向数据科学](https://towardsdatascience.com/search?q=tensorflow.js)



### 🤩 由 TensorFlow.js 提供支持

- [Handwritten Digit Classification](https://github.com/aaronhma/ngconf-2020) - [NgConf 2020](https://www.ng-conf.org/) 演讲的展示项目**使用 TensorFlow.js 进入机器学习世界**。
- [NSFWJS - Indecent Content Checker](https://github.com/infinitered/nsfwjs) - 一个友好的库，用于检查图像以查看它们在客户端或服务器上是否是不雅内容！
- [Pose Animator](https://pose-animator-demo.firebaseapp.com/camera.html) - 一种通过动作捕捉在浏览器中使 SVG 字符栩栩如生的工具。
- [LipSync](https://lipsync.withyoutube.com/) - 基于网络的、有趣的互动体验，可与您喜爱的歌曲进行口型同步。
- [BodyPix](https://storage.googleapis.com/tfjs-models/demos/body-pix/index.html) - 浏览器中的实时人物和身体部位分割。


## 🔨 工具

### 图书馆

- [TensorFlow.js](https://github.com/tensorflow/tfjs) - 用于训练和部署 ML 模型的 WebGL 加速 JavaScript 库。
- [ml5.js](https://ml5js.org/) - 友好的网络机器学习。
- [face-api.js](https://github.com/justadudewhohacks/face-api.js) - 用于在浏览器中进行人脸检测和人脸识别的 JavaScript API，以及带有 tensorflow.js 的 Node.js。
- [handtrack.js](https://github.com/victordibia/handtrack.js/) - 用于直接在浏览器中构建实时手部检测（边界框）原型的库。
- [magenta.js](https://magenta.tensorflow.org/get-started/#magenta-js) - 在浏览器中通过机器智能生成音乐和艺术。
- [Danfo.js](https://danfo.jsdata.org/) - 高性能、直观且易于使用的数据结构，用于操作和处理结构化数据。 （灵感来自熊猫）


### 工具/实用程序

- [Glitch](https://glitch.com/@TensorFlowJS) - 免费在浏览器中构建快速、全栈的 Web 应用程序（Tensorflow.js 模板）。
- [CodePen](https://codepen.io) - *bit.ly/codepen-tfjs* 构建、测试和发现前端代码（Tensorflow.js 模板）的最佳场所。
- [TensorFlow Playground](https://playground.tensorflow.org) - 在您的浏览器中修补神经网络！
- [AiJS](https://aijs.rocks/) - 精选的鼓舞人心的人工智能驱动的 JavaScript 应用程序集合。
- [Teachable Machine](https://teachablemachine.withgoogle.com/) - 训练计算机识别您自己的图像、声音和姿势！
- [TensorPlayground.com](https://www.tensorplayground.com/1.0.0/) - 直接在浏览器中编写、修改和查看张量。然后与其他人分享代码。
